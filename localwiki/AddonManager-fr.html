Le Gestionnaire d'extensions (Addon manager) est un outil permettant d'installer et de gérer des ateliers et des macros complémentaires fournis par la communauté FreeCAD. La liste complète des ateliers est disponible sur add-ons repository.

available in version 0.17


Comment faire

  • Ouvrez le menu Outils → Addon manager.
  • Pour installer un atelier ou une macro: faites défiler la liste et sélectionnez l'atelier complémentaire. La description de l'atelier complémentaire sera affichée sous la liste, ainsi que le lien vers la page complémentaire. Puis appuyez sur le bouton Install / update.
    Pour les macros, cliquez sur l'onglet Macros.
  • Pour supprimer un atelier ou une macro: faites défiler la liste, sélectionnez l'atelier ou la macro complémentaire puis appuyez sur le bouton Remove.
  • Pour mettre à jour un atelier ou une macro: faites défiler la liste, sélectionnez le greffon puis appuyez sur le bouton Install / update.
  • Pour vérifier les mises à jour: appuyez sur le bouton . (Note: l'icône peut sembler différente en fonction de votre système d'exploitation.) Les mises à jour disponibles seront signalées sous la liste. Appuyez à nouveau sur le bouton de mise à jour pour mettre à jour toutes les mises à jour en même temps.

Limitations

  • Les modules complémentaires ne font pas partie du package FreeCAD officiel et ne sont pas pris en charge par l'équipe FreeCAD. Vous devez lire les informations fournies sur chacune des pages complémentaires avant d'en installer une pour vous assurer de savoir ce que vous installez.
  • Les rapports de bogues et les demandes de fonctionnalités doivent être faits directement sur chaque page d'extension.
