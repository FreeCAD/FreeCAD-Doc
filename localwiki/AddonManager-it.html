Addon manager è uno strumento per installare e gestire ambienti e macro aggiuntivi forniti dalla comunità di FreeCAD. L'elenco completo dei workbench è disponibile su add-ons repository.

disponibile dalla versione 0.17


Uso

  • Aprire il menu Strumenti → Addon manager.
  • Per installare un workbench o una macro: scorrere l'elenco e selezionare il componente aggiuntivo. La descrizione del componente aggiuntivo viene visualizzata sotto l'elenco, assieme al collegamento alla pagina del componente aggiuntivo. Quindi premere il pulsante Install / update.
    Per le macro, fare clic sulla scheda Macro.
  • Per rimuovere un workbench o una macro: scorrere l'elenco, selezionare il componente aggiuntivo quindi premere il pulsante Remove.
  • Per aggiornare un workbench o una macro: scorrere l'elenco, selezionare il componente aggiuntivo quindi premere il pulsante Install / update.
  • Per verificare gli aggiornamenti: premere il pulsante . (Nota: l'icona può apparire diversa a seconda del sistema operativo.) Gli aggiornamenti disponibili verranno riportati sotto l'elenco. Premendo di nuovo il pulsante di aggiornamento si aggiorneranno tutti gli aggiornamenti contemporaneamente.

Limitazioni

  • I componenti aggiuntivi non fanno parte del pacchetto FreeCAD ufficiale e non sono supportati dal team di FreeCAD. È necessario leggere le informazioni fornite su ciascuna delle pagine del componente aggiuntivo prima di installare qualsiasi di essi, per essere sicuri di sapere cosa si sta installando.
  • Segnalazioni di bug e richieste di funzioni dovrebbero essere fatte direttamente sulla pagina del componente aggiuntivo.
