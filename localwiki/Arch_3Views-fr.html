Description

Cette commande n'est pas couramment utilisée. Elle sert à générer, une vue à plat de la forme à partir d'un objet Mesh courant, pour être utilisé par l'outil Équipement Arch.

Utilisation

  • Sélectionnez un objet Mesh
  • Sélectionnez Menu Arch → Utilitaires → 3Views
