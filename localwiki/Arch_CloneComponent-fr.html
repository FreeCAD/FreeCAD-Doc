Description

Cet outil produit Arch Composant qui sont des clones d'objets Arch sélectionnés. Contrairement à l'outil Draft Clone, qui produira un clone du même type que l'objet sélectionné, cet outil produit un objet Arch Composant générique, qui peut donc assumer n'importe quel rôle, pas nécessairement le rôle de l'objet cloné.

Le composant clone aura simplement sa propriété CloneOf définie sur l'objet sélectionné.
