Description

Cet outil identifie les orifices (forage, séquences de bords circulaires (edges)) dans un objet shape, et, tente de les fermer avec l'ajout d'une nouvelle face construite sur la séquence de bords. Vous devez cependant, vous assurer que le résultat est un solide.

Utilisation

  1. Selectionnez un objet shape
  2. Pressez le bouton Ferme la figure, entrez dans le menu Arch → Utilitaire.

Script

Cet outil peut être utilisé dans une macro, et utilisé dans la console Python en utilisant la fonction : 

closeHole (shape)
ferme un orifice dans une forme ouverte
