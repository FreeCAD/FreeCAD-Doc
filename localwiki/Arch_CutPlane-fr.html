Description

L'outil Couper un objet suivant un Plan vous permet de couper un objet Architectural selon un plan définit par une face :

  • Vous pouvez couper un objet du coté normal de la face ou du coté opposé à cette face.
  • Cela ajoute un composant de soustraction CutVolume à l'objet Architecturale

Sur l'image ci dessus, deux structures sont coupées selon leur plan respectif.

Utilisation

  1. Sélectionner l'objet à couper, puis une face (la face doit être sélectionnée en dernier)
  2. Appuyer sur le bouton Couper un objet selon un Plan
  3. Choisir si la partie à couper est derrière sur la face normale ou la face de face
  4. Cliquer sur le bouton Ok

Script

L'outil couper sur un plan peut être utilisé dans une macro, et, à partir de la console Python, en utilisant la fonction suivante : 

cutComponentwithPlane (archObject,face,faceSide)
  • archObject est l'objet à couper
  • face est la face d'un objet qui définit le plan de coupe
  • faceSide est le coté à couper. 0 = Derrière, 1 = Devant
