Arch DAE/es

El formato Collada (DAE) es un formato de archivo estándar para el intercambio de datos de mallas. El Módulo de Arquitectura puede importar mallas desde archivos .dae, y exportar objetos de Arquitectura y otros objetos basados en formas al formato .dae.

Ten en cuenta que los objetos importados serán objetos de Malla, y tendrán que convertirse en formas o objetos de Arquitectura para un uso óptimo. El Módulo de Arquitectura dispone de varias herramientas para ayudarte a realizar esa operación.

La funcionalidad de importación de Collada en el módulo de Arquitectura depende de in the Arch module depends on pycollada. Si no está instalado en tu sistema, la importación/exportación de Collada estará desactivada.
