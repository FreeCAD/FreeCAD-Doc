Arch DAE/it

Il formato Collada (DAE) è un formato di file standard per lo scambio di dati Mesh.

Il Modulo Architettura può importare mesh da file .DAE, ed esportare elementi di Architettura e altri oggetti basati su forme (ad esempio gli oggetti Parte) in file .DAE.

Notare che gli oggetti importati sono oggetti Mesh, e devono essere convertiti in oggetti costruiti su Forme di base o in oggetti Architettura per un uso ottimale. Il Modulo Architettura dispone di diversi strumenti che consentono di effettuare tale operazione.

La funzionalità di importazione di Collada nel modulo Architettura dipende da pycollada. Se non è installato nel sistema, l'importazione o esportazione di Collada è disabilitata.
