Description

L'outil Équipement vous offre un moyen simple et pratique d'insérer des éléments autonomes, non structurels tels que des meubles, des équipements sanitaire ou appareils électriques à vos projets. Les Équipements peuvent être basés sur une forme ou un treillis Part shapes , qui leur permet de bénéficier de la solidité et des atous de la géométrie BRep, et générer de belles vues lorsqu'elles sont rendues dans le Section Plane.

A partir de la version 0.17, les objets d'équipement ont aussi une propriété HiRes où un objet Mesh peut être attaché. Les objets équipement peuvent ensuite être créés pour afficher ce maillage dans la vue 3D au lieu de leur forme, ce qui permet d'utiliser des objets maillés haute résolution tels que des meubles détaillés que l'on trouve généralement sur les sites Web.

Lors de l'utilisation de l'exportateur Arch OBJ, tous les équipements en mode d'affichage Mesh seront exportés en tant que maillage au lieu d'objet forme.

Utilisation

  1. Sélectionnez une forme Part forme, et optionnellement un objetMaillage
  2. Appuyez sur le bouton Équipement Arch, appuyez la touche E ou alors Q

Options

  • Les équipements partagent les propriétés communes et les comportements de tous les Arch Composants

Propriétés

  • DONNÉESModèle: Une description du modèle de cet équipement.
  • DONNÉESUrl: Une URL de la page du produit, où plus d'informations sur cet équipement peut être trouvée.

Script

L'outil Équipement peut-être utilisé dans des macros et depuis la console python en utilisant la fonction suivante: 

makeEquipment ( baseObject )
  • Crée un objet équipement à partir d'un objet de base (Maillage ou de Pièce)
  • Retourne le nouvel objet équipement, ou None si l'opération a échoué.

Exemple: 

import Part, Arch
box = Part.makeBox(2,2,2)
base = FreeCAD.ActiveDocument.addObject("Part::Feature","Box")
base.Shape = box
Arch.makeEquipment(base)
