Descrizione

Esplora IFC è una semplice utility per esplorare il contenuto di un file IFC. I file IFC sono dei file di testo, e sono quindi leggibili in un editor di testo, ma le informazioni sono condensate e difficile da esplorare. Questa utility presenta esattamente lo stesso contenuto, ma visualizzato in un modo più semplice.

Lo scopo di questo explorer è semplicemente quello di permettere di controllare ciò che è realmente scritto in un file IFC, nel caso in cui si desideri verificare se i contenuti sono stati importati o esportati correttamente verso o da un'applicazione che supporta file IFC come FreeCAD.
