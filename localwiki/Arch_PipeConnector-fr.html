Description

Disponible dans la version 0.17

Cet outil vous permet de créer des raccords entre deux ou plusieurs Arch Tubes sélectionnés.

Utilisation

  1. Sélectionnez 2 or 3 Arch Tubes. Si vous avez sélectionné trois tubes, deux doivent être parfaitement alignés.
  2. Cliquez le bouton Arch Tube Conneteur, ou pressez les touches P et C

Propriétés

  • DONNÉESRadius: Le rayon de la courbe de la connexion

Travail typique

Voir Arch Tube Travail_typique

Script

L'outil Tube Connexion peut être utilisé dans une macro ou dans la console Python en utilisant la fonction suivante: 

import Arch
Arch.makePipeConnector ( objectsList )
