Description

Cet outil identifie et sélectionne les objets mailles non solides (non-manifold) dans un groupe d'Objets de maillage sélectionné.

Utilisation

  1. Sélectionnez un objet mesh
  2. Cliquez sur le bouton Select non solid dans Arch → menu Utilitaires
