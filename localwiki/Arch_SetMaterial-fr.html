Description

L'outil matériaux permet d'ajouter des matériaux dans le document actif, et attribue un matériau à un objet Arch. Les matériaux peuvent prendre toutes les propriétés d'un certain matériel, et contrôler la couleur de l'objet auquel il est attaché. Les données sont stockées dans le dossier Matériaux dans le document actif.

Utilisation

  1. Optionnellement , sélectionnez un ou plusieurs objets pour leur attribuer un nouveau matériau.
  2. Cliquez sur le bouton Set Material
  3. Charge un matériau prédéfini, ou crée un nouveau matériau.
  4. Cliquez sur le bouton OK

Options

  • Lors de la création d'un nouveau matériau, un panneau de tâches vous permet de définir différentes options:

  • Choose preset: Choisissez un matériau prédéfini, et utilisez le tel quel, adaptez le ou modifiez le
  • Name: Donnez un nom au matériau.
  • Edit button: Ouvre un matériau prédéfini dans FreeCAD Material editor, et donnez lui des propriétés additionnelles et ajoutez en.
  • Description: Description des détails du matériau.
  • Color: Affiche les couleurs de matériau, si vous appliquez une autre couleur au matériau, tous les matériaux de ce type auront cette couleur.
  • Code: Un nom et un numéro de spécification déterminé dans le système Masterformat ou Omniclass.
  • Code browser button: Pas encore implémenté - permettra d'ouvrir la référence dans un navigateur Web.
  • URL: Une URL pour connaître les informations du matériau utilisé.
  • URL button:Ouvre l'URL dans un navigateur web.
