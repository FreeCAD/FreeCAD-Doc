Description

L'outil Espace vous permet de définir un volume vide, soit en le fondant sur une forme solide, en définissant ses limites ou en mélangeant les deux. Si elle est basée uniquement sur des limites, le volume est calculé en partant de la zone de délimitation de toutes les limites indiquées, et en soustrayant les espaces derrière chaque frontière. L'objet spatial définit toujours un volume solide. La surface au sol d'un objet spatial, est calculée par l'intersection du plan horizontal au centre de la masse du volume de l'espace, elle peut également être affichée, en réglant le mode d'affichage de l'objet Espace sur "détails".

Dans l'image ci-dessus, un objet spatial est créé à partir d'un objet solide existant, puis deux faces murales sont ajoutées en tant que limites et le mode d'affichage est "détaillé" pour montrer la surface du plancher.

Utilisation

  • Sélectionnez une face ou un objet existant
  • Cliquez le bouton Espace , ou pressez les touches S, P

Propriétés

  • DONNÉESBase: L'objet de base, doit être un solide
  • DONNÉESLimites: Une liste des limites des éléments

Script

L'outil Espace peut être utilisé dans la console Python ou dans une macro en utilisant la fonction: 

makeSpace(objects)
  • Crée un objet Espace avec l'objet donné.
  • Les objets peuvent être un objet document, dans ce cas, il devient la forme de base de l'objet Espace, ou une liste d'objets sélectionnés et retournés par la commande FreeCADGui.Selection.getSelectionEx(), ou une liste de tuples (object, subobjectname).
  • Retourne un objet Espace nouvellement créé.

Exemple: 

import FreeCAD, Arch, Part
b = Part.makeBox(2,2,2)
FreeCAD.ActiveDocument.addObject("Part::Feature","Box").Shape=b
sp = makeSpace([FreeCAD.ActiveDocument.Box])

Après avoir créé un objet Espace, les faces sélectionnées peuvent être ajoutées avec la fonction suivante: 

import FreeCADGui
Arch.addSpaceBoundaries(sp, FreeCADGui.Selection.getSelectionEx())

Les limites peuvent également être éliminées avec: 

Arch.removeSpaceBoundaries(sp, FreeCADGui.Selection.getSelectionEx())

Limites

  • Non disponible dans la version 0.14 de FreeCAD
  • Les propriétés "Limites" ne sont pour le moment pas éditables via le GUI
