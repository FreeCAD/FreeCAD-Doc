Descripción

Esta herramienta alterna la visibilidad de todas las restas de un objeto Arch entre visible y oculto. Si un objeto Arch contiene algunos objetos restados visibles y otros ocultos, todos se volverán visibles/ocultos después de ejecutar este comando.

Como utilizar

  1. Seleccione un objeto Arch que tenga sustracciones
  2. Seleccione el menú Arch → Utilidades → Toggle Subcomponents‏‎

Opciones

La opción Editar → Preferencias → Arch → Aplicar estilo de construcción permite dar a todas las sustracciones nuevas el color de construcción dado en Preferencias → Draft → Configuración general de Draft, que dan un mejor resultado con este comando.
