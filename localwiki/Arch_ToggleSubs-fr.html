Description

Cet outil commande la visibilité Afficher/Cacher de toutes les soustraction d'un objet Arch. Si un objet Arch contient des objets visibles et des objet de soustraits invisibles, alors tous les objets deviennent Visible/Invisibles après activation de cette fonction.

Comment l'utiliser

  1. Sélectionnez un objet Arch qui contient des soustractions
  2. Sélectionnez menu Arch → Utilitaires → Afficher/Cacher les sous composants

Options

Dans la fenêtre Édition → Préférences → Arch → Paramètres généraux l'option Préférence permet de donner une couleur à tous les nouveaux objets soustraits la construction des couleurs donnée par les paramètres Préférences → Draft → General Draft , donnent un meilleur rendement avec cette commande.
