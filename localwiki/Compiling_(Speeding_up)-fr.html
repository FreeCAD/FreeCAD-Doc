Compiling (Speeding up)/fr

Lors du développement de FreeCAD, il faut compiler à partir de la source et la phase de compilation/construction peut prendre un temps précieux de développement. Voici quelques conseils pour raccourcir ce processus et rendre le temps de compilation plus efficaces.

Contents

  • 2 Désactiver les modules
  • 3 make -j
  • 4 distcc

CCache

Installez ccache pour mettre en cache les builds

Désactiver les modules

Utiliser cmake-curses-gui, cmake-qt-gui, ou désactiver cmake flags pour les modules sur lesquels vous ne travaillez pas

make -j

Utiliser make -j # pour spécifier le nombre de démarrages. Une valeur suggérée est votre nombre de cœurs d'ordinateur, par ex. 

make -j $(nproc)

distcc

Distcc peut être utilisé pour la compilation distribuée sur un réseau.
