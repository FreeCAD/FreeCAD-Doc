Offizielle FreeCAD Installationspakete

Die erste stabile 0.17-Version von FreeCAD '0.17.13509' wurde am 06.04.2018 veröffentlicht. Die Bugfix-Version '0.17.13519' wurde am 23.04.2018 veröffentlicht. Um nachzulesen, was neu ist, lesen Sie die Release Hinweise.

Hinweise für Windows-Benutzer

  • Das 32-Bit-Installationsprogramm (x86) unterstützt die folgenden Windows-Versionen: XP/VISTA/7/8/10.
  • Das 64-Bit-Installationsprogramm (x64) unterstützt die folgenden Windows-Versionen: VISTA/7/8/10.

Hinweise für Mac OS X Benutzer

Mac OS X 10.11 El Capitan ist die minimal unterstützte Version.

Hinweise for GNU/Linux users

FreeCAD kann zwar von den meisten offiziellen Repositories von Linux-Distributionen installiert werden, aber die Version, die diese bereitstellen, ist möglicherweise ziemlich alt und es fehlen viele Funktionen. Stattdessen können Sie das verlinkte AppImage herunterladen, als ausführbar markieren und ohne Installation starten. Auf der Seite Installieren unter Linux finden Sie weitere Installationsoptionen, darunter auch, wie Sie aktuelle Pakete für Ubuntu und Derivate erhalten.

Entwicklungsversionen
