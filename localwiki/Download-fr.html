Veuillez noter que FreeCAD n'a toujours pas atteint le statut d'une version 1.0 à nos yeux, et qu'il pourrait ne pas être prêt pour un usage en production. Néanmoins, de nombreux utilisateurs l'utilisent. Pour évaluer si FreeCAD est approprié pour votre projet, veuillez consulter le forum FreeCAD.

Version stable courante

La version 0.17 initiale de FreeCAD (0.17.13509) a été publiée le 6 avril 2018. La version avec correctifs 0.17.13522 a été publiée le 08 mai 2018. Pour découvrir les nouveautés, consultez les notes de version.

Notes aux utilisateurs de Windows

  • L'installateur 32 bits (x86) est compatible avec les versions de Windows suivantes : 7/8/10.
  • L'installateur 64 bits (x64) est compatible avec les versions de Windows suivantes : 7/8/10.

Notes aux utilisateurs de MacOS X

MacOS X 10.11 El Capitan est la version minimum compatible.

Note aux utilisateurs de GNU/Linux

FreeCAD peut être installé depuis les dépôts officiels de la plupart des distributions Linux, mais la version qu'il proposent pourrait être assez vieille et pourrait manquer de nombreuses fonctionnalités. Vous pouvez plutôt télécharger le fichier AppImage lié ci-dessus, le marquer comme exécutable puis le lancer sans aucune installation. Veuillez consulter la page Installation sous Unix pour plus d'options d'installation, incluant comment obtenir des paquets à jour pour Ubuntu et dérivées.

Versions de développement

FreeCAD est en développement perpétuel! Si vous voulez tester la version de développement 0.18, consultez la page FreeCAD releases.

Modules additionnels et macros

La communauté FreeCAD propose une multitude de modules supplémentaires et de macros. Ils peuvent désormais être installés directement depuis FreeCAD à l'aide du Gestionnaire d'extensions (Addon manager).
