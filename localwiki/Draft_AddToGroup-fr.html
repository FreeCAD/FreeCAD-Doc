Description

Cet outil, affiche un menu, qui vous permet d'ajouter les objets sélectionnés dans un groupe existant, ou les retirer de leur groupe actuel.

Utilisation

  1. Sélectionnez le ou les objets
  2. Pressez sur le bouton Ajouter au groupe.
