Description

S'accroche au centre des arcs et des cercles.

Utilisation

  1. Assurez-vous que le mode d'accrochage est activé
  2. Choisissez l'élément à insérer dans le document FreeCAD
  3. Déplacez le curseur près de l'arc/cercle/bord circulaire auquel vous souhaitez attacher votre élément
  4. Sur le côté droit du curseur apparaît le symbole du mode d'accrochage
  5. Cliquez pour joindre votre élément à l'extrémité sélectionnée de l'élément cible

Fonctionne également avec Draft Move et Draft Rotate.

Notes

Le mode Snap peut être utilisé avec

  • Les éléments de dessin (arcs, cercles)
  • Les éléments d'ébauche (arcs, cercles)
  • Les bords des faces circulaires de la forme
  • les bords des faces circulaires du solide

En cliquant sur l'icône du mode d'accrochage, vous activez/désactivez le mode d'accrochage.
