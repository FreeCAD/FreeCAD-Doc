Draft DAT/fr

Le format de données profil aérodynamique est un simple fichier contenant les métadonnées profil aérodynamique.
A la première et deuxième ligne, se trouvent les coordonnées de la surface supérieure et inférieure.

Il existe de nombreuses versions différentes de ce format, donc, ce module d'importation, essaie d'être aussi intelligent que possible, pour interprêter le maximum de versions.
