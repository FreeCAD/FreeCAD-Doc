説明

このツールを使うとワイヤーやB-スプライン.から点を取り除くことができます。

使用方法

  1. ワイヤーまたはB-スプライン.を選択します
  2. Draft DelPointボタンを押します
  3. wireまたはBSpline上の点をクリックします

オプション

  • この機能は Draft Editからも利用可能です

スクリプト処理

利用できません。ただしWiresやBSplinesから点を取り除くのは簡単です。

例： 

import FreeCAD,Draft
points = FreeCAD.ActiveDocument.ActiveObject.Points
points.pop(0)
FreeCAD.ActiveDocument.ActiveObjects.Points = points
