Description

Vous permet d'aligner sur l'une des trois lignes composant Draft dimension


Utilisation

  1. Assurez-vous que le mode d'accrochage est activé
  2. Choisir un élément (par exemple une ligne de brouillon) à insérer dans le document FreeCAD
  3. Déplacez le curseur près de la dimension de l'élément auquel vous souhaitez attacher votre élément
  4. Dessous du côté droit du curseur, le symbole du mode d'accrochage apparaît
  5. Cliquer pour attacher votre élément à la ligne Draft dimension sélectionnée de l'élément cible
