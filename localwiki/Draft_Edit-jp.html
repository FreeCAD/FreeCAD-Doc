説明

このツールを使うと選択されているオブジェクトの特定のプロパティをグラフィカルに編集することができます。編集可能なオブジェクトのプロパティはワイヤーの節点、長方形の縦横長さ、円の半径などです。

使用方法

  1. ワイヤー、ライン、長方形、B-スプライン、円のいずれかを選択します
  2. Draft Editボタンを押すか、Treeパネル内のオブジェクトをダブルクリックします
  3. 動かしたい点をクリックします
  4. 3Dビュー上で別の点をクリックするか、座標を入力します

オプション

  • Editツールで一度に編集できるのは一つの選択されたオブジェクトだけです。
  • Editツールで編集できるのは製図オブジェクトのワイヤー、長方形、円、円弧だけです。それ以外のオブジェクトはまずはじめに製図オブジェクトに変換する必要があります。、
  • 編集用節点を動かすためにはその点をクリックしてください。選択解除するには再度クリックしてください。
  • 点を指定した後でX、Y、またはZを押すと次の点を指定した軸上に拘束できます。
  • 手入力で座標を指定する場合は各X、Y、Z成分を数値で入力してからENTERを押してください。
  • 描画中にCTRLキーを押すと距離と無関係に最も近いスナップ位置に操作点をスナップすることができます。
  • 描画中にSHIFTキーを押すと最後に入力した点に対して水平方向、垂直方向に次の点を拘束することができます。
  • 編集を終了するにはESCキー、Cancelボタン、あるいは再度 Draft Editボタンを押してください。
  • Add pointボタンを押して線分をクリックするとワイヤーやB-スプラインに点を追加することができます。
  • Del pointボタンを押して点をクリックするとワイヤーや

B-スプラインから点を取り除くことができます。

スクリプト処理

利用できません。上記の各オブジェクトは直接プロパティを変更することができます。
