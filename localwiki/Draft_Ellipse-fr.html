Description

L'outil Ellipse crée une ellipse dans le plan sélectionné en saisissant deux points qui définissent les angles d'une zone rectangulaire dans laquelle l'ellipse prendra sa place. Il prendra la définition de ligne et couleur précédemment défini sous l'onglet tâches.

Utilisation

  1. Pressez le bouton Draft Ellipse ou appuyez sur les touches E puis L
  2. cliquez un premier point sur la vue 3D, ou tapez une coordonnée
  3. cliquez un deuxième point sur la vue 3D, ou tapez une coordonnée

Options

  • Pour entrer manuellement les coordonnées, entrer simplement les nombres, puis appuyez sur ENTER entre chaque composantes X, Y et Z.
  • Appuyez sur T ou activez la case à cocher/décocher Continuer. Si le mode continuer est activé, l'outil Ellipse redémarrera après avoir donné le deuxième point, ce qui vous permet de dessiner une autre ellipse sans devoir appuyer à nouveau sur le bouton Ellipse.
  • Appuyez sur la touche CTRL tout en dessinant force l'accrochage de votre point à l'endroit le plus proche du composant logiciel enfichable, indépendamment de la distance.
  • Appuyez sur la touche Maj en dessinant contrains votre deuxième point horizontalement ou verticalement par rapport au premier point.
  • Appuyez sur la touche I ou Filled pour que l'ellipse apparaisse comme une face fermée.
  • Appuyez sur sur la touche ESC ou Annuler pour annuler la commande.
  • Ellipse, en cas d'affichage en mode "Flat Lines", vous pouvez afficher un motif hachuré, en définissant sa propriété "Pattern" ci-dessous.

Propriétés

  • DONNÉESMajor Radius: Le grand rayon de l'ellipse
  • DONNÉESMinor Radius: Le petit rayon de l'ellipse
  • DONNÉESMake Face: Remplit l'ellipse avec une face
  • VUEPattern: Spécifie le type de hachure que prendra l'ellipse
  • VUEPattern Size: Spécifie la taille de la hachure

Voir aussi Draft Pattern page.

Script

L'outil Ellipse peut être utilisé dans une macro et dans la console Python en utilisant la fonction : 

makeEllipse (majorradius, minorradius, [placement], [facemode])
  • Crée un objet ellipse avec le grand rayon et le petit rayon.
  • Si un placement est donné, il est utilisé.
  • Si facemode a la valeur False, l'ellipse apparaît comme wireframe (fil), sinon comme une face.
  • Retourne l'objet nouvellement créé.

Exemple: 

import Draft
myEllipse = Draft.makeEllipse(4,2)
