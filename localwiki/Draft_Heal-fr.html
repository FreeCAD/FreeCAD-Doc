Description

En raison de l'évolution rapide de FreeCAD, quelques définitions d'objets peuvent changer au fil du temps et entre les versions, provoquant des erreurs de chargement ou contenir des erreurs lorsque le fichier est ouvert avec une version plus récente de FreeCAD. Cette commande tente de corriger ces mauvais objets en recréant un nœud à partir de zéro, puis de copier le contenu des propriétés de l'ancien objet vers le nouveau. Il peut être exécuté avec des objets sélectionnés, dans ce cas, il travaille sur les objets sélectionnés, ou sans objet sélectionné. Tous les objets du présent document seront ensuite analysés pour recherchée les erreurs. Si aucune erreur n'est trouvée, cette commande ne fera rien.

Cette commande ne peut corriger les objets issus du Module Draft.

Utilisation

  1. Sélectionnez un ou plusieurs objets défectueux pour les corriger, ou ne rien sélectionner pour numériser l'ensemble du document
  2. Appuyez sur le bouton Heal
