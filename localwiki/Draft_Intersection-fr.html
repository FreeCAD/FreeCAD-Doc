Description

Ce mode d'accrochage vous permet de sélectionner l'intersection de deux segments de ligne ou d'arc.


Utilisation

  1. Assurez-vous que le mode d'accrochage est activé
  2. Choisissez l'élément à insérer dans le document FreeCAD

Sur Linux

  1. Déplacer le curseur près de l'intersection des éléments auxquels vous souhaitez attacher votre élément
  2. Sur le côté droit du curseur apparaît le symbole du mode d'accrochage
  3. Cliquez pour attacher votre élément au point d'intersection sélectionné des éléments cibles

Sur Windows

  1. Appuyez sur la touche CTRL et déplacez la souris près de l'intersection
  2. Sur le côté droit du curseur apparaît le symbole du mode d'accrochage
  3. Cliquez pour attacher votre élément au point d'intersection sélectionné des éléments cibles

Fonctionne également avec Draft Move et Draft Rotate


Notes

Le mode Snap peut être utilisé avec

  • les éléments du sketcher (lignes)
  • les éléments Draft (DWires, Lignes)

En cliquant sur l'icône du mode d'accrochage, vous activez/désactivez le mode d'accrochage.
