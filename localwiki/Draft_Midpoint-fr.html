Description

Ce mode d'accrochage vous permet de sélectionner le point central des éléments.


Utilisation

  1. Assurez-vous que le mode d'accrochage est activé
  2. Choisissez l'élément à insérer dans le document FreeCAD
  3. Déplacez le curseur près du milieu de l'élément auquel vous souhaitez attacher votre élément
  4. En dessous du côté droit du curseur, le symbole du mode d'accrochage apparaît
  5. cliquer pour attacher votre élément à l'extrémité de l'élément cible sélectionné


Notes

Le mode Snap peut être utilisé avec

  • les éléments du sketcher (lignes, arcs)
  • les éléments de projet (DWires, Lignes, Arcs)
  • les bords des formes
  • les bords des solides

En cliquant sur l'icône du mode d'accrochage, vous activez/désactivez le mode d'accrochage.
