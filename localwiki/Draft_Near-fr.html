Description

S'accroche au point le plus proche de l'objet le plus proche.

Utilisation

  1. Assurez-vous que le mode d'accrochage est activé
  2. Choisissez l'élément à insérer dans le document FreeCAD
  3. Déplacez le curseur près de l'objet auquel vous souhaitez attacher votre élément
  4. Un petit cercle jaune indiquera où vous serez attaché à l'élément cible
  5. Cliquez pour joindre votre élément à l'élément cible

Fonctionne également avec Draft Move et Draft Rotate.

Notes

Le mode d'accrochage détectera toutes les possibilités d'accrochage des autres modes d'accrochage de base ou s'attachera simplement au point le plus proche sur la cible.
