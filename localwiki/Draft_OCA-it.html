Apertura

Questa funzione importa file OCA/GCAD in un nuovo documento.

Il formato di file OCA è il risultato dello sforzo della comunità per creare un formato di file aperto, semplice e libero.

Il formato OCA è in gran parte basato sul formato del file GCAD generato da gCAD3D.

Entrambi i formati possono essere importati in FreeCAD, ed i file OCA esportati da FreeCAD possono essere aperti in gCAD3D.

Attualmente si importano i seguenti oggetti di OCA:

  • Linee
  • Archi e circonferenze
  • Aree chiuse

Importazione

Funziona allo stesso modo di Apertura, ma crea gli oggetti nel documento attivo, invece di creare uno nuovo documento.

Esportazione

Gli oggetti che attualmente si possono esportare sono:

  • Linee e spezzate (polilinee)
  • Archi e circonferenze
  • Aree

Preferenze

Nella scheda Preferenze (menu Modifica -> Preferenze -> Draft) si possono essere specificare i seguenti parametri:

  • Importa aree chiuse o no
