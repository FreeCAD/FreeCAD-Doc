Description

Ce mode d'accrochage vous permet de sélectionner un point sur une ligne imaginaire parallèle à un segment de ligne.

Utilisation

  1. Assurez-vous que le mode d'accrochage est activé
  2. Choisissez l'élément à insérer dans le document FreeCAD

Sur Linux

  1. Déplace le curseur sur l'objet désiré pour activer son accrochage parallèle
  2. Déplacez le curseur jusqu'à ce que le symbole du mode d'accrochage apparaisse en bas à droite du curseur.
  3. Cliquez pour placer votre élément parallèlement à l'objet sélectionné

Sur WIndows

  1. Appuyez sur CTRL et déplacez le curseur sur l'objet désiré pour activer son accrochage parallèle
  2. Déplacez le curseur jusqu'à ce que le symbole du mode d'accrochage apparaisse en bas à droite du curseur.
  3. Cliquez pour placer votre élément parallèlement à l'objet sélectionné

Fonctionne également avec Draft Move et Draft Rotate

Notes

Le mode Snap peut être utilisé avec

  • Les éléments du sketcher (lignes, arcs)
  • Les éléments Draft (DWires, Lignes, Arcs)

En cliquant sur l'icône du mode d'accrochage, vous activez/désactivez le mode d'accrochage.
