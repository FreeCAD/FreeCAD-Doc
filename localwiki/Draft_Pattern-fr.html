Draft Pattern/fr

Description

Tout objet Draft pouvant être fermé, tel que Rectangle, Cercle, Ellipse, Wire ou Polygone quand l'objet est fermé et mis en mode d'affichage "Flat Lines", peut afficher un motif de hachures, au lieu de la couleur de la face, en réglant les propriétés "Pattern"

Utilisation

  1. Sélectionnez l'objet
  2. Dans la sélection Vue -> Données: définir Make Face à True pour s'assurer que l'objet aura une face pour définir le motif
  3. Dans la sélection Vue-> Vue -> Pattern :spécifiez un motif de hachures pour remplir le wire
  4. Dans Combo Vue -> Vue -> Pattern Size: spécifiez la taille du motif de hachures

Note:

  • Le motif n'est pas visible lorsque l'élément est soit sélectionné ou pré-sélectionné.
  • Si vous créez une face avec une modification, l'objet résultant n'est plus un objet Draft donc les options pattern ne sont plus disponibles
  • Le Draft Wire peut être fermé et transformé en une face directement lors de sa création, en cochant la case "Filled"
