説明

製図モジュールは作業平面システムによって特徴づけられています。製図モジュールでは3D空間内に製図コマンドが動作できる独自の平面を指定することができます。作業平面の定義にはいくつかの方法があります：

  • 選択された面から作成
  • 現在のビューから作成
  • プリセットされた上面、前面、側面から作成
  • 指定なし。この場合、コマンド開始時に自動で作業平面が現在のビューに合わせられます。また既存の面に対して描画を開始した場合はその面に合わせられます。

使用方法

  1. Draft SelectPlaneボタンを押します . If your button doesn't look like this, see this note.

オプション

  • 既存の面に作業平面を設定する場合は3Dビューで既存のオブジェクトの面を選択してから Draft SelectPlaneボタンを押します。
  • {KEY|VIEW}}を押すとカメラの視線に垂直で原点(0,0,0) を通過する現在のビュー平面に作業平面が設定されます。
  • NONEを押すと現在の作業平面を解除します。以降の2D操作はビューに依存したものになります。
  • オフセット値を設定することができ、これによって選択した平面から一定距離の位置に作業平面を設定できます。

スクリプト処理

作業平面はスクリプトやマクロから簡単に作成したり操作したりすることができます。独自のものを作ることが可能で、それを現在の製図作業平面とは関係なく使用することが可能です。

例： 

import WorkingPlane
myPlane = WorkingPlane.plane()

現在の製図作業平面にアクセスすることも可能です 

import FreeCAD
draftPlane = FreeCAD.DraftWorkingPlane

作業平面には完全な scripting API スクリプト処理API]]があります。これには位置操作や配置の相互変換を行うための便利な関数が含まれています。
