Description

L'outil ShapeString insère une forme composée représentant une chaîne de texte à l'emplacement donné dans le document courant. Les attributs du texte, hauteur, police ... peuvent être spécifiés.

Utilisation

  1. Appuyez sur le bouton Draft Shape from text ... ou sur les touches S puis S
  2. Cliquez un point dans la vue en 3D, ou une coordonnée
  3. Entrez le texte désiré, puis validez avec ENREE
  4. Entrez la taille désirée, puis validez avec ENREE
  5. Entrez le suivi souhaité, puis validez avec ENREE
  6. Appuyez de sur nouveau sur ENREE pour accepter le fichier police de caractères choisie, ou,
  7. Appuyez sur le bouton avec les 3 petits points ... pour sélectionner une nouvelle police de caractères.

Options

  • Pour entrer les coordonnées manuellement, il suffit d'entrer les nombres, et frapper sur la touche ENREE entre chaque affectation de la composante X, Y et Z.
  • Appuyez sur ESC pour annuler et quitter l'opération.
  • Vous pouvez définir une police de caractère par défaut dans le Menu → Préférences → Draft → onglets Paramètres visuels et Textes et dimensions.

Propriétés

  • DONNÉESPosition: Le point de base de la forme composée
  • DONNÉESString: Le contenu de la chaîne de texte
  • DONNÉESSize: La hauteur des lettres dans les FC unités
  • DONNÉESTracking: L'inter-caractère d'espacement dans l'unité de FreeCAD
  • DONNÉESFont File: Définition de la police de caractères pour dessiner la chaîne

Script

L'outil ShapeString peut être utilisé dans une macro Python à partir de la console en utilisant la fonction suivante: 

makeShapeString(String,FontFile,[Size],[Tracking])
  • Transforme une chaîne de texte en une forme composée avec une police de caractères spécifiée.

Exemple: 

import FreeCAD,Draft
Draft.makeShapeString("This is a sample text",
                       "/usr/share/fonts/truetype/msttcorefonts/Arial.ttf",
                       200.0,10)

Sélectionner une Police de caractère

ShapeString utilise la géométrie interne d'une police de caractères pour créer les formes dans FreeCAD. Pour ce faire, il doit lire le fichier de police de caractères réel (*.tff, etc). Si la boîte de sélection de police de caractères est vide, vous devez taper le chemin complet vers le fichier de police de caractères ou utiliser la touche ... pour sélectionner un fichier de police de caractères.

Limites

  • Cet outil n'est pas encore disponible v0.13. Il sera inclus dans une future version 0.14.
  • Les polices TrueType(*.ttf), OpenType(*.otf) et Type1(*.pfb) sont pris en charge.
  • Une trop faible hauteur du texte peut entraîner une déformation du glyphe du caractère généré, à cause de la perte de détails dû a la faible échelle.
  • La version actuelle est limitée à l'orientation de gauche à droite sur une ligne de base horizontale.
  • Pour créer des textes circulaire utilisez la macro Circular Text
