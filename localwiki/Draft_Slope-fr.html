Description

Cet outil permet de définir la pente (verticale horizontale) des fils ou des lignes sélectionnés. En définissant une valeur de pente, tous les sommets après le premier auront leur valeur Z modifiée afin que les différents segments des fils ou des lignes aient l'inclinaison donnée.

La valeur de la pente est une valeur tangentielle: 0 signifie horizontale, 1 signifie 45 degrés vers le haut, -1 signifie 45 degrés vers le bas, 0,577 signifie 30 degrés vers le haut, etc.

Comment l'utiliser

  1. Sélectionnez un ou plusieurs objets Wire ou [[Line]]
  2. Sélectionnez le menu Draft → Utilitaires → Set Slope
  3. Définissez la valeur souhaitée dans la zone de panneau de tâches Slope
  4. Appuyez sur le bouton OK
