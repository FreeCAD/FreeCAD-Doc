Description

Lorsque cet emplacement d'accrochage est activé, des emplacements d'accrochage spéciaux définis par objet sont disponibles. Ceux-ci inclus:

  • Sommets des lignes de base de Arch Walls
  • Points de base de Arch Structures
  • Tout point défini dans la propriété Points d'accrochage des objets, s'il existe (actuellement seulement Arch Equipments ont une telle propriété). Les vecteurs 3D définis dans la propriété SnapPoints (une propriété VectorList) sont transformés avec le placement de l'objet avant l'accrochage.
