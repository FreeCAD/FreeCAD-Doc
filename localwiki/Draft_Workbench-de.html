Draft Workbench/de

(Redirected from Draft Workbench/de)

Im Arbeitsbereich Entwurf finden sich Werkzeuge um ebene Zeichnungselemente zu konstruieren. Der Arbeisbereich ist vollständig in Python programmiert und ist auch als Beispiel gedacht, um zu zeigen, wie weit Sie FreeCAD mittels Python erweitern können, ohne dabei den C++-Quellcode verändern zu müssen.

Zeichnen von Objekten

Mit diesen Werkzeugen werden Objekte erstellt:

  • 2-Punkt Linie: Zeichnet ein Linien-segment zwischen 2 Punkten
  • Wire (Mehrpunkt-Linie): Zeichnet eine Linie, die sich über mehrere Punkte fortsetzt
  • Kreis: Zeichnet einen Kreis ab dem Mittelpunkt mit einem Radius
  • Bogen: Zeichnet ein Bogen-segment aus Mitte,Radius, Startwinkel und Endwinkel
  • Ellipse: Draws an ellipse from two corner points
  • Polygon: Zeichnet ein Mehreck ab dem Mittelpunkt mit einem Radius
  • Rechteck: Zeichnet ein Rechteck zwischen 2 gegenüberliegeden Punkten
  • Text: Erzeugt einen mehrzeiligen Texteintrag
  • Abmessung: Erzeugt einen Texteintrag als Maßangabe
  • BSpline: Zeichnet eine Kurve aus einer Serie von Punkten
  • Punkt: Inserts a point object
  • Shape from text...: The ShapeString tool inserts a compound shape
  • Facebinder: Creates a new object from selected faces on existing objects
  • Bezier Curve: Draws a Bezier curve from a series of points
  • Label: Platziert eine Beschriftung mit einem Pfeil, der auf ein ausgewähltes Element zeigt verfügbar in Version 0.17

Objekte ändern

Mit diesen Werkzeugen kann man Objekte verändern. Sie wirken auf angewählte Objekte, ist kein Objekt gewählt, werden Sie aufgefordert mindestens eins auszuwählen.

  • Verschieben: Bewegt Objekt(e) von einem Punkt zu einem anderen
  • Drehen: Rotiert Objekt(e) von einem Ausgangswinkel zu einem Endwinkel
  • Versetzen: Bewegt Segmente eines Objekts um einen bestimmten Wert
  • Trim/Extend (Trimex): Beschneidet oder Erweitert ein Objekt
  • Hochstufen: Vereinigt mehrere Objekte in ein Objekt der nächst höheren Ebene
  • Herabstufen: Teilt ein Objekt in einzelne der nächst niedrigeren Ebene
  • Skalieren: Dimensioniert ein ausgewähltes Objekt in Bezug zu einem bestimmten Punkt
  • Zeichnung: Überträgt ausgewählte Objekte in eine 2D-Zeichnung
  • Bearbeiten: Verändert ein ausgewähltes Objekt
  • Wire to BSpline: Converts a wire to a BSpline and vice-versa
  • Punkt hinzufugen: Adds a point to a wire or BSpline
  • Punkt entfernen: Deletes a point from a wire or BSpline
  • Form in 2D Ansicht: Creates a 2D object which is a flattened 2D view of another 3D object
  • Entwurf zu Skizze: Converts a Draft object to Sketch and vice-versa
  • Anordnung: Creates a polar or rectangular array from selected objects
  • Path Array: Creates an array of objects by placing the copies along a path
  • Klonen: Clones the selected objects
  • Mirror: Mirrors the selected objects
  • Stretch: Stretches the selected objects verfügbar in Version 0.17

Hilfsprogramme

Zusätzliche Werkzeuge sind über das Kontextmenü verfügbar, abhängig von den ausgewählten Objekten.

  • Ebene markieren: Sets a working plane from a standard view or a selected face
  • Linie beenden: Ends the drawing of the current wire or bspline, without closing it
  • Close line: Ends the drawing of the current wire or bspline, and closes it
  • Undo line: Undoes the last segment of a line
  • Konstruktionsmodus umschalten: Toggles the Draft construction mode on/off
  • Toggle continue mode: Toggles the Draft continue mode on/off
  • Aktuellen Style anwenden: Applies the current style and color to selected objects
  • Anzeige-Modus unschalten: Switches the display mode of selected objects between "flat lines" and "wireframe"
  • Zur Gruppe hinzufugen: Quickly adds selected objects to an existing group
  • Wahle Gruppe: Selects the contents of a selected group
  • Toggle snap: Toggles object snapping on/off
  • Gitternetz umschalten: Toggles the grid on/off
  • Show snap bar: Shows/hides the snapping toolbar
  • Heal: Heals problematic Draft objects found in very old files
  • Flip Dimension: Flips the orientation of the text of a dimension
  • VisGroup: Creates a VisGroup in the current document
  • Slope: Changes the slope of selected Lines or Wires verfügbar in Version 0.17
  • AutoGroup: Automatically place new objects in a given group verfügbar in Version 0.17
  • Set Working Plane Proxy: Add a proxy object in the document to store a Working Plane position verfügbar in Version 0.17
  • Add to Construction group: Add selected objects to the Construction group verfügbar in Version 0.17

Importieren & Exportieren

Dies sind Funktionen zum Öffnen, Importieren oder Exportieren von anderen Dateiformaten. DasÖffnen wird ein neues Dokument mit dem Inhalt der Datei öffnen, während das Importieren den Dateiinhalt am gegenwärtigen Dokument anhängen wird. Exportieren werden die ausgewählten Objekte in einer Datei speichern. Wenn nichts ausgewählt ist, werden alle Objekte exportiert werden. Seien Sie sich bewusst, dass der Zweck des Draftmoduls ist, mit 2D-Objekten zu arbeiten, daher konzentrieren sich jene Importroutinen nur auf 2D-Objekte, und obwohl DXF und OCA-Formate auch Objekt-Definitionen im 3D-Raum unterstützen (Objekte sind nicht notwendigerweise flach), werden sie volumetrische Objekte wie Maschen, 3D-Flächen, etc. nicht importieren, sondern eher Linien, Kreise, Texte oder flache Formen. Zurzeit unterstützte Dateiformate sind:

  • Präferenz ... Importieren & Exportieren
  • Autodesk .DXF: Importiert and exportiert Zeichnungstauschformat (drawing exchange format)-Dateien, die mit 2D-CAD Anwendungen erstellt wurden
  • SVG (as geometry): Importiert and exportiert Skalierbare Vektorgrafiken (Scalable Vector Graphics), also SVG-Dateien, die mit Vektor-Zeichenprogrammen erstellt wurden
  • Open Cad format .OCA: Importiert and exportiert OCA/GCAD Dateien, also ein potenziell neues, offenes CAD-Datei Format
  • Airfoil Data Format .DAT: Importiert DAT-Dateien, die Tragflächenprofile (Airfoil profiles) beschreiben
  • Autodesk .DWG: Importiert und exportiert DWG-Dateien mit dem DXF-Importierer, wenn der Teigha Converter installiert ist
  • FreeCAD- und DWG- Import: Importiert und exportiert DWG-Dateien
  • FreeCAD- und DXF- Import: Importiert und exportiert DXF-Dateien

Zusätzliche Funktionen

  • Fangen (Snapping): Erlaubt, neue Punkte an speziellen Stellen auf vorhandene Gegenstände zu legen
  • Festlegen (Constraining): Erlaubt, neue Punkte horizontal oder vertikal in Bezug auf vorherige Punkte anzulegen
  • manuelle Koordinateneingabe: Ermöglicht die manuelle Koordinateneingabe, anstatt auf dem Bildschirm anzuklicken
  • Kopiermodus: Alle Änderungswerkzeuge können entweder die ausgewählten Objekte ändern oder erstellen eine modifizierte Kopie von ihnen.
    Durch Drücken und Halten von ALT während das Objekt modifiziert wird , z.b verschoben oder gedreht, wird beim Loslassen der Taste eine Kopie erzeugt.
  • Construction Mode: Ermöglicht Ihnen, Geometrien abgesondert vom Rest zu erstellen, dazu einfach Ein-/ Ausschalten
  • Arbeitsfläche: ermöglicht eine Fläche zur Bearbeitung durch die folgenden Anweisungen im 3D-Raum auszuwählen

Scriptgesteuert (Scripting)

Das Draft-Modul bietet eine komplette Draft API, die Sie in Skripten und Makros nutzen können.
