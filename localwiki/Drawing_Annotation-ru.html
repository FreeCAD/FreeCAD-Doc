Описание

Эта команда позволяет поместить текстовый блок на чертёжный лист.

Использование

  1. Create a Drawing page
  2. Refresh the view if a Drawing view wasn't opened
  3. Press the Drawing Annotation button
  4. Adjust the desired properties, such as text contents, font, size and position.

Ограничения

  • Многострочные тексты не поддерживаются встроенным просмотрщиком SVG, базирующемся на Qt, но команда Открыть в браузере показывает эти тексты корректно.
