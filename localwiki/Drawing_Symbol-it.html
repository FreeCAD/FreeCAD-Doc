Descrizione

Questo comando consente di aggiungere il contenuto di un'immagine SVG in una pagina di disegno selezionata. Questo contenuto può poi essere spostato e riscalato sulla pagina. Il contenuto dell'immagine SVG viene copiato nel documento FreeCAD, quindi è indipendente dal file originale, e viene visualizzato nello stesso modo anche su un altro computer che non dispone del file SVG originale.

Utilizzo

  1. Creare una pagina di disegno
  2. Aggiornare la vista se il disegno non viene aperto
  3. Premere il pulsante Simbolo
  4. Selezionare un file SVG
  5. Regolare le proprietà necessarie, ad esempio, la posizione e la scala.
