Определение

Эта команда позволяет добавить содержимое файла SVG выбранный чертёжный лист. Это содержимое может перемещаться и масштабироваться на странице. Содержимое изображения SVG копируется внутрь документа FreeCAD, становясь независимым от оригинального файла, и изображается так же на других компьютерах, где нет оригинального файла SVG.

Использование

  1. Create a Drawing page
  2. Refresh the view if a Drawing viewer wasn't opened
  3. Press the Drawing Symbol button
  4. Select a SVG file
  5. Adjust the needed properties, like position and scale.
