Introduction

Dans ce tutoriel, nous allons découvrir comment exporter les fichiers de FreeCAD au formats STL/OBJ. Les formats de maillage STL/OBJ n'ont aucune dimension, l'exportation par FreeCAD supposera que l'unité utilisée dans le modèle est exprimée en mm. Si cela n'est pas le cas, vous devrez convertir votre projet en mm .

Faire un test

Vous pouvez utiliser votre propre fichier FreeCAD, mais vous pouvez aussi créer un fichier test

  • ouvrir FreeCAD
  • Créer un nouveau document
  • Activer l'atelier Part
  • Insérer un cube
  • Insérer un cône
  • Sélectionnez les deux objets dans la vue 3D
  • Créer une fusion avec les deux éléments
  • Sauver votre fichier

Exportation méthode 1: En utilisant "Fichier > Exportation"

  • Activer l'atelier Mesh Design ou tous les ateliers
  • Choisissez Édition → Préférences → Import-Export → Onglet Formats de maillage
  • Ici vous devez spécifier la déviation maximum du maillage (0,1 mm est la valeur par défaut) pour contrôler la finesse que vous désirez donner a votre maille
  • Fermez la fenêtre en cliquant sur le bouton OK
  • Dans la vue 3D sélectionnez le solide que vous désirez exporter
  • Puis cliquez sur Fichier → Exporter... et choisissez le type de fichier à exporter par exemple "STL mesh (*.stl *.ast)"
  • Tapez votre nom de fichier et ne pas l'extension, et cliquez sur Enregistrer

Exportation Méthode 2: En utilisant l'atelier maillage de FreeCAD

  • Activez l'atelier Maillage
  • Dans la vue 3D sélectionnez le solide à convertir en maille
  • Dans la barre de menus choisissez Maillages → Créer un maillage à partir d'une forme...
  • Dans la vue 3D sélectionnez un ou plusieurs maillages disponibles et spécifiez les options disponibles, pour plus d'informations référez vous à la page (Maillage vers Forme)
  • Cliquez sur le bouton OK et un objet Maillage dans la vue 3D (avec un icône maillage vert)
  • Sélectionnez l'objet maille dans la vue 3D et cliquez sur le bouton droit de la souris
  • Cliquez sur Exporter le maillage pour exporter l'objet
  • Vous pouvez donner un nom de fichier (par défaut le nom sera le "nom de l'objet (Meshed).stl") et le type de fichier ("STL Binaire (*.stl)")
  • Cliquez sur Enregistrer vous venez de terminer

Quelle méthode choisir ?

  • La méthode 2 doit être préférée. Surtout lorsque vous avez plus d'un corps à convertir, vous pouvez utiliser les outils de Mesh Workbench, comme la fusion de maillages avant l'exportation.
