Introduction

Cette page rassemble toutes les informations sur le solveur actuel utilisé dans le Module FEM de FreeCAD. Les interfaces entre les solveurs et FreeCAD en pre et post traitement sont aussi décrites. Pour le moment (Début 2016 et FreeCAD 0.16dev) le seul solveur supporté par le module FEM est [CalculiX]. Selon le système d'exploitation utilisé vous aurez besoin d'installer CalculiX d'abord. Page [FEM Install].

Interface

L'interaction entre le module FEM et CalculiX se fait au moyen de fichiers texte. Le module FEM écrit un fichier CalculiX, démarre CalculiX, enregistre les sorties de CalculiX et li les fichiers résultats s'ils sont disponible. L'outil Solver job control gère la totalité du processus. L'intervention de l'utilisateur dans le processus est possible.

Preprocessing interface

CalculiX utilise un fichier d'entrée au format abaqus. Le fichier d'entrée peut être édité avant le démarrage du solveur. Les unités utilisées dans CalculiX sont indépendantes des unités utilisées dans FreeCAD. Ce doit etre des mm et N. (A faire: vérifier. Qu'est ce qui arrive si un maillage en pouces est utilisé dans FreeCAD. La densité a été introduite. Avec cela nous avons des kg et s et non plus des N?! Que penser de ça ?! )

L'interface module FEM - CalculiX supporte les objects suivants:

FEM Elements
  • Tet4 et Tet10
  • S3 et S6
  • B31 et B32
  • voir FEM_Mesh_CalculiX
Analysis
  • linear static analysis
  • Frequency analysis
  • Coupled thermal structural analysis
Matériaux
  • 1 Matériau élastique linéaire et isotropique (uniformité dans toutes les directions)
  • Matériaux multiples en développement.
