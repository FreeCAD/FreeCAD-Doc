Description

Crée une contrainte FEM pour fixer une géométrie bloquée tous les 6 degrés de liberté sur l'objet sélectionné.

Utilisation

  1. Cliquez sur le bouton ou dans le menu Model → Mechanical Constraints → Constraint fixed sur la barre de menu.
  2. Dans la vue 3D sélectionnez l'objet à contraindre, l'objet peut être:
    1. sommets (coins)
    2. bords
    3. faces

Limitations

Vous ne pouvez pas mélanger les objets dans la même contrainte. Utilisez une contrainte pour chaque type d'objet.
