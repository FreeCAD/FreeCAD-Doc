Description

Applique une contrainte de pression sur une face.

Utilisation

  1. Cliquer sur le bouton ou dans la barre du menu Model → Mechanical Constraints → Créer une contrainte FEM de pression
  2. cliquez sur le bouton Ajouter une référence puis sélectionnez une face dans la vue 3D
  3. Donnez la pression dans la fenêtre d'édition
  4. Activez la case à cocher pour inverser la direction si nécessaire

Limitations

Notes

La pression sur la face est toujours uniforme et perpendiculaire à la face sélectionnée.
