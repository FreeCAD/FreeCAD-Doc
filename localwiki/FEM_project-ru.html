Мозговой штурм

Потенциальные генераторы сеток

Некоторые генераторы представлены здесь:

  • Netgen очень хороший генератор сеток, LGPL с поддержкой некоторых библиотек
  • Salome сложный пакет, да и из него трудно получить необходимый нам код!?
  • PythonOCC python библиотека оболочка позволяющая взаимодействовать Salome и генератору сеток Netgen, может быть самым быстрым и наиболее полным путем интеграции генераторов

Потенциальные решатели

Некоторые решатели представлены здесь:

  • Calculix
  • Code-Aster очень многофункциональный решатель с привязками к python
  • PythonOCC библиотека python которая предлагает (в настоящее время) грубый интерфейс для взаимодействия с Code_Aster, который также может быть быстрым способом получить работающее решение.
