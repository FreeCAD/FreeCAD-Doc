Progetti correlati a FreeCAD

  • FreeCAD repository and Draft module roadmap on Yorik's site. Yorik is working on the Draft module.
  • FreeCAD development page on Jacques-Antoine's site. Jacques-Antoine is working on a new Fold module.
  • FreeCAD Ship Design Module by Jose Luis Cercós Pita. È stato incorporato nel progetto principale di FreeCAD, ambiente di lavoro Ship.
  • Blender:FreeCAD project on Milovann's site
  • pythonOCC is a python binding collection for openCasCade, the 3D modeling kernel used by FreeCAD. With pythonOCC, the full power of OpenCasCade can be accessed and used from within FreeCAD.
  • David is a very low cost 3D scanning system that can work with FreeCAD
  • Architecture workflow : An example of how FreeCAD can begin to have its preliminary place in an architecture workflow...

Addons della comunità FreeCAD

Gli ambienti lavoro, script e addons sviluppati dalla comunità per FreeCAD sono disponibili presso

  • Addons Launchpad ppa for download. The contents of this ppa are not supported by FreeCAD
  • https://github.com/FreeCAD/FreeCAD-addons FreeCAD GitHub addons page. This github repository can also be used to access many of the above addons. Includes plugin manager by microelly2.

Addons include:

  • Animation Workbench
  • Assembly2 Workbench
  • Bolts
  • CADQuery
  • Drawing Dimensioning
  • EaglePCB2FreeCAD
  • Fasteners
  • fcgear
  • kicadstepuptools
  • Lattice
  • Lattice2
  • Parts Library
  • Plugin Manager
  • Sheetmetal
  • Symbols Library
  • Timber
  • Workfeature
