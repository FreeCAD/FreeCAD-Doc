FreeCAD Community addons

Community developed workbenches, scripts and addons for FreeCAd are available at

  • Addons Launchpad ppa for download. The contents of this ppa are not supported by FreeCAD
  • https://github.com/FreeCAD/FreeCAD-addons FreeCAD GitHub addons page. This github repository can also be used to access many of the above addons. Includes plugin manager by microelly2.

Addons include:

  • Animation Workbench
  • Assembly2 Workbench
  • Bolts
  • CADQuery
  • Drawing Dimensioning
  • EaglePCB2FreeCAD
  • Fasteners
  • fcgear
  • kicadstepuptools
  • Lattice
  • Lattice2

  • Plugin Manager
  • Sheetmetal
  • Symbols Library
