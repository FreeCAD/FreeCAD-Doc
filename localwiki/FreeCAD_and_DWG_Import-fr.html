Pourquoi je ne peux pas importer les fichiers DWG dans FreeCAD ?

Le format DWG est un format de fichier binaire fermé (propriétaire) qui n'est pas directement pris en charge par FreeCAD. Il nécessaire d'utiliser un convertisseur externe, ce convertisseur convertit d'abord le fichier en DXF pour être ensuite importé dans FreeCAD.

Veuillez noter qu'à l'heure actuelle, il est impossible d'importer des fichiers DWG 3D dans FreeCAD. Les données 3D sont intégrées dans le format binaire .SAT (ACIS), qui est propriétaire et non-documenté.

Que dois-je faire pour importer les fichiers DWG?

Teigha Converter

  • Page d'accueil: https://www.opendesign.com/guestfiles/TeighaFileConverter
  • licence: freeware
  • optionnel, utilisé pour permettre l'importation et l'exportation de fichiers DWG

Le Convertisseur Teigha est un petit utilitaire disponible gratuitement qui permet de convertir plusieurs versions de fichiers DWG et DXF. FreeCAD peut l'utiliser pour permettre l'importation et l'exportation DWG, en convertissant de manière transparente les fichiers DWG au format DXF , puis FreeCAD utilise son importateur DXF standard pour importer le contenu du fichier. Les restrictions de DXF importer s'appliquent.

Installation

L'installation se fait sur toutes les plate-formes en téléchargeant et en installant le logiciel TeighaFileConverter sur https://www.opendesign.com/guestfiles/TeighaFileConverter. Si après l'installation l'utilitaire n'est pas trouvé automatiquement par FreeCAD, vous pourriez avoir besoin de donner le chemin de l'exécutable du convertisseur manuellement, pour ce faire activez l'atelier Draft puis allez dans le menu Édition -> Préférences -> Importer / Exporter -> DWG et renseigner le chemin d’accès.

Pour plus de détails, voir ce tutoriel.

Utilisation

Le programme peut être utilisé en ligne de commande ou avec l'interface graphique. N'oubliez pas de convertir les fichiers DWG au format ASCII.

Commande en ligne Line Format is:

  1. Quoted Input Folder
  2. Quoted Output Folder
  3. Output_version {"ACAD9","ACAD10","ACAD12", "ACAD13","ACAD14", "ACAD2000","ACAD2004", "ACAD2007","ACAD2010"}
  4. Output File type {"DWG","DXF","DXB"}
  5. Recurse Input Folder {"0","1"}
  6. Audit each file {"0","1"}
  7. [optional] Extension de format du fichier (défaut:"*.DWG;*.DXF")

Exemple pour Linux
TeighaFileConverter "/home/DWG-data" "/home/DXF-data" "ACAD2010" "DXF" "0" "1" "test.dwg" Le deuxième numéro (audit) doit être 1 sinon il ne fonctionnera pas.

Example for Windows
"C:\Program Files\ODA\Teigha File Converter 3.08.2\TeighaFileConverter.exe" "Path-To-Input-Directory" "Path-To-Output-Directory" "ACAD2010" "DXF" "0" "1" "Name-Of-A-Test-File.dwg"

Quelles sont les Alternatives?

DraftSight

DraftSight est un programme commercial qui peut être utilisé librement (http://www.3ds.com/products-services/draftsight/overview/) et peut importer la plupart des fichiers DWG, peut sauver et exportater dans des formats qui sont utilisables par FreeCAD. Draftsight tourne sur

  • Windows 32 bit
  • Windows 64 bit
  • Mac OS X
  • Linux Fedora
  • Linux Ubuntu

Un enregistrement est requis pour activer et utiliser le programme.

DoubleCAD XT

Il y a aussi DoubleCAD XT (https://www.turbocad.com/content/doublecad-xt-v5). Ce programme est libre d'utilisation pour une utilisation personnelle ou commerciale. Vous devez vous enregistrer pour recevoir le code d'activation via E-Mail. Ce programme ne fonctionne que sous Windows.

Exportez vos fichiers AutoCAD dans un format reconnu de FreeCAD

Exportez votre fichier AutoCAD dans un des nombreux formats reconnu par FreeCAD, à savoir les versions DXF R12 ou R14, SVG et IGES sont supportées. Toutes ces versions utilisées par FreeCAD sont des alternatives au format DWG.

Il est important de savoir que, contrairement à ce que l'on croit, qu'il n'y a pas de différence entre le contenu d'un fichier enregistré dans les formats DWG ou DXF, à condition de travailler avec la même version (ex. DWG 2014 vs 2014 DXF). Les deux formats sont maintenus par Autodesk, et ils utilisent tous deux exactement les mêmes caractéristiques. La différence est que le fichier DWG est fermé (fichier codé) et que le fichier DXF est ouvert (fichier texte).

Que puis-je faire pour aider?

Promouvoir l'utilisation des formats alternatifs

Autrement dit, cessez d'accepter des travaux effectués au format DWG. Dans la pratique, c'est plus facile à dire qu'à faire. Pourtant, ce ne serait pas une mauvaise pratique pour les utilisateurs et les partisans de FreeCAD évitez d'utiliser le format DWG chaque fois que c'est possible.
