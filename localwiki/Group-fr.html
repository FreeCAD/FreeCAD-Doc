Description

Le Groupe est un moyen d'organiser les objets dans le document. Les objets à l'intérieur du groupe peuvent être activés ou désactivés en modifiant la visibilité du groupe. Les groupes peuvent être imbriqués, par ex. d'autres groupes peuvent être créés dans le groupe.

Utilisation

  1. Clic droit sur le document actif dans la vue combinée ou sur un groupe existant Tree View et sélectionnez Créer un Groupe...
  2. Cliquez à la souris sur l'objet et maintenez le puis déplacez le dans le groupe

Options

  • Pour renommer un groupe, sélectionnez le groupe et pressez sur la touche F2, ou faites clic droit et sélectionnez Renommer

Propriétés

  • DATALabel: Nom du groupe

Script

Cette commande ajoute un groupe dans le document actif: 

App.ActiveDocument.addObject("App::DocumentObjectGroup","Group")
