Description

Des utilisateurs avancés ont étendu les fonctionnalités de FreeCAD avec divers ateliers (workbenches) personnalisés, qui ne sont pas (encore !) intégrés dans le code source FreeCAD, mais ces ateliers sont facilement installables dans une installation FreeCAD existante. Ici, nous allons découvrir diverses méthodes d'installation pour les différents systèmes d'exploitation.

Note Starting from version 0.17, FreeCAD features an addons manager in the Tools menu, that allows to easily install a series of external workbenches. The instructions below are only necessary if you are running an earlier version, or if you want to install a workbench that is not in the list offered by the addons manager.

Installation dans Windows

Méthode d'installation recommandée

Utiliser la macro addons-installer from Github.
Pendant Google Summer of Code 2016 l'étudiant Mandeep Singh a travaillé sur une amélioration de la macro
la version améliorée (est disponible ici) mais cette version a besoin d'autres améliorations et mises au points avant de pouvoir être pleinement intégrée dans FreeCAD.

Méthode d'installation alternative “installation manuelle”

  • téléchargez l'atelier sur github en cliquant sur le bouton “clone or download” dans la page github (coins droit supérieur) et choisissez "Download ZIP"
  • dézippez l'archive téléchargée sur votre disque dur
  • dans FreeCAD, localisez le chemin des macros et choisissant “Edition > Préférences > Général > Macro” et localisez ”le chemin des macros”
  • supposed your Windows-Login is “User-Name” the default macro path is “C:\Users\User-Name\AppData\Roaming\FreeCAD\”
  • dans le répertoire des macros, créez (s'il n'est pas déjà présent) un répertoire nommé “Mod”
  • dans le répertoire Mod créez un répertoire nommé avec le nom de l'atelier téléchargé, par exemple “Assembly2”
  • maintenant déplacez le contenu di fichier dézippé dans le répertoire nouvellement créé (ici Assembly2)
  • après avoir redémarrer FreeCAD cliquez sur la boite déroulante d'ateliers et sélectionnez votre atelier

Recommandations pour mettre à jour un atelier additionnel

Sur Windows, lors de la mise à jour d'un atelier déjà installé, Windows conserve les anciens fichiers .py converti en .pyc.
Étant donné que cela peut entraîner des problèmes de compatibilité, il est recommandé de désinstaller l'atelier, de redémarrer FreeCAD et d'installer la nouvelle version de cet atelier.


Installation dans Linux (Ubuntu/Mint)

Méthode d'installation recommandée

Ajoutez community-ppa dans ppa-manager.
Installez l'atelier via synaptic packet manager.

Méthode d'installation alternative “Installation via Git” 

$ sudo apt-get install git python-numpy python-pyside
$ mkdir ~/.FreeCAD/Mod
$ cd ~/.FreeCAD/Mod
$ git clone https://github.com/hamish2014/FreeCAD_assembly2.git

Vous possédez maintenant un nouveau atelier dans FreeCAD appelé "Assembly 2". Après l'installation, utilisez git pour faire les mises à jours des dernières versions en utilisant BASH 

$ cd ~/.FreeCAD/Mod/FreeCAD_assembly2
$ git pull
$ rm *.pyc

Méthode d'installation alternative “installation manuelle”

  • téléchargez l'atelier sur github en cliquant sur le bouton “clone or download” dans la page github (coins droit supérieur) et choisissez "Download ZIP"
  • dézippez l'archive téléchargée sur votre disque dur
  • dans FreeCAD, localisez le chemin des macros et choisissant “Edition > Préférences > Général > Macro” et localisez ”le chemin des macros”
  • par défaut, le répertoire des macros (caché) est "/.FreeCAD" dans votre bureau
  • dans le répertoire des macros, créez (s'il n'est pas déjà présent) un répertoire nommé “Mod”
  • dans le répertoire Mod créez un répertoire nommé avec le nom de l'atelier téléchargé, par exemple “Assembly2”
  • maintenant déplacez le contenu di fichier dézippé dans le répertoire nouvellement créé (ici Assembly2)
  • après avoir redémarrer FreeCAD cliquez sur la boite déroulante d'ateliers et sélectionnez votre atelier

Installation sur Mac

Méthode d'installation recommandée


Méthode d'installation alternative' cet exemple décri la procédure à suivre avec l'atelier assembly

  • téléchargez le fichier zip dans le dépôt git
    • (Windows) assurez vous que FreeCAD est installé dans "/Programme (ou Programme file (x86) ) /FreeCAD v0.15 puis allez dans "C:\Users\nom utilisateur\AppData\Roaming\FreeCAD\Mod"
    • (Linux) assurez vous que FreeCAD est installé dans "/Applications/FreeCAD/v0.15", puis allez dans "/Applications/FreeCAD/v0.15" dans votre explorateur, sélectionnez FreeCAD.app
  • faites clic droit dans le répertoire et créez un dossier "assembly2"
  • dézippez le le fichier .zip téléchargé dans le dossier "assembly2"


Copier ou décompressez drawing dimensioning dans le dossier FreeCAD.app/Contents/Mod où FreeCAD est installé.

Dysfonctionnement généralités

  • N'utilisez pas de caractères spéciaux (par exemple: accentués, tréma...) dans votre nom utilisateur, sinon FreeCAD risque de ne pas reconnaître le juste chemin des macros
  • Si vous possédez un nom d'utilisateur de ce type, changez votre nom d'utilisateur sans accentués et autres caractères spéciaux
  • Vous ne voyez pas d'atelier dans la barre déroulante ? Sur la barre d'outil de FreeCAD, faites clic droit de la souris Personnaliser... > Atelier et assurez-vous qu'il soit présent dans la fenêtre de droite (ou Outils > Personnaliser... > Ateliers)
  • Notes pour les utilisateurs avec un système 32 bits et FreeCAD 0.16.6706. Après l'installation, les nouveaux ateliers peuvent ne pas être disponibles. Dans ce cas
    • ouvrez la Vue Rapport pendant le démarrage de FreeCAD et lisez l'erreur affichée
    • voir la discussion sur le forum: http://forum.freecadweb.org/viewtopic.php?t=12839#p102933
