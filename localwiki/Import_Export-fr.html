Import Export/fr

Cette page rassemble les différents formats de fichiers qui peuvent être importés ou exportés par FreeCAD. La plupart de ces formats de fichiers sont importés par un module spécifique. Ce module n'a pas besoin d'être chargé pour importer ou exporter ces formats, mais doit être chargé pour afficher la page des préférences correspondant au format.
