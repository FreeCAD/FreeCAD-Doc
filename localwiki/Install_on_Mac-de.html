Install on Mac/de

FreeCAD kann in in einem Schritt auf Mac OS X installiert werden, unter Verwendung eines Installationswerkzeuges (Installer).

Mac 10.9 Mavericks 64-bit


Diese Seite beschreibt die Benutzung und die Eigenschaften des FreeCAD-Installierers. Auch Anweisungen zum Deinstallieren sind vorhanden. Sobald es installiert ist starte!

Einfache Installation

Der FreeCAD-Installierer wird bereitgestellt als ein Installationspaket (.mpkg) in einer Disk-Image-Datei eingebettet.

Die letzte Version des FreeCAD-Installierers erhält man auf der Download Seite. Nach dem Herunterladen der Datei, einfach das Disk-Image einhängen (mounten). Dann in den Anwendungsordner oder einen Ordner Ihrer Wahl ziehen.

Das war es. Einfach auf die Anwendung (App) klicken, um FreeCAD zu starten. Wenn eine Meldung erscheint, dass FreeCAD nicht geöffnet werden, kann, weil es von einem nicht identifizierbaren Entwickler stammt, dann den Ordner "Anwendung" (Application) öffnen, die Anwendung (App) mit der rechten Maustaste anklicken und das Öffnen der Anwendung akzeptieren.


De-Installation

Momentan gibt es keinen De-Installierer für FreeCAD. Um FreeCAD und alle installierten Komponenten zu entfernen, ziehen Sie die folgenden Dateien und Ordner in den Papierkorb:

  • In /Applications:
    • FreeCAD
