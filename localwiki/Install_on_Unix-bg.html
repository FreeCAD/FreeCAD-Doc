Install on Unix/bg

Инсталацията на FreeCAD на повечето известни linux дистрибуции се поддържа от самата дистрибуция. Тоест FreeCAD се инсталира директно от мениджъра на пакетите (package manager) на вашата дистрибуция. Общността на FreeCAD също предлага няколко "официални" пакета когато нови версии на програмата излизат, както и няколко експериментални PPA repositories за тестване на най-новите промени в програмата.

Ubuntu and Ubuntu-базирани дистрибуции

Повечето Linux дистрибуции базирани на Ubuntu използват същите канали за пакети. Освен официалните варианти (Kubuntu, Lubuntu and Xubuntu), това включва неофициални дистрибуции като Linux Mint, Voyager и други. Следващите инструкции важат за всички тези дистрибуции:

Официална Ubuntu repository

FreeCAD е в официалните Ubuntu repositories и може да се инсталира в Software Center или чрез следната команда в терминал: 

sudo apt-get install freecad

Вероятно е тази версия да е по-стара и да няма най-новите възможности на програмата. 

sudo apt-get install freecad

But chances are this stable version will be outdated, and not have the latest features. To get the last release version, please use the PPA.

Последната стабилна версия от "stable releases" PPA или "daily" PPA

Общността на FreeCAD поддържа PPA repository на Launchpad с най-новата "стабилна" FreeCAD версия. Има и още по-нова "daily" PPA repository на Launchpad която автоматично се обновява всеки ден с прясно компилирана версия от master branch-a на сорс-дървото на FreeCAD. Тази версия често съдържа много bug fixes и feature updates.

PPA repository on Launchpad with the latest stable FreeCAD version.

Добавете следните PPA към източниците на пакети (Software Sources) на вашата система (вижте What are PPAs and how do I use them? за инструкции ако не знаете как): 

ppa:freecad-maintainers/freecad-stable

Когато излезе диалог за обновяване на източниците на пакети натиснете ОК.

Сега можете да инсталирате FreeCAD и неговата документация през Ubuntu Software Center или чрез мениджърът на пакети който предпочитате.

Definition of PPA (Personal Package Archive) on help.ubuntu.com.

Инсталация през конзолата

Напишете (или копирайте) следните команди в конзолата за да добавите PPA и да инсталирате FreeCAD заедно с документацията му:

За последната стабилна версия ("stable" PPA) 

sudo add-apt-repository ppa:freecad-maintainers/freecad-stable

Don't forget to retrieve the updated the package lists: 

sudo apt-get update

Update synchronizes your list of available packages with the servers.

След това: 

sudo apt-get install freecad freecad-doc && sudo apt-get upgrade

Upgrade downloads and installs the newest packages version of the installed programs. It will be applied to all the softwares installed on your computer.

Execute this command to open the stable version of FreeCAD: 

freecad

Нестабилна версия на FreeCAD

If you want to install the last unstable version of FreeCAD, please use the PPA named "freecad-daily" PPA repository on Launchpad. This way you can access the bleeding edge of FreeCAD development. This PPA is automatically compiled daily from the official FreeCAD's master branch repository. It will usually contain numerous bug fixes and feature updates.

За последната всекидневна версия ("daily" PPA) 

sudo add-apt-repository ppa:freecad-maintainers/freecad-daily
sudo apt-get update
sudo apt-get install freecad-daily && sudo apt-get upgrade

Execute this command to open the daily build of FreeCAD: 

freecad-daily

Debian и други debian-базирани дистрибуции

След Debian Lenny, FreeCAD фигурира директно в software repositories на Debian и може да се инсталира чрез synaptic или просто чрез: 

sudo apt-get install freecad

OpenSUSE

FreeCAD обикновенно се инсталира чрез: 

zypper install FreeCAD

Gentoo

FreeCAD може да се компилира/инсталира с командата: 

emerge freecad

Други

Ако вашата дистрибуция поддържа инсталация на FreeCAD но не е документирана на тази страница, моля кажете ни на нашия форум тук!

Има алтернативни неофициални FreeCAD пакети в интернет, например за системи като slackware и fedora. Можете да ги открите лесно с търсачка.

Ръчна инсталация на .deb базирани дистрибуции

Ако по някаква причина не можете да ползвате гореописаните методи може да свалите .deb пакетите на страницата Download/bg.
Template:DownloadUnixStable/bg

След като свалите .deb файлът за вашата система ако имате Gdebi инсталиран(обикновенно го има), можете просто да отворите папката където е свален .deb файлът и да кликнете 2 пъти върхи него. Всички дурги нужни пакети ще бъдат автоматично инсталирани от вашия мениджър на пакети. Алтернативно може да инсталирате чрез терминала като отидете отново в директорията където е .deb файлът и напишете: 

sudo dpkg -i Name_of_your_FreeCAD_package.deb

където трябва да смените Name_of_your_FreeCAD_package.deb с името на файла който сте свалили.

След инсталацията на FreeCAD нова икона ще се добави в секцията "Graphic" на вашето старт меню (ако вашата дистрибуция има такова)
