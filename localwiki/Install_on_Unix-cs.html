Instalace FreeCADu na většinu známých Linuxových systémů zatím byla komunitou schválena a FreeCAD by měl být přímo dostupný v manažeru balíčků na Vaší distribuci. Jestliže jste FreeCAD nainstalovali, je čas začít!

Ubuntu a na Ubuntu založené systémy

Mnoho Linuxových distribucí je založeno na Ubuntu a sdílí jeho repozitáře. Kormě oficiálních variant (Kubuntu, Lubuntu, Xubuntu) existují ještě neoficiální distra jako je Linux Mint, Voyager a další. Dále uvedené volby by měly být kompatibilní s těmito systémy.

Oficiální repozitář Ubuntu

FreeCAD je dostupný z repozitáře Ubuntu a může být nainstalován z Centa Softwaru nebo následujícím příkazem z terminálu: 

sudo apt-get install freecad

Ale je dost velká pravděpodobnost, že taková verze bude už zastaralá a nebude mít nejnovější vlastnosti. 

sudo apt-get install freecad

But chances are this stable version will be outdated, and not have the latest features. To get the last release version, please use the PPA.

Nejnovější stabilní verze z PPA

Komunita FreeCADu poskytuje PPA repozitář na Launchpadu s poslední stabilní verzí FreeCADu.

PPA repository on Launchpad with the latest stable FreeCAD version.

Do Software Sources přidejte následující PPA (čtěte Co jsou PPAs a jak je mám použít? nevíte-li jak): 

ppa:freecad-maintainers/freecad-stable

Když Vás dialogové okno bude žádat o refresh (obnovení) zdroje softwaru, klikněte OK.

Nyní můžete nainstalovat FreeCAD a dokumentaci FreeCADu prostřednictvím Ubuntu Software Center nebo pomocí balíčkového manažera, který jste zvyklí používat.

Definition of PPA (Personal Package Archive) on help.ubuntu.com.

Instalace z konzoly

Zapište (nebo překopírujte) následující příkazy do konzoly pro přidání PPA a instalaci FreecADu společně s dokumentací:

For the "stable" PPA 

sudo add-apt-repository ppa:freecad-maintainers/freecad-stable

Don't forget to retrieve the updated the package lists: 

sudo apt-get update

Update synchronizes your list of available packages with the servers.

Potom: 

sudo apt-get install freecad freecad-doc && sudo apt-get upgrade

Upgrade downloads and installs the newest packages version of the installed programs. It will be applied to all the softwares installed on your computer.

Execute this command to open the stable version of FreeCAD: 

freecad

Nestabilní verze FreeCADu

If you want to install the last unstable version of FreeCAD, please use the PPA named "freecad-daily" PPA repository on Launchpad. This way you can access the bleeding edge of FreeCAD development. This PPA is automatically compiled daily from the official FreeCAD's master branch repository. It will usually contain numerous bug fixes and feature updates.

For the daily PPA: 

sudo add-apt-repository ppa:freecad-maintainers/freecad-daily
sudo apt-get update
sudo apt-get install freecad-daily && sudo apt-get upgrade

Execute this command to open the daily build of FreeCAD: 

freecad-daily

Debian a další systémy založené na Debianu

Od verze Debianu Lenny je FreeCAD dostupný přímo ze softwarového depozitáře Debianu a může být instalován pomocí Synapticu nebo jednoduše: 

sudo apt-get install freecad

OpenSUSE

FreeCAD je typicky instalován následovně: 

zypper install FreeCAD

Gentoo

FreeCAD může být nainstalován jednoduše zadáním: 

emerge freecad

Další

Pokud zjistíte, že Váš systém by byl vhodný pro FreeCAD, ale není uveden v této stránce, dejte nám prosím vědět na fóru!

Mnoho alternativních, neoficiálních balíčků FreeCADu je dostupných na netu, například pro systémy Slackware nebo Fedora. Hledání na netu Vám rychle dá nějaké výsledky.

Ruční instalace na .deb systémech

Jestliže z nějakých důvodů nemůžete použít výše uvedené metody, můžete ještě pořád stáhnout jeden z .deb balíčků dostupných na stránce Download page.

Ubuntu 32/64bit AppImage 64bit

Když jste stáhli .deb balíček korespondující s Vaší verzí systému a máte nainstalován balíček Gdebi (většinou už tam je), stačí v něm najít stažený balíček .deb a dvojkliknout na něj. Potřebné závislosti budou zpracovány automaticky systémovým manažerem balíčků. Alternativně můžete instalovat i z terminálu, kdy najdete stažený .deb balíček a zadáte: 

sudo dpkg -i Name_of_your_FreeCAD_package.deb

název Name_of_your_FreeCAD_package.deb nahradíte názvem staženého .deb balíčku.

Po nainstalování FreeCADu bude do sekce "Grafika" ve spouštěcím menu přidána ikona pro spuštění FreeCADu.
