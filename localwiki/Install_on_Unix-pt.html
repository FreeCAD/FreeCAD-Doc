Ubuntu e sistemas baseados em Ubuntu

Muitas distribuições do Linux são baseadas no Ubuntu e partilham os mesmos repositórios. Além existem as variantes oficiais (Kubuntu, Lubuntu e Xubuntu), e as distribuições não oficiais como Linux Mint, Voyager e outros. As opções de instalação em baixo devem ser compatíveis com todos estes sistemas.

Repositório oficial Ubuntu

O FreeCAD está disponível a partir dos repositórios oficiais do Ubuntu e pode ser instalado via Centro de Software ou através deste comando num terminal: 

sudo apt-get install freecad

Mas o mais provável é que esta versão esteja desatualizada, e não tenha as funcionalidades mais recentes. 

sudo apt-get install freecad

But chances are this stable version will be outdated, and not have the latest features. To get the last release version, please use the PPA.

Última Versão estável a partir de "stable releases" PPA ou "daily" PPA

A comunidade FreeCAD disponibiliza o repositório PPA no Launchpad com a última versão "estável" do FreeCAD. Existe também uma versão mais atualizada "diáriamente" repositório PPA no Launchpad diáriamente compilado automáticamente a partir do ramo master do repositório oficial do FreeCAD, que normalmente contém inumeras correções de "bugs" e funcionalidades atualizadas.

PPA repository on Launchpad with the latest stable FreeCAD version.

Adicione às fontes de software do seu sistema o seguinte PPA (leia O que são PPAs e como os posso usar? se ainda não souber como): 

ppa:freecad-maintainers/freecad-stable

Quando uma caixa de dialogo lhe pedir para atualizar as suas fontes de software, clique em OK.

Agora pode instalar o FreeCAD e a documentação do FreeCAD através do Centro de software do Ubuntu, ou pelo gestor de pacotes à da sua escolha.

Definition of PPA (Personal Package Archive) on help.ubuntu.com.

Instalação pela linha de comandos

Escreva (ou copie e cole) estes comandos num terminal para adicionar o PPA e instalar o FreeCAD em conjunto com a documentação:

For the "stable" PPA 

sudo add-apt-repository ppa:freecad-maintainers/freecad-stable

Don't forget to retrieve the updated the package lists: 

sudo apt-get update

Update synchronizes your list of available packages with the servers.

Depois: 

sudo apt-get install freecad freecad-doc && sudo apt-get upgrade

Upgrade downloads and installs the newest packages version of the installed programs. It will be applied to all the softwares installed on your computer.

Execute this command to open the stable version of FreeCAD: 

freecad

Versão instável do FreeCAD

If you want to install the last unstable version of FreeCAD, please use the PPA named "freecad-daily" PPA repository on Launchpad. This way you can access the bleeding edge of FreeCAD development. This PPA is automatically compiled daily from the official FreeCAD's master branch repository. It will usually contain numerous bug fixes and feature updates.

For the daily PPA: 

sudo add-apt-repository ppa:freecad-maintainers/freecad-daily
sudo apt-get update
sudo apt-get install freecad-daily && sudo apt-get upgrade

Execute this command to open the daily build of FreeCAD: 

freecad-daily

Debian e outros sistemas baseados em debian

Desde a versão Lenny do Debian, o FreeCAD está disponível diretamente a partir dos repositórios de software Debian e pode ser instalado via Synaptic ou simplesmente com: 

sudo apt-get install freecad

OpenSUSE

FreeCAD é normalmente instalado com: 

zypper install FreeCAD

Gentoo

O FreeCAD pode ser compilado/instalado fazendo: 

emerge freecad

Outros

Se descobrir que o seu sistema disponibiliza o FreeCAD mas não está documentado nesta página, por favor avise-nos no forum!

Há muitos pacotes do FreeCAD alternativos, não oficiais, disponíveis na net, por exemplo para sistemas slackware ou fedora. Uma busca na net pode fornecer-lhe alguns resultados rápidamente.

Instalação manual em sistemas baseados em .deb

Se por alguma razão não puder usar um dos métodos descritos acima, pode sempre transferir um dos pacotes .deb disponíveis na página das transferências. Ubuntu 32/64bit AppImage 64bit


Assim que tiver o .deb correspondente à versão do seu sistema, se tiver o pacote Gdebi instalado (normalmente está), precisa simplesmente de ir à pasta para onde fez a transferência, e fazer duplo-clique no ficheiro. As dependências necessárias serão resolvidas automaticamente pelo gestor de software do seu sistema. Alternativamente pode também instalá-lo pelo terminal, navegando para a pasta do ficheiro, e escrevendo: 

sudo dpkg -i Name_of_your_FreeCAD_package.deb

mudando Nome_do_seu_pacote_FreeCAD.deb pelo nome do ficheiro de que transferiu.

Depois de instalar o FreeCAD, um atalho aparecerá na secção "Gráficos" do seu menu Iniciar.

Instalação noutros sistemas Linux/Unix

Infelizmente, atualmente, não existe uma distribuição disponível para outros sistemas Linux/Unix, portanto terá de compilar você mesmo o FreeCAD.
