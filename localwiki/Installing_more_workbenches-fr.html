Installing more workbenches/fr

En dehors des ateliers standard fournit dans chaque installation de FreeCAD, il est très facile d'ajouter d'autre ateliers qui étendent les fonctionnalités de FreeCAD. Pour l'instant, tant que FreeCAD est jeune, il n'y a pratiquement pas d'autre ateliers disponible. Mais FreeCAD est extensible depuis le début donc ajouter des nouveaux ateliers est très facile, et dans le futur de nouveaux ateliers seront disponible.

Les ateliers supplémentaires sont livré dans un dossier, qui doit être simplement copié dans le dossier Mod de FreeCAD. Le dossier Mod est scanné à chaque lancement de FreeCAD et les ateliers disponible sont automatiquement ajoutés. Il y a deux façons de les installer :

Installer dans le système

Les ateliers ou modules supplémentaires, peuvent être installé dans votre dossier d’installation de FreeCAD de manière à être disponible pour tout les utilisateurs. Selon votre système d'exploitation vous pourriez avoir besoin des droits administrateur ou autre privilèges pour accéder au dossier d'installation de FreeCAD.

Copier ou déplacer le dossier contenant votre ateliers dans le dossier Mod qui se trouve dans le dossier d'installation de FreeCAD. Sur Windows le dossier se trouve dans un endroit comme ceci : C:\Program Files\FreeCAD\Mod, sur un système basé sur Debian dans en endroit comme ceci : /usr/lib/FreeCAD/Mod

Installation dans le système utilisateur

Les ateliers installés de la sorte ne seront disponible que pour l'utilisateur propriétaire de ce dossier. Mais cela ne demande pas de privilèges administrateur.

Copier ou déplacer le dossier contenant le nouvel atelier dans le dossier Mod de votre dossier utilisateur FreeCAD. Sur Windows cela devrait être quelque chose comme : C:\Program Files\FreeCAD\Mod, sur Linux : ~/.FreeCAD/Mod où ~ est votre dossier home. Sur MAC OSX, le dossier devrait être ~/Library/Preferences/FreeCAD/Mod. Une manière de trouver le dossier préférences sur Mac est d'utiliser la fonction Go->Go to Folder... sous le menu "Finder", puis entrer "~/Library/Preferences/FreeCAD".


Plus d'informations pour créer votre propre atelier sont disponible sur Developer hub/fr et sur Power users hub/fr.
