Otros proyectos de CAD

  • OpenPLM Una plataforma de PLM basada en la web de código libre
  • BRL CAD Gran paquete de código libre, algo anticuado
  • Heekscad Una aplicación de CAD/CAM basada en OpenCasCade y WxWidgets
  • Narocad Otro proyecto basado en OpenCasCade, sólo para Windows
  • Salome Es más un sistema FEM pero también tiene una parte de modelado
  • PythonOCC Un enlace completo en Python para OpenCasCade, con constructor de interfaces gráficas de usuario GUI propia
  • PythonCAD Una simple aplicación de CAD 2D programada en Python
  • LibreCAD un proyecto de CAD 2D basado en QCad
