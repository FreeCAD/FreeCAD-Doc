BOLTS est un projet de construction d'une bibliothèque de pièces standard libre et open-source pour les applications de CAO. Le projet fourni une base de données commune qui peut être utilisée par plusieurs applications CAO et s'appuie sur la puissance paramétrique de FreeCAD pour créer plusieurs variantes et différentes tailles d'une pièce.

BOLTS pour FreeCAD offre un widget qui permet d'insérer des pièces dans le document actif. Les tailles et modèles d'une pièce peuvent être sélectionnés dans le widget, et affiche une étiquette descriptive de la pièce.

The BOLTS widget and a bearing from BOLTS


Plus d'informations sur BOLTS se trouve sur la web page.

La page Web fournit également une liste de toutes les pièces qui comprennent, les dessins et tableaux de dimensions.

note: BOLTS fait aussi partie des addons FreeCAD et peut être installé depuis la macro d'installation des addons.
