Beschreibung

Dieses Makro zeigt nach der Angabe von drei Punkten oder einer kreisförmigen Kante den berechneten Mittelpunkt und den berechnten Radius eines Kreises an. Eine Linie wird vom Mittelpunkt zum ersten Punkt auf dem Kreis gezeichnet, Sie kann für sprätere Messungen verwendet werden.


Gebrauch

Kpoiere das Makro in den Makro-Zielpfad und starte es (siehe How to install macros für weitere Details).

Wähle die drei Punkte und das Ergebnis wird in dem "Report view" angezeigt. Kanten können auch ausgewählt werden und zählen als zwei Punkte. Eine kreisförmige Kante kann auch ausgewählt werden. Wenn zwei Kanten ausgewählt wurden, wird der Enpunkt der zweiten Kante in der Berechnung nicht berücksichtigt.

Man kann entweder die Elemente auswählen und dann das Makro starten oder das Makro ohne Auswahl starten und die Elemente nach dem Start des Makros auswählen. Wenn nicht genug Elemente ausgewählt werden bevor das Makro gestartet wird, wird man aufgefordert, die fehlen Elemente auszuwählen.
