Description

Cette macro vous permet de connaître les informations de volume, surface, centre de gravité et le moment d'inertie de l'objet sélectionné

.

Installation

Si vous êtes sous Linux, vous devez créer un dossier nommé "Mod" dans le dossier caché .FreeCAD qui se trouve dans votre dossier personnel. Ensuite, créez un dossier nommé "Info" dans le dossier "Mod", et extraire le contenu de l'archive dans le dossier "Info". Sous Windows faites la même procédure dans le répertoire C:\Program Files\FreeCAD\Mod.

Utilisation

Lancez FreeCAD, ouvrez votre fichier STEP et passer à l'atelier "Info" avec le commutateur établi ou en allant sur le menu Affichage -> Workbench. Maintenant, sélectionnez votre solide, et cliquez sur l'icône "Info"; la barre des tâches de gauche indiquera des informations sur le modèle, y compris le volume, surface, centre de masse et du moment d'inertie.
