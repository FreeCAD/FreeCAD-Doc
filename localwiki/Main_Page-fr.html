Introduction

Ceci est la documentation wiki de FreeCAD. L'information contenue ici est ce qui constitue la documentation incorporée dans le programme FreeCAD. Vous avez deux manières de parcourir la documentation: en naviguant dans la section pour les utilisateurs, ou en suivant le manuel. FreeCAD est un projet en perpétuel évolution, écrit par une communauté d'utilisateurs et de développeurs. N'hésitez pas à nous aider en contribuant si vous trouvez qu'il manque des informations ou qu'elles sont incorrectes.


Contenu

Pour les utilisateurs : Voici les trois points d'entrée principaux pour la documentation, pour les trois grandes catégories d'utilisateurs de FreeCAD : Explique comment installer et utiliser l'application FreeCAD et détaille le contenu des différents ateliers.
Pour les utilisateurs avancés : Cette page explique comment utiliser les macros et scripts python pour améliorer votre production.
Pour les développeurs : Cette section explique comment écrire du code pour l'application FreeCAD, l'étendre ou l'utiliser dans vos propres applications.


Manuel

Le manuel Freecad est une autre façon, plus linéaire, de présenter l'information contenue dans ce wiki. Il est prévu pour être lu comme un livre, et constitue une introduction aux nombreuses autres pages des modules mentionnés ci-dessus. ebook versions (en anglais) est également disponible.

Comment participer

Il y a beaucoup de travail à faire dans le projet FreeCAD. Si vous êtes intéressé pour nous aider, il y a des tâches de programmation en C++ ou en python pour les développeurs, mais il y a aussi des choses que vous pouvez faire même si vous ne savez pas programmer

  • telles que la rédaction de la documentation
  • l'aide aux nouveaux arrivants,
  • la traduction de l'interface de l'application et de la documentation
  • la préparation d'installeurs ou de paquets de la dernière version de FreeCAD pour votre système d'exploitation favori
  • aider d'autres personnes et faire découvrir FreeCAD autour de vous.

La page aider FreeCAD décrit tout cela avec plus de détails. Depuis 2016, FreeCAD participe aussi au Google Summer of Code. Le Carrefour des contributeurs est une autre initiative pour rassembler les façons possibles d'aider et contribuer au projet FreeCAD.

Code Source

FreeCAD peut être compilé en utilisant cMake. Le lien git du code source est sous licence LGPL et déposé sur GitHub. Ici les instructions de compilation pour Windows, Unix/Linux et MacOSX. Le code source de la documentation est sur le site freecadweb, genéré par Doxygen, et documentée sur le wiki.

À propos du développement

Consultez la feuille de route du développement pour être au courant de ce qui est planifié, le journal des changements et la feuille de route sur le système de suivi de bogues pour observer les progrès destinés à la prochaine version, ou les statistiques de projet pour de plus amples informations sur le code de FreeCAD. Toutes les communications sur le développement se déroulent sur le forum, n'oubliez pas de le visiter si vous souhaitez participer.
