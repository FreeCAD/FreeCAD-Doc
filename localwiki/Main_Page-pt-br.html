Main Page/pt-br


Esta é a documentação wiki de FreeCAD. A informação aqui contida é a que forma a documentação offline fornecida para o FreeCAD. Existem duas maneiras principais para navegar nesta documentação: explorando os tópicos ou percorrendo o manual. Este é um trabalho orgânico, em andamento, escrito pela comunidade de usuários e desenvolvedores do FreeCAD. Se encontrar erros ou informação faltante aqui, por favor colabore!


Tópicos de usuários

Para usuários: Esta página contém documentação útil para usuários do FreeCAD em geral: Uma lista de bancadas, instruções detalhadas para instalar e usar o FreeCAD, tutoriais, e tudo que você precisa para começar.
Para usuários avançados: Esta página junta a documentação para usuários avançados e pessoas interessadas em escrever scripts em python. Você achará ali também uma coleção de macros, instruções para instalar e usá-las, e mais informação sobre como customizar o FreeCAD.
Para desenvolvedores: Esta seção contém materiais para programadores: Como compilar o FreeCAD você mesmo, como o código fonte é estruturado e como se achar nele, como desenvolver novas bancadas e inserir o FreeCAD dentro da sua aplicação.

Participe

Como participar

Há muito para ser feito dentro do projeto FreeCAD, caso esteja interessado em nos ajudar. Há atividades de programação, para programadores em C++ ou Python, mas também há muitas coisas que podem ser feitas sem precisar saber programar, como escrever documentação, ajudar os novos usuários, traduzir a aplicação e a documentação, ajudar com o empacotamento da última versão do FreeCAD para seu sistema operacional preferido ou, simplesmente, ajudar para que outras pessoas descubram o FreeCAD. A página de Ajuda do FreeCAD descreve tudo isso com mais detalhes.

The help FreeCAD page describes it all with more details. Starting from 2016, FreeCAD is also participating to the Google Summer of Code. The Contributors hub page is another effort to gather the possible ways to help and contribute to the FreeCAD project.

Código fonte

O FreeCAD pode ser compilado utilizando o cMake ou o autotools, o link do Git é https://github.com/FreeCAD/FreeCAD e há instruções para Windows, Unix/Linux e MacOSX.

Sobre o desenvolvimento

Visite o Mapa de desenvolvimento para ver notícias sobre o que está sendo planejado, o Histórico de mudanças e a roteiro das páginas de mantis para ver o progresso para a próxima versão, ou as Estatísticas do projeto para ter ainda mais informação sobre o código de FreeCAD. Todas as comunicações sobre o desenvolvimento se realizam no fórum de discussão, não deixe de visitá-lo se estiver interessado em participar.
