Material editor/fr

L'éditeur de matériaux alloue, édite ou sauve les informations contenues dans FreeCAD material. Le module matériaux est aussi utilisé par les ateliers FEM et Arch

L'éditeur de matériau est accessible par:

  • Le bouton Éditeur de matériaux de nouveau matériau et le panneau de creation de Arch Workbench
  • Via python:
 import MaterialEditor
 MaterialEditor.openEditor()

Options

  • Browser button: Ouvre l'url des propriétés dans un navigateur web
  • Material card: Choix d'un champs prédeterminé
  • Open: Ouvre un fchier .FCMat
  • Save as: Sauve le contenu de l'éditeur dans un nouveau fichier .FCMat file
  • Preview: Pas encore implémenté
  • Properties editor: Alloue ou édite le contenu des propriétés du matériau
  • Add property: Alloue ou ajoute une nouvelle propriété
  • Delete property: Efface les propriétés du matériau

Note:

  • Les boutons OK et Cancel ont le même effet dans l'éditeur de matériaux et ne sont pas utilisés directement dans les propriétés du matériau de l'objet existant.
