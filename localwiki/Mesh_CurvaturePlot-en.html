Description

Creates a colored 3d-plot with information about the deviation of the faces in relation to their neighbouring faces.

Use

  1. Select the mesh you would like to make a curvature plot from in the tree view.
  2. Click on or choose Meshes → Curvature plot from the top menu.
  3. The plot will be created as a seperate new object in the tree view.
  4. Hide all other objects but the curvature plot to make the plot visible.
