Introduction

This tool allows you to export a mesh object as file *.STL, *.AST, *.BMS, *.OBJ, *.OFF, *.iv, *.PLY from the current document.

Use

  1. Select mesh object to be exported
  2. Click on or choose Mesh → Export Mesh... from the top menu.
