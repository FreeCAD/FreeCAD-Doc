Einleitung

Dieses Tool erlaubt es eine Datei der Formate : *.STL, *.AST, *.BMS, *.OBJ, *.OFF, *.iv, *.PLY zum aktuellen Dokument hinzu zu fügen.

Benutzung

  1. Klicke auf oder wähle Mesh → Netz importieren... aus dem Menü.
  2. Wähle die zu importierende Datei
