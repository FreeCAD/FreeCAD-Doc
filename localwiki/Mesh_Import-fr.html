Introduction

This tool allows you to add a file *.STL, *.AST, *.BMS, *.OBJ, *.OFF, *.iv, *.PLY to the current document.

Use

  1. Click on or choose Mesh → Import Mesh... from the top menu.
  2. Select the file to be imported
