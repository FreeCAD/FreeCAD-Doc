Introduzione

Questo strumento consente di aggiungere un file *.STL, *.AST, *.BMS, *.OBJ, *.OFF, *.iv, *.PLY al documento corrente.

Uso

  1. Cliccare su o scegliere Mesh → Importa mesh... dal menu principale.
  2. Selezionare il file da importare.
