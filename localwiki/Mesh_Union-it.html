Introduzione

Esegue una unione (fusione) delle mesh selezionate.

Il comando crea una mesh nuova e indipendente composta da tutte le facce esterne di tutte le mesh selezionate.

Tutte le facce interne vengono rimosse.

Uso

  1. Selezionare le mesh da fondere nella vista ad albero o nella vista 3D.
  2. Scegliere Mesh → Booleane → Unione


Esempio: un cubo fuso con una sfera

Limitazioni

Note

se non si vogliono rimuovere le facce interne, si può usare Unisci
