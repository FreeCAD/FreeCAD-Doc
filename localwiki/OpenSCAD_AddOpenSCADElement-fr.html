Description

Ajoutez un élément OpenSCAD en entrant le code OpenSCAD dans le panneau des tâches, puis exécuter le binaire de OpenSCAD (il faut que OpenSCAD soit installé sur votre système)

Lorsque mesh est sélectionné, OpenSCAD retourne un maillage.

Chaque fois que vous appuyez sur Ajouter, le code OpenSCAD est exécuté, et les éléments sont importés.

Si OpenSCAD est exécuté avec succès, ses messages sont affichés en tant qu'avertissements dans la fenêtre de rapport. Ce sera aussi le cas si le chemin d'accès vers les fichiers importés, inclus et utilisés sont corrompus.

Dans le cas de dysfonctionnement, il est fortement recommandé de jeter un coup d'œil dans la fenêtre de rapport, car il pourrait y avoir d'autre affichages, créés par l'importateur.

Si OpenSCAD échoue, ces messages seront affichés comme des erreurs.

Les bibliothèques devraient être accessibles comme d'habitude, alors que l'exemple est accessible comme indiqué ci-dessous 

include <../examples/example001.scad>;

inclurait le fichier exemple001.scad également connu sous le nom d'icône OpenSCAD

Set up au sein de FreeCAD

  • OpenSCAD doit être installé sur votre ordinateur avant de pouvoir disposer de la fonctionnalité.
    • installer OpenSCAD de la façon appropriée pour votre système d'exploitation. Pour plus d'informations, consultez le site web OpenSCAD
  • FreeCAD a besoin de savoir où se trouve l'exécutable OpenSCAD
    • Exécutez FreeCAD et allez dans le menu Outils, sélectionnez Éditer paramètres...
    • A la gauche de la fenêtre allez dans Préférences > Mod > OpenSCAD. Cliquez sur OpenSCAD.
    • Puis cliquez sur le bouton droit dans la fenêtre de droite, et sélectionnez "Nouvel article chaîne",
    • une boîte de dialogue s'affiche, et entrez le texte openscadexecutable,
    • la boîte de dialogue va maintenant vous demander votre texte, entrez le chemin vers l'exécutable OpenSCAD (e.g. Ubuntu based Linux distributions /usr/bin/openscad)
    • Fermez et redémarrez FreeCAD, une nouvelle icône OpenSCAD apparaîtra sur la barre d'outils, puis dans le menu de OpenSCAD, dans le workbench de FreeCAD OpenSCAD
  • Il est également possible d'ajouter un autre paramètre optionnel, qui contrôle le nombre de côtes maximum d'un polygone considéré comme un cercle.
    • La procédure pour ajouter ce paramètre est similaire à celle ci-dessus, sauf que vous ajoutez un "Nouvel article entier" et entrez le texte UseMaxFN, puis entrez une valeur appropriée.

À partir de FreeCAD version 0.14, FreeCAD recherchera l'exécutable OpenSCAD si le paramètre mentionné ci-dessus est vide.
