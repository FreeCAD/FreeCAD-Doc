Description

Converti les bords en faces. Utile pour préparer la géométrie DXF importée pour pouvoir l'extruder.

Utilisation

  1. Sélectionnez les bords à convertir dans la vue 3D.
  2. Choisir OpenSCAD → Convertit les bords en faces dans le menu supérieur.
