Description

Développe tous les emplacements dans l'arbre des entités.

Utilisation

  1. Sélectionner l’objet dans la vue 3D.
  2. Choisissez OpenSCAD → Développer les placements du menu supérieur.

Limitations


Notes

Cette commande "réinitialise" les valeurs de placement d'un objet, rendant la position actuelle à 0/0/0.
Marche avec:

  • objets booléens
  • objets extrudés (dérivés de Part-Workbench)
  • objets retournés (dérivés de Part-Workbench)
