Description

Augmente la tolérance des arêtes/faces/sommets des objets sélectionnés.

Utilisation

  1. Sélectionner un objet dans l'arborescence
  2. Choisser OpenSCAD → Augmenter la tolérance dans le menu principal.
  3. Sélectionner Tolerance-Object dans l'arborescence de l'arborescence.
  4. Sélectionnez l'onglet de données dans la vue de propriété ou dans la vue combinée.
  5. Modifiez les valeurs des bords/faces/sommet à votre convenance.
