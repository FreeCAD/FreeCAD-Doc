Описание

Увеличивает допуски для ребер/граней/вершины выбранного объекта (ов).

Использование

  1. Выбрать объект в древе проекта
  2. Выбрать в верхнем меню OpenSCAD → Increase Tolerance Feature.
  3. Выбрать Tolerance-Object in feature tree in the tree view.
  4. Выбрать вкладку Данные в виде свойств.
  5. Установить желательные значения для рёбер/граней/вершин.
