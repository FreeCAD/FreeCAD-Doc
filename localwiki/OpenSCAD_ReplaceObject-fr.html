Description

Remplacer un objet dans l'arborescence.

Utilisation

  1. Multi-sélectionnez dans l'ordre correct, l'ancien, le nouveau et l'objet parent dans l'arborescence.
  2. Cliquez sur ou choisissez OpenSCAD → Remplacer l'objet dans le menu principal.

Limitations


Notes

Vous pouvez utiliser cette fonction par exemple pour remplacer une pièce dans un objet fusion/cut/common/compound.
