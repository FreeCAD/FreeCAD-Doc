Description

Insère un Cube primitif dans un corps actif (body) comme première fonction, ou le fusionne aux fonctions existantes.

Disponible dans la version 0.17


Comment l'utiliser

  1. Presser le bouton Cube additif.Note: Le Cube additif fait partie d'un menu d'icônes appelé Créer une primitive d'addition. Après le démarrage de FreeCAD, le cube additif est celui affiché dans la barre d'outils. Si une autre primitive est affichée, cliquer sur la flèche vers le bas et choisissez Cube additif.
  2. Définir les paramètres primitifs et d'attachement.
  3. Cliquer sur OK.
  4. Un cube apparaît dans le Corps actif.

Options

Le Cube peut être édité après sa création de deux façons:

  • Double-cliquer son étiquette dans l'arborescence, ou faire un clic droit sur l'étiquette et sélectionner Éditer la primitive dans le menu contextuel; ceci ouvre les Propriétés de la primitive.
  • Via l'éditeur de propriétés.

Propriétés

  • DONNÉESAttachment: définit les modes d'attachement ainsi que le décalage d'attachement. Voir Attachement.
  • DONNÉESLabel: Donne le nom du cube, changer si nécessaire.
  • DONNÉESLength: Longueur du cube sur l'axe X.
  • DONNÉESWidth: Largeur du cube sur l'axe Y.
  • DONNÉESHeight: Hauteur du cube sur l'axe Z.
