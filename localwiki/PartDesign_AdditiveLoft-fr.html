Description

Lissage additif produit un solide dans le corps de pièce actif en créant une forme lissée (transformation) entre deux esquisses ou plus. Si le corps actif possède déjà une ou des fonctions, le lissage sera ajouté à la suite (et sera fusionné avec le résultat précédent).

Disponible dans la version 0.17

A gauche: sections (A), (B) et (C); à droite, résultat du Lissage additif.

Comment l'utiliser

  1. Cliquez sur le bouton Lissage Additif.
  2. Dans la boite de dialogue Sélectionnez une fonction, sélectionnez une esquisse à utiliser comme première section et cliquez sur OK.
    • Autre possibilité: présélectionnez une esquisse avant de cliquer sur .
  3. Dans paramètres de lissage, cliquez sur Add Section.
  4. Sélectionnez une esquisse dans la vue 3D. Re-sélectionnez d'autres esquisses si nécessaires en cliquant sur Add Section.
  5. Choisissez des options si nécessaires puis cliquez sur OK.

Options

  • "Ruled surface" (surface réglée) produit une transformation droite entre les sections. N'a aucun effet avec seulement deux sections. Si l’option n'est pas sélectionnée, les transformations seront douces.
  • "Closed" (fermée): produit une transformation supplémentaire entre la dernière section et la première, en créant une boucle.
  • Cliquez sur Remove Section pour supprimer une esquisse dans la vue 3D.

Propriétés

  • DONNÉESLabel: nom donné à l'opération, ce nom peut être changé à la convenance.
  • DONNÉESSections: liste les sections utilisées.
  • DONNÉESRuled: voir Options.
  • DONNÉESClosed: voir Options.
  • DONNÉESMidplane: sans objet.
  • DONNÉESInversé: sans objet.
  • DONNÉESRefine: true (vrai) ou false (faux). Si la valeur est true, nettoie le solide des bords résiduels laissés par les fonctions. Voir Affiner la forme pour plus de détails.

Limites

  • Les esquisses doivent être toutes fermées.
  • Il n'est pas possible de faire un lissage avec un sommet.
  • Une section ne peut pas être sur le même plan que celle qui la précède.
  • Pour un meilleur résultat sur le lissage, il est recommandé que toutes les sections aient le même nombre de segments. Par exemple, pour un lissage entre un rectangle et un cercle, le cercle peut être décomposé en 4 arcs connectés.
