Description

PartDesign Clone crée une copie liée d'un objet sélectionné. La plupart des types d'objets sont acceptés, pourvu qu'ils consistent en un solide unique. L'utilisation principale pour PartDesign Clone est d'utiliser un objet créé dans un autre plan de travail pour une fonctionnalité Opération booléenne. Il peut également être simplement utilisé pour créer des copies liées.

Disponible dans la version 0.17



Clone d'un engrenage interne alors qu'il est translaté dans l'espace 3D comme un objet indépendant.

Utilisation

  1. Dans l'arborescence du modèle, sélectionnez l'objet à cloner.
  2. Appuyez sur le bouton Créer un clone.

Propriétés

  • DONNÉESBase Feature: définit l'objet d'origine sur lequel le clone est basé. Pour le remplacer, appuyez sur le bouton ... pour obtenir la liste des objets disponibles.
  • DONNÉESPlacement: définit l'orientation et la position du Clone dans l'espace 3D. Voir Placement (Positionnement).
  • DONNÉESLabel: label donné à l'objet Clone. Changez en fonction de vos besoins.

Limitations

  • Un seul objet peut être utilisé pour un clone PartDesign.
  • Seuls les objets qui consistent en un solide unique sont pris en charge. Donc, les composés (Compound) comme le Conteneur Pièce, le Composé Part ou le Réseau Draft ne sont pas pris en charge.
