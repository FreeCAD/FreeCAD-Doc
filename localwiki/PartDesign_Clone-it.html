Descrizione

PartDesign Clone crea una copia collegata di un oggetto selezionato. Sono accettati la maggior parte di tipi di oggetti. L'uso principale di PartDesign Clone consiste nell'utilizzare un oggetto creato in un altro ambiente per una funzione booleana. Può anche essere semplicemente usato per creare copie collegate.

disponibile dalla versione 0.17



Clona dell'ingranaggio interno mentre viene traslato nello spazio 3D come oggetto indipendente

Utilizzo

  1. Nell'albero del modello, selezionare l'oggetto da clonare.
  2. Premere i pulsante Crea un clone.

Proprietà

  • DATIBase Feature: imposta l'oggetto originale su cui si basa il clone. Per sostituirlo, premere il tasto ... per ottenere l'elenco degli oggetti disponibili.
  • DATIPlacement: definisce l'orientamento e la posizione del Clone nello spazio 3D. Vedere Posizionamento.
  • DATILabel: etichetta data all'oggetto Clone. Modificabile a piacere.

Limitazioni

  • Per un clone di PartDesign può essere usato solo un singolo oggetto.
