Description

Cet outil permet de créer un profil 2D d'une roue d'engrenage à développante.

Ce profil 2D est pleinement paramétrique et peut être extrudé avec la fonction Protrusion .

Pour faire un engrenage hélicoïdal, il faut l'extruder avec l'outil Sweep en utilisant une hélice comme chemin


Utilisation

  1. Aller dans le menu Part Design → Involute gear...
  2. Régler les paramètres du pignon
  3. Cliquer sur OK.
  4. Le pignon est créé hors du body. Sélectionnez et envoyez le dans le body pour une application dans un autre travail comme extrusion (protrusion)

Paramètres

Nombre de dents

Définit le nombre de dents désirées.

Modules

Diamètre Primitif divisé par le nombre de dents. Ou encore : Module X Nb de dents = Diamètre Primitif (ce sont les diamètres tangents dans un couples d'engrenages).

De ceux-ci dépendent l’entre-axes des arbres supports, et le rapport de réduction de vitesse entre les 2 arbres

Angle de pression

Angle aigu entre la ligne d'action (de poussée) et une normale à la ligne reliant les centres d'engrenage (tangente aux cercles primitifs). 

La valeur par défaut est de 20 degrés. (plus d'info)

Haute Précision

True or false (vrai ou faux)

Engrenage Extérieur

True or false (vrai ou faux)
