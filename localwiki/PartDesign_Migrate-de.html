Description

Die PartDesign-Workbench in FreeCAD v0.17 enthält neue Werkzeuge und Elemente, die von älteren FreeCAD-Versionen (0.16 und älter) nicht erkannt werden. FreeCAD-Dokumente, die in älteren Versionen erstellt wurden, können weiterhin geöffnet und bearbeitet werden. Um von den neuen Funktionen zu profitieren, müssen sie über das Menü PartDesign → Migrieren migriert werden.

verfügbar in Version 0.17


Anwendung

  1. Öffnen Sie ein älteres FreeCAD-Dokument (≤ 0,16).
  2. Wechseln Sie zur PartDesign-Workbench.
  3. Gehen Sie zum Menü 'Part Design' 'Migrieren' .
  4. Wenn die Migration funktioniert, wird eine Baugruppe(Part Container) erstellt, welche eine oder mehrere Bauteile(Bodies) enthält. , die jeweils eine Formelement ( feature) Historie beherbergen.

Beschränkungen

  • Überprüfen Sie vor dem Start des Migrationsprozesses, ob das Modell mit aktivierten automatischen Verfeinerungsoptionen erstellt wurde (unter Bearbeiten → Voreinstellungen → Part Design → Allgemein) und legen Sie Ihre Einstellungen entsprechend fest. Dies kann leicht bestimmt werden, indem nacheinander die Sichtbarkeit der Merkmale in der Modellstruktur umgeschaltet wird. Wenn keine verbleibenden Kanten zwischen Features wie Pads und Pockets verbleiben, wurden die automatischen Verfeinerungsoptionen deaktiviert.
  • Wenn ein zu migrierendes Dokument nur Skizzen und PartDesign-Funktionen enthält, ist der Migrationsprozess wahrscheinlich erfolgreich. Einige Funktionen wie Fasen und Verrundungen müssen möglicherweise neu erstellt werden.
  • Wenn das zu migrierende Dokument einen gemischten Teil- / Teileentwurf / Entwurf-Arbeitsablauf aufweist, wird die Konvertierung wahrscheinlich fehlschlagen oder bestenfalls zu unerwarteten Ergebnissen führen und muss manuell migriert werden.
