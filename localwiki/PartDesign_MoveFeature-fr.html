Description

Déplacer vers un autre corps, comme cette commande est étiquetée dans le menu contextuel, permet de déplacer des objets sous un corps vers un autre corps. L'objet déplacé est placé juste en dessous de la fonction résultante de l'autre corps. Les esquisses, la géométrie de référence et les fonctionnalités sont prises en charge.

Disponible dans la version 0.17


Comment l'utiliser

  1. Dans l'arborescence du modèle, cliquer avec le bouton droit sur l'objet à déplacer.
  2. Sélectionner Déplacer vers un autre corps.
  3. Dans la boîte de dialogue Sélectionner le corps, sélectionner le corps pour déplacer l'objet dans le menu déroulant.
  4. Appuyez sur OK.
