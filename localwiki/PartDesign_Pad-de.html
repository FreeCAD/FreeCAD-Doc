Beschreibung

Das Aufpolstern (Pad) Werkzeug extrudiert eine Skizze zu einem Festkörper ( Solid) senkrecht zur Ebene der Skizzenfläche. Ab 0.17 können Flächen eines vorhandenen Festkörpers direkt basierend auf ihrem Umriß extrudiert werden.

Skizze (A) links und der daraus resultierende Festkörper (B) rechts.

v0.16 und vorher Wurde die Aufpolsterung von einer Fläche des Festkörpers, oder eines Formelements dessen, aus erstellt ist das Resultat fest damit verbunden.

Benutzung

  1. Selektier die Skizze, welche Aufgepolstert werden soll. Ab v0.17 kann dies auch eine Fläche eines vorhandenen Festkörpers sein.
  2. Klicke das Aufpolstern Icon.
  3. Setze die Parameter der Aufpolsterung (Siehe unterhalb).
  4. Klicke OK OK.

Optionen

Währen des Aufpolsterns schaltet die Kombiansicht in den Aufgabenbereich und zeigt dort den Parameter der Aufpolsterung Dialog

Typ

Typ bietet fünf verschiedene Möglichkeiten zum Festlegen der Länge, auf welche extrudiert werden soll.

Dimension

Geben Sie einen numerischen Wert für die Länge des Pads ein. Die Standardrichtung für die Extrusion ist in Richtung positiver Normale des extrudierten Objekts, dies kann geändert werden, indem Sie die Option 'Umgekehrt' ankreuzen. Extrusionen erfolgen normal zur definierenden Ebene. Mit der Option 'Symmetrisch zu Ebene' 'wird die Fläche um die Hälfte der gegebenen Länge auf jede Seite der Ebene verlängert. Negative Dimensionen sind nicht möglich. Verwenden Sie stattdessen die Option 'Umgekehrt' .

Zwei Längen (ab 0.17)

Es kann eine zweite Länge eingegeben werden, in der sich das Pad in die entgegengesetzte Richtung (in die die Ebene hinein) erstrecken soll. Auch dies kann geändert werden, indem Sie die Option 'Reversed' aktivieren.

Zur letzten

Das Pad extrudiert bis zur letzten Fläche des Ausgangskörpers in Extrusionsrichtung. Wenn keine solche Fläche vorhanden ist, wird eine Fehlermeldung angezeigt.

Zur ersten

Die Aufpolsterung wird in Extrusionsrichtung bis zur ersten Seite des tragenden Objekts extrudiert. Wenn keine Fläche vorhanden ist, wird eine Fehlermeldung angezeigt.

Bis zur Oberfläche

Die Aufpolsterung wird zu einer Fläche im Ausgangskörper extrudiert, welche durch Klicken ausgewählt werden kann. Wenn keine Fläche vorhanden ist, werden keine Auswahlmöglichkeiten akzeptiert.

Länge

Definiert die Länge der Extrusion(m, cm, mm, nm, ft oder ', in oder "). Einheiten können unabhängig von der gewählten Standardeinheit angegeben werden.

Symmetrisch zu einer Fläche

Diese Option extrudiert die Gesamtlänge exakt hälftig in beide Richtungen.

Umgekehrt

Kehrt die Extrusionsrichtung um.

Eigenschaften

  • Daten-EigenschaftRefine: v0.17 und danach true oder false. Bereingt überflüssige entstandene Kanten nach der Berechnung. Diese Option richtet sich nach den Vorgeben des Benutzers in den Grundeinstellungen (zu finden in Bearbeiten → Einstellungen → Part design → Allgemein → Modelleinstellungen). Dies kann im nachhinein geändert werden.

Diese Eigenschaft wird im Dokument mit gespeichert.

Einschränkungen

  • Wie alle Part Design-Features erstellt Aufpolsterung einen Volumenkörper, daher muss die Skizze ein geschlossenes Profil enthalten oder es wird ein Fehler auftreten. Diese Fehler wäre "Failed to validate broken face". Innerhalb eines größeren kann es mehrere eingeschlossene Profile geben, sofern sich keine davon schneiden (z. B. ein Rechteck mit zwei Kreisen).
  • Der für Zur ersten und Zur letzten verwendete Algorithmus ist:
    • Denken Sie eine Linie durch den Schwerpunkt der Skizze
    • Finden Sie alle Flächen des Ausgangskörpers, welche durch diese Linie geschnitten werden
    • Wählen Sie die Fläche aus, an der sich der Schnittpunkt am nächsten / am weitesten von der Skizze befindet
Dies bedeutet, dass die gefundene Fläche nicht immer das ist, was Sie erwartet haben. Wenn Sie auf dieses Problem stoßen, verwenden Sie stattdessen den Typ "Bis zur Oberfläche" und wählen Sie die gewünschte Fläche aus aus.

Für den ganz speziellen Fall der Extrusion zu einer konkaven Oberfläche, wo die Skizze größer als diese Oberfläche ist, wird die Extrusion versagen. Dies ist ein ungelöster Fehler.

  • v0.16 und davor Es gibt keine automatische Bereinigung, z. benachbarter planarer Oberflächen zu einer einzigen Oberfläche. Sie können dies manuell in der Part-Workbench mit Refine shape (was einen nicht verknüpften, nicht parametrischen Solid erzeugt) oder mit dem Refine Shapefeature aus der [[[OpenSCAD Workbench]] ], die ein parametrisches Feature erstellt nachbilden.
