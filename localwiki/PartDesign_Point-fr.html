Description

Crée un point de référence qui peut être utilisé comme référence pour les esquisses ou toute autre géométrie de référence.

Disponible dans la version 0.17


Un point de référence attaché à une sphère avec un décalage d'attachement de Z = 2


Comment faire

  1. Appuyez sur le bouton Créer un point de référence .
  2. Définissez les paramètres du point. Sélectionnez une première référence dans la vue 3D pour filtrer les modes d'attachement disponibles.
  3. Selon la référence sélectionnée, il peut y avoir un ou plusieurs modes d'attachement disponibles dans la liste. Le plus probable sera automatiquement sélectionné et affiché en gras dans la liste. Le texte Attaché avec le mode ainsi que le nom du mode d'attachement apparaissent en vert en haut du panneau Paramètres.
  4. Pour ajouter une référence supplémentaire, appuyez sur le bouton suivant Référence. Une fois le bouton appuyé, son étiquette devient Sélection ... jusqu'à ce qu'une sélection soit faite.
  5. Sélectionnez un mode d'attachement dans la liste.
  6. Définissez les valeurs de décalage d'attachement.
  7. Appuyez sur OK.

Options

Double-cliquez sur l'étiquette Point de référence dans l'arborescence du modèle ou cliquez avec le bouton droit de la souris et sélectionnez Modifier la référence dans le menu contextuel pour éditer ses paramètres. Pour plus de détails sur le mode de référencement et le décalage de la référence, voir Référencement .

Propriétés

  • DONNÉESMapMode: répertorie le mode d'attachement utilisé.
  • DONNÉESAttachment Offset: applique une transformation (translation et rotation) en référence au placement de l'attachement.
  • DONNÉESLabel: nom donné à l'objet, ce nom peut être changé à la convenance.

Limitations

  • Le point de référence ne peut pas être utilisé comme section pour les fonctions Balayage et Lissage.
