Einleitung

'Polares Muster von Formelementen(Features) erstellen' - Dieses Werkzeug verwendet ein ausgewähltes Formelement und erstellt daraus eine Reihe von (um eine ausgewählte Achse) gedrehten Kopien.
Das resultierende Objekt wird als neues separates Objekt in der Baumansicht erstellt.

Optionen

Beim Erstellen eines Polaren Form-Musters bietet der Dialog "Polardiagramm-Parameter" zwei verschiedene Möglichkeiten zum Festlegen der Pattern-Rotationsachse.

Normale Skizzenachse

Eine Achse, die normal zur Skizze ist und im Ursprung der Skizze des verwendeten Merkmals beginnt, wird als Achse für das polare Muster genommen.
Die Drehrichtung des Musters kann umgekehrt werden, indem Sie auf "Richtung umkehren" klicken.

Wählen Sie die Referenz

Hier können Sie eine Kante als Referenz auswählen.
Wenn diese Option gewählt wird, ist nur eine Kante des Formelements, das gerade vor der Polarmusteroperation verwendet wurde, wählbar. Sie können diese Option folgendermaßen verwenden:

bis 0.16

  • Erstellen eines Helfer-Features (z. B. eines Pads), wobei eine der Kanten die Achse des Polarmusters ist
  • erstellen Feature (Pad oder Pocket) zu gemustert werden
  • Wählen Sie die Richtcharakteristik und wählen Sie die Kante als Referenz
  • Verwenden Sie eine andere Hilfsfunktion (z. B. Tasche), um Ihre erste Hilfsfunktion zu entfernen.

ab 0.17

  • Sie können auch Bezugslinien als Referenz nutzen

Die Musterrichtung kann umgekehrt werden, indem Sie auf "Richtung umkehren" klicken.


Winkel und Vorkommen

Gibt den Winkel an, der vom Muster abgedeckt werden soll, sowie die Gesamtzahl der Musterformen (einschließlich des ursprünglichen Elements). Zum Beispiel würden vier Vorkommen in einem Winkel von 180 Grad einen Abstand von 60 Grad zwischen Mustern ergeben. Es gibt eine Ausnahme: Wenn der Winkel 360 Grad beträgt werden z.b vier Vorkommen um 90 Grad voneinander getrennt, weil das erste und das letzte Vorkommen identisch sind.
