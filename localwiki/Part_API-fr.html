Part API/fr

Le module Part , est un lien direct, entre FreeCAD, et ,le noyau OpenCasCade. Il fournit principalement des TopoShapes qui sont les types d'objets principaux utilisés par OpenCascade. Le module Part contient également une variété de fonctions utiles, pour créer, et, manipuler des topoShapes. Exemple : 

import Part
mycube = Part.makeBox(2,2,2)
Part.show(mycube)


__fromPythonOCC__ ( OCC.Object )

Description: Méthode d'assistance pour convertir un shape pythonocc en un shape interne

Retour: A Part.Shape



__sortEdges__ ( list of edges )

Description: Méthode d'assistance pour trier une liste non triée de bords (edges), ce qui par la suite deux bords adjacents partagent un sommet commun

Retour: a list of edges



__toPythonOCC__ ( Part.Shape )

Description: Méthode d'assistance pour convertir une forme interne en une forme pythonocc

Retour: an OCC.Shape



cast_to_shape ( Part.Shape )

Description: Cast du type de forme réelle

Retour:



export ( list,string )

Description: Exporter une liste d'objets dans un seul fichier.

Retour:



getSortedClusters ( list of edges )

Description: Méthode d'assistance pour trier et cluster une variété des bords

Retour:



insert ( string,string )

Description: Insérez le fichier (chemin d'accès donné comme premier argument) dans le document donné (second argument).

Retour:



makeBox ( length,width,height,[pnt,dir] )

Description: Rend une zone située au point avec les dimensions (longueur, largeur, hauteur). Par défaut le point est au Vecteur(0,0,0) et la direction au Vecteur(0,0,1)

Retour: Crée une forme



makeCircle ( radius,[pnt,dir,angle1,angle2] )

Description: Fait un cercle d'un rayon donné. Par défaut le point est au Vecteur(0,0,0), et la direction est, Vecteur(0,0,1), angle1 est 0° et angle2 est 360°

Retour: Crée une forme



makeCompound ( list )

Description: Crée un composé parmi une liste de formes.

Retour: Crée une forme



makeCone ( radius1,radius2,height,[pnt,dir,angle] )

Description: Fait un cône avec le rayon et la hauteur. Par défaut le point est au Vecteur(0,0,0), et la direction est, Vecteur(0,0,1) et l'angle est de 360°

Retour: Crée une forme



makeCylinder ( radius,height,[pnt,dir,angle] )

Description: Rend un cylindre avec une taille et un rayon donné. Par défaut le point est au Vecteurr (0,0,0), et la direction est, Vecteur(0,0,1), et, l'angle est de 360°

Retour: Crée une forme



makeHelix ( pitch,height,radius,[angle] )

Description: Rend une hélice avec une hauteur donnée, la hauteur et le rayon. Par défaut, une surface cylindrique est utilisée pour créer l'hélice. S'il y a un quatrième paramètre, une surface conique est utilisée à la place

Retour: Crée une forme



makeLine ( (x1,y1,z1),(x2,y2,z2) )

Description: Crée une ligne en deux points

Retour: Crée une forme



makeLoft ( shapelist<profiles>,[boolean<solid>,boolean<ruled>] )

Description: Creates a loft shape using the list of profiles. Optionally make result a solid (vs surface/shell) or make result a ruled surface

Retour: Crée une forme de loft.



makePlane ( length,width,[pnt,dir] )

Description: Crée un plan. Par défaut le point est au Vecteur(0,0,0) et la direction au Vector(0,0,1)

Retour: Crée une forme



makePolygon ( list )

Description: Crée un polygone avec une liste de vecteurs

Retour: Crée une forme



makeRevolution ( Curve,[vmin,vmax,angle,pnt,dir] )

Description: Rend une forme de révolution en faisant tourner la courbe, ou une partie de celui-ci autour d'un axe donné par (point, direction). Par défaut, vmin et vmax sont définis aux limites de la courbe, l'angle est de 360°, le point est au Vecteur(0,0,0) et la direction est au Vecteur(0,0,1)

Retour: Crée une forme



makeRuledSurface ( Edge or Wire,Edge or Wire )

Description: Crée une surface réglée hors de deux arêtes ou fils. Si les fils sont utilisés ils doivent avoir le même nombre d'arêtes.

Retour: Crée une forme



makeShell ( list )

Description: Crée un shell sur une liste de faces.

Retour: Crée une forme



makeSolid ( Part.Shape )

Description: Crée un solide hors du shell, à l'intérieur d'une forme.

Retour: Crée une forme



makeSphere ( radius,[pnt, dir, angle1_Debut,angle2_Fin,angle3] )

Description: Crée une sphère de rayon donné. Par défaut le point est au Vecteur(0,0,0), et la direction est au Vecteur(0,0,1), angle1 est -90°, angle2 est 90° et angle3 est 360°

Retour: Crée une forme



makeTorus ( radius1,radius2,[pnt,dir,angle1,angle2,angle] )

Description: Crée un tore avec une donnée de rayon d'angles. Par défaut le point est au Vecteur (0,0,0), et la direction est au vecteur (0,0,1), angle1 est 0°, angle2 est 360° et l'angle est de 360°

Retour: Crée une forme



makeTube ( edge,float )

Description: Crée un tube.

Retour: Crée une forme



open ( string )

Description: Crée un nouveau document et charge le fichier dans le document.

Retour:



read ( string )

Description: Charge le fichier et retourne un shape.

Retour: a shape



show ( shape )

Description: Ajouter la forme du document actif ou en crée un si aucun document n'existe.
