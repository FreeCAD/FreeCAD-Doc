Una herramienta para crear a varias primitivas geométricas paramétricos.

  • Actualmente esta herramienta puede crear una paramétrica
    • Plane
    • Box
    • Cylinder
    • Cone
    • Sphere
    • Ellipsoid
    • Torus
    • Prism available in version 0.14
    • Wedge
    • Helix
    • Spiral available in version 0.14
    • Circle
    • Ellipse
    • Line (Edge)
    • Point (Vertex)
    • Regular Polygon available in version 0.14
