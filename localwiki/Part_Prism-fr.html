Description

Dans la boîte de dialogue créer des primitives un outil de création de Prisme est disponible. Un prisme est un solide délimité par une section transversale polygonale régulière et une hauteur.

La boîte de dialogue Création de Primitives est accessible par le bouton Créer des Primitives situé dans menu Part ou sur la barre d'outils Part Workbench.


Paramètres

  • Polygon - le nombre de faces du polygone pour créer le Prisme
  • Cirumradius - le cirumradius est le cercle circonscrit dont la distance est compris entre le centre du polygone et le sommet.
  • Height - la hauteur du Prisme.

Disponible dans la version 0.14
