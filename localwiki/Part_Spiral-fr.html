Description

Un outil Spirale géométrique primitive est disponible dans la boîte de dialogue Création de primitives ou par le menu Part workbench.

La boîte de dialogue Creation de Primitives est accessible en cliquant sur l’icône situé dans le menu Part ou sur la barre d'outil de Part Workbench.



Disponible dans la version 0.14
