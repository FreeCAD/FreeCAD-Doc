Description

Cet outil crée un nouveau cheminen duplicant un autre chemin plusieurs fois avec un intervale donné.

Utilisation

  1. Sélectionnez un objet Path
  2. Pressez le bouton Array
  3. Ajustez les propriétés voulues

Propriétés

  • DATAOffset: L'espace entre les copies multiples
  • DATACopies: Le nombre de copies
