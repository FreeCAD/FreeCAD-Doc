Descrizione

Questo strumento crea un nuovo percorso duplicando più volte un altro percorso ad una determinato intervallo di distanza.

Utilizzo

  1. Selezionare un oggetto Path da replicare
  2. Premere il pulsante Schiera
  3. Applicare
  4. Regolare le proprietà desiderate

Proprietà

  • DATAType: Il tipo di schiera (polare, lineare, in una o due direzioni)
  • DATAOffset: La spaziatura tra le copie della schiera per ogni direzione
  • DATACopies: Il numero di copie per ogni direzione
