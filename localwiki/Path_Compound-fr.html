Description

Cet outil crée un chemin composé formé par une séquence de chemins sélectionnés.

Utilisation

  1. Sélectionnez un ou plusieurs objets Path
  2. Pressez le bouton Compound

Options

  • Plus de chemins peuvent être ajoutés au composé en les tirant dans la vue arborescente
  • Les chemins peuvent être enlevés du composé en les tirant du document racine de la vue arborescente
  • Les chemins peuvent être modifiés en double-cliquant sur le composé de la vue arborescente
