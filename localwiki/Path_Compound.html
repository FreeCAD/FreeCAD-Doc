Description

This tool creates a compound path formed by a sequence of selected paths

Usage

  1. Select one or more Path objects
  2. Press the Compound button

Options

  • More paths can be added to the compound by dragging them on it in the tree view
  • Paths can be removed from the compound by dragging them from the compound to the document root in the tree view
  • Paths order can be modified by double-clicking the compound in the tree view
