Description

Cet outil crée une copie du chemin sélectionné

Utilisation

  1. Sélectionnez un objet Path
  2. Pressez le bouton Copy

Propriétés

  • DATABase: Le chemin de base devant être copié

Options

La copie reste liée avec le chemin original. Si l'original change, la copie aussi.
