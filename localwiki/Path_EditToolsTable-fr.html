Description

Le gestionnaire du magasin d'outils permet d'éditer les différents outils de la collection contenue dans l'objet machine d'un Travail. Cette collection d'outils est utilisée par les différentes opérations contenues dans le projet pour obtenir les informations nécessaires sur l'outil actuel.

Il sert également à la sélection d'un outil que vous souhaitez utiliser dans votre travail.

La manipulation est simple:

  • Importer ...: Importe un magasin d'outils à partir d'un fichier XML.
    Avertissement

Ceci est actuellement en partie cassé et ne fonctionne pas si vous n'avez jamais eu de fichier xml auparavant.

  • Exporter ...: Exporte le magasin d'outils vers un fichier XML.
  • Nouvel outil: ouvre une boîte de dialogue dans laquelle vous pouvez entrer les paramètres de votre outil. Voir Nouvel outil
  • Supprimer: supprime les lignes actuellement sélectionnées.
    Avertissement

Les outils sont supprimés de votre magasin d'outils même si vous annulez la boîte de dialogue

  • Monter: Vous ne pouvez pas modifier le numéro d'outil, vous pouvez déplacer la ligne sélectionnée pour diminuer son numéro d'outil
  • Descendre: Vous pouvez déplacer la ligne sélectionnée vers le bas pour augmenter son numéro d'outil
  • Créer un ou des contrôleurs d'outils: Si vous cochez une ou plusieurs cases à gauche de la liste d'outils, ce bouton devient actif. Si vous cliquez dessus, les outils sélectionnés seront insérés dans votre travail actuel.

Usage

  1. Sélectionnez une Tâche
  2. Appuyez sur le bouton 16px Gestionnaire d'outils
  3. Créer de nouveaux outils ou ajuster les propriétés des outils existants

Options

  • Les outils peuvent être réorganisés en utilisant les boutons Monter / Descendre
  • Les collections d'outils complètes peuvent être importées à partir de fichiers xml (le format des collections d'outils de FreeCAD) ou à partir des magasins d'outils de HeeksCAD
