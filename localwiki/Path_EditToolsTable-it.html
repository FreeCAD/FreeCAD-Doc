Descrizione

L'editor della tabella utensili permette di modificare i diversi strumenti della tabella utensili contenuta nell'oggetto macchina di una Lavorazione. Questa tabella utensili è utilizzata per le diverse operazioni contenute nel progetto per ottenere le informazioni necessarie sullo strumento corrente.

Serve per selezionare lo strumento che si desidera utilizzare nella lavorazione.

La gestione è semplice:

  • Import...: Importa una tabella utensili da un file XML.
    Attenzione 
Questo attualmente è difettoso e non funziona se non si ha mai avuto un file xml prima.
  • Export...: Esporta la tabella degli utensili in un file XML.
  • New Tool: Apre una finestra di dialogo in cui inserire i parametri dell'utensile. Vedere Nuovo utensile
  • Delete: Cancella le righe attualmente selezionate.
    Attenzione 
Gli utensili vengono eliminati dalla tabella degli utensili anche se si annulla la finestra di dialogo
  • Move up: Non è possibile modificare il numero dell'utensile, ma è possibile spostare la riga selezionata verso l'alto per diminuirlo
  • Move down: È possibile spostare la riga selezionata verso il basso per aumentare il numero dell'utensile
  • Create Tool Controller(s): Quando si seleziona una o più caselle di controllo a sinistra nell'elenco degli strumenti, questo pulsante diventa attivo. Cliccando, gli strumenti selezionati vengono inseriti nella lavorazione corrente.

Utilizzo

  1. Selezionare una Lavorazione
  2. Premere il pulsante 16px gestione utensili
  3. Creare nuovi strumenti o regolare le proprietà di strumenti esistenti

Opzioni

  • Gli strumenti possono essere riordinati utilizzando i pulsanti sposta-in-alto/basso
  • Tabelle complete di utensili possono essere importate da file xml (FreeCAD ha un proprio formato tooltable) o da tooltables HeeksCAD
