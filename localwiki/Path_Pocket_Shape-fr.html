Description

Cet outil crée une opération de poche à partir des faces inférieures ou des parois sélectionnées d'une ou de plusieurs poches de l'objet.

L'objet Path Pocket est fait pour travailler avec Path Job.

Utilisation

  1. Sélectionnez le fond ou une face(s) d'une poche. Alors qu'il est généralement plus facile de sélectionner le fond, les faces doivent être sélectionnées lorsqu'une poche est traversante.
  2. Pressez le bouton Pocket Shape
  3. Ajuster les propriétés voulues

Propriétés

Géométrie de base

  • Ajouter, si ce n'est déjà fait, les éléments qui devraient être la base du chemin
    • Vous pouvez également supprimer ou modifier d'autres éléments

Profondeur

  • DONNÉESStart Depth: Profondeur de départ de l'outil - profondeur de départ dans l'axe Z
  • DONNÉESFinal Depth: Profondeur de fin de l'outil - profondeur de fin dans l'axe Z
  • DONNÉESFinish Depth: La hauteur du dernier niveau de coupe-ensemble pour une meilleure finition
  • DONNÉESStep Down: Descente de l'outil - la quantité de matériau qui sera enlevé par passe

Hauteur

  • DONNÉESSafe Height: La hauteur au-dessus de la zone de travail où les mouvements rapides sont autorisés.
  • DONNÉESClearance Height: La hauteur nécessaire pour dégager les clames et les obstructions

Opération

  • DONNÉESTool Controller: Définit le contrôleur d'outil utilisé dans l'opération
  • DONNÉESCut Mode: Spécifie un mouvement CW ou CCW pour la coupe
  • DONNÉESPattern: Sélectionnez de quelle manière les mouvements horizontaux doivent être effectués
  • DONNÉESAngle: Sélectionnez l'angle de la trajectoire par rapport à l'axe x
  • DONNÉESStep Over Percent: Sélectionnez le pas horizontal supérieur à 100% = diamètre de l'outil
