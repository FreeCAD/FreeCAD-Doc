Descrizione

Questo strumento crea un'operazione di scavo basato sulle facce o sulle pareti inferiori selezionate in una o più tasche dell'oggetto in lavorazione.

L'oggetto Tasca di Path è fatto per essere parte di una Lavorazione.

Utilizzo

  1. Selezionare una faccia o una parete inferiore di una tasca. Mentre di solito è più facile selezionare il fondo, le parete devono essere selezionati quando una tasca passa attraverso tutto.
  2. Premere il pulsante Tasca
  3. Regolare le proprietà desiderate

Proprietà

Geometria base

  • Aggiunge, se non è ancora stato fatto, gli elementi che devono essere usati per la base del percorso
    • è possibile cancellare, sostituire o aggiungere altri elementi

Profondità

  • DATIStart Depth: Profondità iniziale dell'utensile - - profondità iniziale in Z
  • DATIFinal Depth: Profondità finale dell'utensile - livello di taglio più basso in Z
  • DATIFinish Depth: L'altezza dell'ultimo livello di taglio - impostato per una finitura migliore
  • DATIStep Down: Profondità incrementale dell'utensile - la quantità di materiale che verrà rimosso ad ogni passata

Altezza

  • DATISafe Height: L'altezza al di sopra della quale sono consentiti i movimenti in rapida.
  • DATIClearance Height: L'altezza necessaria per evitare morsa e ostruzioni

Operazioni

  • DATITool Controller: Definisce il controller dell'utensile utilizzato nell'operazione
  • DATICut Side: Definisce se il taglio deve essere fatto in senso orario CW, o antiorario CCW
  • DATIPattern: Stabilisce in che modo devono essere eseguiti i movimenti orizzontali
  • DATIAngle: Seleziona l'angolo del percorso relativo all'asse x
  • DATIStep Over Percent: Select the horizontal step over - 100% = diametro dell'utensile
