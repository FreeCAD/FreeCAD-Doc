Description

Cet outil crée une opération de profilage, en débutant par la face sélectionnée d'un objet Part.

Utilisation

  1. Sélectionnez une face sur un objet Part. Vous pouvez aussi, alternativement, choisir les bords de cette face.
  2. Pressez le bouton Profile
  3. Ajustez les propriétés voulues

Options

  • L'objet Path Profile est conçu pour être une partie du Path Project. Pour configurer correctement le profilage, une table d'outil doit être présente dans l'objet projet machine et une commande Tool Change doit être placée avant le profilage.

Propriétés

  • DATAActive: Positionner à False, pour empêcher l'opération de générer un code
  • DATAUse Placements: mis à True dans le cas d'une utilisation de la propriété de placement de l'opération de profilage pour transformer le chemin d'outil avec le post-traitement
  • DATATool Number: Le numéro d'outil en utilisation

Propriétés de Profondeur


  • DATAClearance Height: Hauteur nécessaire pour enlever les copeaux et les obstructions
  • DATAStep Down: Descente Incrémentale de l'outil
  • DATAUse Start Depth: mis à True, pour une profondeur de démarrage spécifiée manuellement
  • DATAStart Depth: Démarrage par la profondeur de l'outil - première coupe en profondeur Z
  • DATAFinal Depth: Profondeur finale de l'outil - valeur en Z la plus basse
  • DATARetract Height: Hauteur désirée pour retirer l'outil lorsque le chemin est fini
  • DATAComment: Un commentaire optionnel pour ce profilage



Propriétés de l'usinage

Propriétés du point de départ

  • DATAStart Point: Le point de départ de ce chemin
  • DATAUse Start Point: mis à True, si un point de départ est spécifié
  • DATAExtend At Start: longueur supplémentaire du cheminement d'outil avant le démarrage sur le bord du matériau
  • DATALeadIn Line Len: longueur du segment direct du chemin d'outil pour aller à l'angle du premier bord du matériau

Propriétés du point final

  • DATAUse End Point: mis à True, si l'on spécifie un point final
  • DATAExtend At End: longueur supplémentaire du cheminement d'outil après la fin sur le bord du matériau
  • DATALeadOut LineLen: longueur du segment direct du chemin d'outil pour aller à l'angle du dernier bord du matériau
  • DATAEnd Point: Le point final du chemin

Propriétés du Profilage

  • DATASide: Coté du bord qui doit être coupé
  • DATADirection: Direction que doit prendre le chelin d'outil autour du matériau, ClockWise CW ou CounterClockWise CCW
  • DATARollRadius: Rayon au démarrage et à la fin
  • DATAOffsetExtra: Valeur supplémentaire pour rester en dehors du profilage final - pour assurer un chemin d'outil de dégrossi
  • DATASegLen: Valeur de tesselation pour les valeurs de chemins d'outils réalisés à partir de béziers, bsplines et ellipses
