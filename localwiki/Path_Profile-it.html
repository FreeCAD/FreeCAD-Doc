Descrizione

Lo strumento Contornatura crea un semplice contorno di taglio esterno di un oggetto 3D complesso basato su Part. L'intero modello in lavorazione funge da input per l'operazione, indipendentemente dal fatto che sia selezionata una geometria del corpo quando viene invocato il comando Contornatura. Si può abilitare un punto di partenza dall'editor delle operazioni, utilizzando una posizione definita nella Proprietà vista -> Punto iniziale.

Utilizzo

  1. Selezionare una faccia di un oggetto basato su Parte. In alternativa, selezionare il bordo iniziale di tale faccia.
  2. Premere il pulsante Profilo
  3. Regolare le proprietà desiderate

Opzioni

  • L'oggetto Profilo di Path è fatto per essere una parte di una Lavorazione. Per configurare correttamente la profilatura, nell'oggetto macchina del progetto deve essere presente una tabella utensile, e prima del Profilo deve essere posizionato un comando Utensile.

Proprietà

  • DATAActive: Rendere False, per evitare operazioni di generazione di codice
  • DATAUse Placements: Rendere True, se si utilizzano le proprietà di posizionamento del profilo per trasformare il percorso utensile in un post processore
  • DATATool Controller: Il Tool controller specifica l'indice dell'utensile da utilizzare

Proprietà di profondità

  • DATAStart Depth: Profondità iniziale di Tool - prima profondità di taglio in Z
  • DATAFinal Depth: Profondità finale di Tool - valore più basso in Z
  • DATAStep Down: Passo incrementale in profondità dello strumento


Proprietà Altezza

  • DATASafe Height: L'altezzza al di sopra della quale sono consentiti i movimenti in rapida.
  • DATAClearance Height: L'altezza necessaria per evitare bloccaggi e ostruzioni



Proprietà Operazioni

  • DATATool Controller: Definisce il controller dell'utensile utilizzato nell'operazione
  • DATADirection: Specifica un percorso CW o CCW (orario o antiorario) per il taglio del contorno esterno
  • DATAUse Start Point: Rendere True, se si specifica manualmente un Punto di partenza, quindi inserire i Punti di partenza nel campo Punti di partenza delle Proprietà dati

Proprietà Start Point

  • DATAStart Point: Il punto di partenza di questo percorso
  • DATAUse Start Point: Impostare True, se si specifica un Start Point
  • DATAExtend At Start: lunghezza extra del percorso utensile prima dell'inizio del bordo della parte
  • DATALeadIn Line Len: lunghezza del tratto rettilineo del percorso utensile che entra ad angolo rispetto al primo bordo della parte

Proprietà End Point

  • DATAUse End Point: Impostare True, se si specifica un End Point
  • DATAExtend At End: lunghezza extra di percorso utensile dopo la fine del bordo della parte
  • DATALeadOut LineLen: lunghezza del tratto rettilineo del percorso utensile che esce ad angolo rispetto all'ultimo bordo della parte
  • DATAEnd Point: Il punto finale di questo percorso

Proprietà Profilo

  • DATASide: Lato del bordo che lo strumento deve tagliare
  • DATADirection: La direzione in cui l'utensile deve muoversi intorno alla parte, orario CW o antiorario CCW
  • DATARollRadius: Raggio iniziale e finale
  • DATAOffsetExtra: Valore aggiuntivo di scostamento dal profilo finale - utile per il percorso dell'utensile di sgrossatura
  • DATASegLen: Valore di Tassellazione per percorsi utensili realizzati da Beziers, bspline e ellissi
