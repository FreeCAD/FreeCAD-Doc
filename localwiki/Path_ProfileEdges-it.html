Descrizione

Lo strumento Profila bordi crea un semplice percorso basato sul contorno dei bordi selezionati. Tramite l'editor delle operazioni si può abilitare un punto di partenza, utilizzando una posizione definita nelle Proprietà vista -> Punti di partenza.

L'oggetto Profila bordi di Path è fatto per essere parte di una Lavorazione.

Utilizzo

  1. Selezionare un ciclo completo di bordi
  2. Premere il pilsante Profila bordi
  3. Adattare le proprietà


! Profila bordi non si preoccupa degli altri dettagli dell'oggetto. Si devono regolare le proprietà con molta attenzione, specialmente la profondità finale, un errore può distruggere la postazione di lavoro


Proprietà

Geometria base

  • Aggiunge, se non è ancora stato fatto, gli elementi che devono essere usati per la base del percorso
    • è possibile cancellare, sostituire o aggiungere altri elementi

Profondità

  • DATAStart Depth: Profondità iniziale dell'utensile - - profondità iniziale in Z
  • DATAFinal Depth: Profondità finale dell'utensile - livello di taglio più basso in Z
  • DATAStep Down: Profondità incrementale dell'utensile - la quantità di materiale che verrà rimosso ad ogni passata

Altezza

  • DATASafe Height: L'altezza al di sopra della quale sono consentiti i movimenti in rapida.
  • DATAClearance Height: L'altezza necessaria per evitare morsa e ostruzioni

Operazioni

  • DATATool Controller: Definisce il controller dell'utensile utilizzato nell'operazione
  • DATACut Side: Definisce da che parte deve essere fatto il taglio
  • DATADirection: Specifica un percorso CW o CCW (verso orario o antiorario) per il taglio del contorno esterno
  • DATAExtra Offset: Valore extra per stare lontano dal profilo finale (sovrametallo) - utile per il percorso utensile di sgrossatura
  • DATAUse Start Point: Da rendere True, se si specifica manualmente un Punto iniziale, quindi inserire il Punto iniziale nel campo Start Point delle Proprietà Dati (non ancora implementato)
  • DATAUse Compensation: Crea un percorso con offset pari al raggio dell'utensile per rispettare le dimensioni delle parti
