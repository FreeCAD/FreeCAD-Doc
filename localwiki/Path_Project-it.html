Descrizione

Lo strumento Progetto serve per creare un oggetto Progetto nel documento attivo. L'oggetto Progetto ha lo scopo di riunire le diverse operazioni Percorso da esportare come programma G-Code.

Il Progetto si comporta come un oggetto Percorso composto, e può contenere qualsiasi numero di oggetti Percorso (Path). Contiene anche un oggetto Macchina con le diverse impostazioni della macchina che il programma in codice G dovrà alimentare.

Utilizzo

  1. Premere il pulsante Progetto
  2. Trascinare gli oggetti Path nel nuovo progetto nella struttura ad albero

Proprietà

  • DATAOutput File: Il percorso di un file g-code in cui questo progetto sarà esportato quando si preme lo strumento Post-elaborazione

Proprietà della macchina

  • DATAMachine Units: Specifies if the machine is configured in inches or metric system
  • DATAPost Processor: Allows to set a post processor script to be used when exporting the project
  • DATAX: X coordinate of the home position
  • DATAY: Y coordinate of the home position
  • DATAZ: Z coordinate of the home position
  • DATAX_Min: Minimal X position of the machine head
  • DATAX_Max: Maximal X position of the machine head
  • DATAY_Min: Minimal Y position of the machine head
  • DATAY_Max: Maximal Y position of the machine head
  • DATAZ_Min: Minimal Z position of the machine head
  • DATAZ_Max: Maximal Z position of the machine head
  • DATATooltable: Holds the table of tools of the machine (editable by double-clicking the machine)
