Description

La commande Terminer une Boucle complète une sélection d'arêtes formant une boucle.

Usage

Sélectionnez deux arêtes partageant un sommet commun. Ensuite, exécutez la commande. Les arêtes restantes qui forment la boucle seront automatiquement sélectionnées. Cette fonction est très utile dans d'autres ateliers.
