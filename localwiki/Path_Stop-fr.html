Description

Cet outil insert une commande Stop. Cette commande stoppera la machine, attendant une action de l'utilisateur pour continuer.

Note this does not stop the spindle.

Utilisation

  1. Pressez le bouton Stop

Propriétés

  • DATAStop: Defines if the Stop is optional or mandatory.
