Description

Le gestionnaire du magasin d'outils permet d'éditer les différents outils de la collection contenue dans l'objet machine d'un Travail. Cette collection est utilisée par les différentes opérations contenues dans le projet pour obtenir les informations nécessaires sur l'outil actuel.

Il sert également à la sélection d'un outil que vous souhaitez utiliser dans votre travail.

La manipulation est simple:

  • Importer ...: Importe une collection d'outils à partir d'un fichier XML.
    Avertissement

Ceci est actuellement en partie cassé et ne fonctionne pas si vous n'avez jamais eu de fichier xml auparavant.

  • Exporter ...: Exporte la collection vers un fichier XML.
  • Nouvel outil: ouvre une boîte de dialogue dans laquelle vous pouvez entrer les paramètres de votre outil. Voir Nouvel outil
  • Supprimer: supprime les lignes actuellement sélectionnées.
    Avertissement

Les outils sont supprimés de votre collection même si vous annulez la boîte de dialogue

  • Monter: Vous ne pouvez pas modifier le numéro d'outil, vous pouvez déplacer la ligne sélectionnée pour diminuer son numéro d'outil
  • Descendre: Vous pouvez déplacer la ligne sélectionnée vers le bas pour augmenter son numéro d'outil
  • Crée un ou des contrôleurs d'outils: Si vous cochez une ou plusieurs cases à gauche de la liste d'outils, ce bouton devient actif. Si vous cliquez dessus, les outils sélectionnés seront insérés dans votre travail actuel.

Utilisation

  1. Sélectionnez un Travail
  2. Appuyez sur le bouton Gestionnaire d'outils
  3. Créez de nouveaux outils ou ajustez les propriétés des outils existants.
    Définissez au moins le diamètre, FreeCAD en a besoin pour calculer la compensation de rayon. Depuis la version 0.17, c'est la seule valeur utilisée pour la création de parcours d'usinage. Cependant, si vous souhaitez utiliser l'outil de simulation plus tard, ajoutez également l'angle de coupe et la hauteur du tranchant.

Options

  • Les outils peuvent être réorganisés en utilisant les boutons de déplacement vers le haut / vers le bas
  • Les collections d'outils complètes peuvent être importées à partir de fichiers xml (le format collection de FreeCAD) ou à partir des collections de HeeksCAD
