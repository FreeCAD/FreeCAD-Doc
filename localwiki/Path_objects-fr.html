Path objects/fr

L'objet de base de l'atelier Path est l'objet Parcours d'usinage. Un objet Parcours d'usinage décrit le mouvement qu'une tête de machine doit effectuer pour découper un bloc de matériau, dans le cas de machines CNC. Les objets Parcours d'usinage peuvent contenir non seulement des informations de mouvement, mais également d'autres instructions pour la machine, telles que la vitesse de coupe de l'outil, ou quel outil doit être utilisé.


Les objets Parcours d'usinage peuvent être créés de différentes manières, en leur donnant manuellement les points par lesquels ils doivent passer, ou en les dérivant d'un objet FreeCAD filaire, ou, plus intéressant, en les dérivant automatiquement d'un objet 3D existant, ou de certaines parties de celui ci. Ceci est une tâche complexe, et les outils ci-dessous sont là pour faire ce travail, ou partiellement, de différentes façons. Cela nécessite une certaine expérience pour les connaître tous et choisir celui qui convient le mieux à la tâche particulière dont vous avez besoin. Parfois, vous pouvez également préférer construire vos parcours d'usinage pas à pas, en utilisant des outils partiels, au lieu d'utiliser les outils les plus complexes.


Les objets Parcours d'usinage peuvent également être combinés à des objets Composés ou Réseaux, vous permettant d'utiliser différents outils pour exécuter l'ensemble les différents cycles de l'opération d'usinage, et de les assembler en un seul parcours final.


Finalement, les objets Projet vous permettent de gérer les différentes opérations d'usinage qui doivent être effectuées par une même machine et d'ajouter des informations spécifiques à la machine comme pour un magasin des outils disponibles sur la machine d'usinage, chacun avec ses caractéristiques. Certains outils de l'atelier Path seront alors en mesure de se référer à ce magasin pour obtenir des informations telles que le diamètre de certains outils de coupe.


FreeCAD est aussi capable d'importer et utiliser des fichiers G-Code, pour lesquels un script de pré-traitement a été écrit.
