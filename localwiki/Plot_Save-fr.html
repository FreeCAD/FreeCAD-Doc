Description

Cet outil active l'enregistrement du fichier Plot à l'endroit désiré.

Avec cette fonction, vous pouvez également sélectionner, la taille, et, la résolution de l'image de sortie.

Utilisation

Cliquez sur l'onglet du tracé que vous souhaitez enregistrer, et, exécuter sur l'outil. Utilisez la touche de sélection du chemin d'accès, afin d'afficher une boîte de dialogue, où vous pouvez choisir, le format, et, l'emplacement de l'image de sortie.

Path selection button

Options

  • File path: Vous pouvez définir le chemin de l'image de sortie (y compris l'extension du format) sur la ligne de texte.
  • Size: Vous pouvez spécifier, la largeur, et, la hauteur de l'image.
  • dpi: Vous pouvez définir la résolution (points par pouce). La résolution finale (en pixels) sera le produit, de la largeur par hauteur.

Script

L'enregistrement de l'outil Plot peut être utilisé dans une macro, et, dans la console Python, en utilisant la fonction suivante : 

save(str, (float, float), float)
  • save (str, : Sauve le fichier Plot au chemin défini
    (float, float), : détermine, la largeur, et, la hauteur de l'image en pouces (Inch)
    float) : détermine, la résolution en points par pouce (Inch).

Exemple : 

import Plot
Plot.save("~/example.pdf", (12.8, 9.6), 50)

Ce code sauvera une image de 12.8 x 9.6 pouces (Inch), équivalent à 640 x 480 pixels, dans un fichier au format .pdf.
