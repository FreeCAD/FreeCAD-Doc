Descrizione

Importa punti da un file nuvola di punti.

Uso

Cliccare su o scegliere Punti → Importa punti... dal menu principale.

Limitazioni

Le coordinate dei punti devono essere memorizzati in un file di testo.
L'estensione del file deve essere ".asc".
Le coordinate devono utilizzare il punto (NON la virgola) e devono essere separate da uno spazio.

Esempio di file 

0 0 0
1.4562 -7.2354 12.2367
5.9423 3.1728 -17.6439

Note

Per provare è possibile utilizzare una versione low poly del famoso stanford bunny.
È possibile scaricare il file .asc pre preparato cliccando quì.

Dopo l'importazione, i punti sono selezionabili nella vista 3D e, per esempio, nell'ambiente Draft possono essere utilizzati per agganciare dei punti finali a questi punti.
Ma, a differenza dei punti inseriti con lo strumento Punto di Draft, nella vista ad albero questi punti sono mostrati come un unico oggetto "nuvola di punti".
