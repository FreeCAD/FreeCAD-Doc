Points Module/it

Il Modulo Punti è fatto per fornire gli strumenti specifici per lavorare con point clouds (nuvole di punti). È ancora in sviluppo.

Questi strumenti sono già disponibili:

  • Importa punti: importa le coordinate di punti 3D da un file di testo
  • Esporta punti: esporta le coordinate di punti 3D in un file di testo
  • Polygon Pick: to do

Le coordinate dei punti devono essere memorizzate in un file di testo.
L'estensione del file deve essere ".asc".
Le coordinate devono utilizzare il punto (non la virgola) e devono essere separate da uno spazio.
