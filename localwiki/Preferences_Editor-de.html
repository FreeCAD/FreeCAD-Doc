Preferences Editor/de

Die Einstellungen für FreeCAD befinden sich im Bearbeiten-Menü, Untermenü → Einstellungen(Preferences falls Original-Sprache Engl. eingestellt)

FreeCAD´s Funktionalität ist in verschiedene Module unterteilt, wobei jedes Modul zuständig ist, für die Bearbeitung eines bestimmten Arbeitsbereiches. FreeCAD verwendet ein Konzept namens Spätes-laden, das bedeutet, dass Komponenten nur geladen werden, wenn sie gebraucht werden. Sie haben vielleicht schon bemerkt, dass, wenn Sie einen Arbeitsbereich aus der Symbolleiste von FreeCAD wählen, dass dieser Arbeitsbereich und alle seine Komponenten erst zu diesem Zeitpunkt geladen werden. Das trifft auch auf die Einstellungen zu.

Ohne ein Modul geladen zu haben, stehen Ihnen 2 Konfigurations-Bereiche zur Verfügung, verantwortlich für allgemeinen Einstellungen des Programms und für die Anzeige-Einstellungen

Die allgemeinen Einstellungen

Starten Sie FreeCAD beispielsweise ohne einen Arbeitsbereich zu laden, haben Sie dann auch nur ein minimales Einstellungs-Fenster. So wie Sie zusätzliche Module laden, werden neue Abschnitte in dem Einstellungs-Fenster angezeigt, die Ihnen ermöglichen, die Details der einzelnen Arbeitsbereich zu konfigurieren.
