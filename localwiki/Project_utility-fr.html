Description

L'utilitaire de projet est un outil qui sert à réparer un fichier .FCStd dans le cas ou pour une raison quelconque aurait été corrompu. Au départ le fichier FCStd est une archive zip, l'outil offre une fonction pour extraire le projet, pour que l'utilisateur puisse par exemple faire des modifications dans le Document.xml (ou tout autre chose) et puis reconstruire à nouveau le fichier valide de FCStd.
