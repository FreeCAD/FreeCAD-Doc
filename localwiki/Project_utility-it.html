Utilità di progetto è uno strumento per riparare un file FCStd che per qualche motivo è rovinato. Dato che un file FCStd è un archivio zip, lo strumento offre una funzione per estrarre il progetto e apportarvi delle modifiche, ad esempio sul Document.XML, oppure fare altre correzioni, e quindi ricostruire un file di FCStd valido.
