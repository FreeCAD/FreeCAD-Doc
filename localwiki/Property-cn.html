Property/cn

属性 是指与 FreeCAD 文档或对象密切相关一些信息，例如数字编号或者字符串等。属性可以被查看，当然如果允许的话也可以使用属性编辑器进行相关属性的修改。

对于 FreeCAD 来说，属性是非常重要的部分，因为从一开始就是利用参数化对象来进行工作的，而这些对象仅仅是用其属性来定义的。

FreeCAD 中自定义脚本式对象具有以下类型的属性： 

Boolean
Float
FloatList
FloatConstraint
Angle
Distance
Integer
IntegerConstraint
Percent
Enumeration
IntegerList
String
StringList
Link
LinkList
Matrix
Vector
VectorList
Placement
PlacementLink
Color
ColorList
Material
Path
File
FileIncluded
PartShape
FilletContour
Circle
