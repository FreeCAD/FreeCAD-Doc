Beschreibung

Fügen Sie einen neuen Roboter (KUKA IR500) in die Szene ein.

Anwendung

Klicken Sie auf , um einen neuen Roboter (KUKA IR500) in die Szene einzufügen.

Das Einfügen von anderen/weiteren Robotern kann auf zwei Arten erfolgen:

  • über das Top-Menü
  1. Wählen Sie Roboter → Roboter einfügen aus dem Top-Menü.
  2. Wählen Sie einen der vier vordefinierten Roboter.
  • über den "Aufgaben"-Reiter
  1. Stellen Sie sicher, dass ein neues Dokument erzeugt wird/wurde (aktivieren über Doppelklick auf Dokument in der Baumansicht).
  2. Wechseln Sie zum "Aufgaben"-Reiter in der Baumansicht.
  3. Wählen Sie einen der vier vordefinierten Roboter.

Beschränkungen

[0.15/0.16]
Nur Roboter mit bis zu sechs Achsen mit Drehbewegungen sind definierbar.
Keine Roboter mit translationalen Bewegungen möglich.
Der Roboter muss über Code und eine vrlm-Datei definiert werden, mehr Informationen finden Sie unter Robot 6-Axis.

Anmerkungen

Die vordefinierten Roboter sind

  • KUKA IR500
  • KUKA IR210
  • KUKA IR125
  • KUKA IR16
