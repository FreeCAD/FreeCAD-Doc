Descrizione

Inserisce un nuovo Robot (KUKA IR500) nella scena.

Utilizzo

Cliccare su per inserire un robot KUKA IR500 nella scena.

L'inserimento di altri o diversi robot può essere fatto in due modi.

  • tramite il menu principale
  1. Selezionare Robot → Inserire Robot dal menu in alto.
  2. Selezionare uno dei quattro robot predefiniti.
  • tramite la tabella "Azioni"
  1. Accertarsi che sia attivato un nuovo documento (per attivare un documento fare doppio clic sul suo nome nella vista ad albero).
  2. Passare nella scheda "Azioni".
  3. Selezionare uno dei quattro robot predefiniti.

Limitazioni

[0.15/0.16]
Sono definibili solo robot con movimento rotatorio fino a sei assi.
Non sono possibili robot con movimenti di traslazione.
Il robot deve essere definito tramite codice e file vrml, per maggiori informazioni fare riferimento alla pagina Robot a 6 Assi.

Note

I robot predefiniti sono

  • KUKA IR500
  • KUKA IR210
  • KUKA IR125
  • KUKA IR16
