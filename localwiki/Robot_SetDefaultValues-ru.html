Описание

Устанавливает значения по умолчанию для скорости, ускорения и целостности при создании промежуточных точек.


Использование

Выберите в верхнем меню Робот → Установка значений по умолчанию, чтобы определить

  • скорость
  • продолжительность (режим)
  • ускорение


Примечание

Программные значения по умолчанию следующие

  • скорость = 1 m/s
  • продолжительность = false
  • ускорение = 1 m/s^2
