Description

Set the default values for way-point creation.


Usage

Choose Robot → Set default values from the top menu to define

  • speed
  • continous (mode)
  • acceleration


Notes

The program's default values are for

  • speed = 1 m/s
  • continous = false
  • acceleration = 1 m/s^2
