Selection API/jp

SelectionモジュールはFreeCADGuiモジュールの一部です。

例： 

import FreeCADGui
sel = FreeCADGui.Selection.getSelection()
addSelection ( FreeCAD.Object )

説明: オブジェクトをSelectionに追加します。

戻り値:


clearSelection ( [string] )

説明: 与えられたドキュメント名のSelectionをクリアします。ドキュメントが与えられていない場合は全てのSelectionがクリアされます。

戻り値:


getSelection ( [string] )

説明: 与えられたドキュメント名の選択されているオブジェクトのリストを返します。ドキュメントが与えられていない場合は全てのSelectionが返されます。

戻り値: オブジェクトのリスト


isSelected ( FreeCAD.Object )

説明: 与えられたおb図イェクトが選択されているかどうかをチェックします。

戻り値:


removeSelection ( FreeCAD.Object )

説明: Selectionからオブジェクトを削除します。
