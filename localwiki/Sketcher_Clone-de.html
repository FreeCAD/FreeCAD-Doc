Beschreibung

Klont ein Sketcher-Element.


Usage

  1. Select sketcher element for clone operation.
  2. Clíck on or choose Sketch → Sketcher tools → Clone from the top menu.
  3. Place clone in the 3D-View.

No extra constraints for clone-behaviour are added.
