Description

Copier un élément du sketcher .


Utilisation

  1. Sélectionnez un élément de dessin pour l'opération de copie
  2. Cliquez sur ou choisissez Esquisse → Outils d'esquisse → Copier dans le menu principal.
  3. Placez la copie dans la vue 3D.

Aucune contrainte supplémentaire n'est ajoutée.


Notes

Disponible depuis FreeCAD V0.16.
