Description

Édite et permet de modifier une esquisse existante.

Utilisation

Cliquez avec le bouton droit sur une esquisse dans l'arborescence et choisissez "Edit Sketch" ou sélectionnez sketch dans l'arborescence et choisissez "Sketch> Edit Sketch" dans le menu déroulant. Choisir Leave Sketch met fin au mode d'édition.
