Description

Créer un réseau à partir d'éléments du sketcher .

Utilisation

  1. Sélectionner les éléments de l'esquisse dans Tasks-Panel ou dans la vue 3D
  2. Cliquer sur ou choisisser Esquisse → Outils Sketcher → RectangularArray dans le menu principal.
  3. Un menu apparaîtra vous demandant les différentes options du tableau
