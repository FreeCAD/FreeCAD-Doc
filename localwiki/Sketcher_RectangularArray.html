Description

Creates an array of selected sketcher elements.

Usage

  1. Select sketcher element(s) in Tasks-Panel or in 3D-View.
  2. Click on or choose Sketch → Sketcher tools → RectangularArray from the top menu.
  3. A menu will pop up asking you for the different options of the array.
  4. Select endpoint for operation in 3D-View.
