Description

Sélectionne les contraintes d'un élément d'esquisse.


Utilisation

  1. Sélectionnez l'élément sketcher dans le panneau Tâches ou dans la vue 3D.
  2. Cliquez sur ou choisissez Esquisse → Sketcher tools → Sélectionnez Contraintes dans le menu principal.
  3. Les contraintes associées seront surlignées dans le panneau Tâches.


Notes

Ne fonctionne pas avec les contraintes Point-on-Object. Disponible depuis FreeCAD V0.15.
