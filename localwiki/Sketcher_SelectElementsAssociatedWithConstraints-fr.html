Description

Selection sketcher Elements associés avec constraintes.


Utilisation

  1. Sélectionner la contrainte dans le panneau Tâches ou dans la vue 3D
  2. Cliquer sur ou choisisser Esquisse → OutilsSketcher → Sélectionner Éléments associés aux contraintes dans le menu supérieur.
  3. Les éléments d'esquisse associés seront surlignés dans le panneau Tâches.
