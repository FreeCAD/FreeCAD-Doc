Description

Select sketcher Elements associated with constraints.


Use

  1. Select constraint in the Tasks-Panel or in the 3D-View.
  2. Click on or choose Sketch → Sketcher tools → Select Elements associated with constraints from the top menu.
  3. The associated sketcher element(s) will be highlighted in the Tasks-Panel.
