Beschreibung

Wählt die horizontale Achse einer Skizze aus.


Use

  1. Enter edit mode of sketch.
  2. Choose Sketch → Sketcher tools → Select Horizontal Axis from the top menu.
  3. The horizontal axis of the sketch will be selected.


Bemerkungen

Verfügbar seit FreeCAD v 0.15.
