Description

Sélectionne l'axe horizontal d'une esquisse.


Utilisation

  1. Entrez le mode d'édition de l'esquisse.
  2. Choisir Esquisse → Sketcher tools → 32px Sélectionner Horizontal Axe dans le menu principal.
  3. L'axe horizontal de l'esquisse sera sélectionnée.


Notes

Disponible depuis FreeCAD V0.15.
