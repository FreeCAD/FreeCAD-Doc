Description

Sélectionne l'origine d'une esquisse.


Utilisation

  1. Entrez le mode d'édition de l'esquisse.
  2. Choisir Esquisse → Sketcher tools → Sélectionnez Origine dans le menu principal.
  3. L'origine de l'esquisse sera sélectionnée.


Notes

Disponible depuis FreeCAD V0.15.
