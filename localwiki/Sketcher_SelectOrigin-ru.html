Description

Selects the origin of a sketch.


Use

  1. Enter edit mode of sketch.
  2. Choose Sketch → Sketcher tools → Select Origin from the top menu.
  3. The origin of the sketch will be selected.


Notes

Available since FreeCAD v 0.15.
