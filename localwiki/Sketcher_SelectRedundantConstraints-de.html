Beschreibung

Wählt die redundanten Beschränkungen einer Skizze aus.


Use

  1. Enter edit mode of sketch.
  2. Choose Sketch → Sketcher tools → Select Redundant Constraints from the top menu.
  3. The redundant constraints of the sketch will be selected.


Bemerkungen

Verfügbar seit FreeCAD v 0.15.
