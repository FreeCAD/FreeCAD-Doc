Description

Sélectionne l'axe vertical d'une esquisse.


Utilisation

  1. Entrez le mode d'édition de l'esquisse.
  2. Choisir Esquisse → Sketcher tools → Sélectionner Vertical Axe dans le menu principal.
  3. L'axe vertical de l'esquisse sera sélectionnée.


Notes

Disponible depuis FreeCAD V0.15.
