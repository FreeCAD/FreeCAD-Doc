Description

Selects the vertical axis of a sketch.


Use

  1. Enter edit mode of sketch.
  2. Choose Sketch → Sketcher tools → Select Vertical Axis from the top menu.
  3. The vertical axis of the sketch will be selected.


Notes

Available since FreeCAD v 0.15.
