Beschreibung

Kopiert ein Sketcher-Element symmetrisch zu einer ausgewählten Linie.


Usage

  1. Select arc/line/rectangle to be copied and line for copy operation.
  2. Choose Sketch → Sketcher tools → Symmetry from the top menu.
  3. Chosen element will be copied symetrical to the chosen second element.

No extra symmetry constraint will be added.


Bemerkungen

Verfügbar seit FreeCAD v 0.16.
