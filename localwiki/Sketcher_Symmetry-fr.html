Description

Copies un sketcher éléments symétriquement a une ligne choisie.


Utilisation

  1. Sélectionner arc / ligne / rectangle à copier et ligne pour l'opération de copie
  2. Choisir Esquisse → Outils d'esquisse → Symétrie dans le menu principal

L'élément choisi sera copié symétriquement par rapport au second élément choisi. Aucune contrainte de symétrie supplémentaire ne sera ajoutée.


Notes

Disponible depuis FreeCAD V0.16.
