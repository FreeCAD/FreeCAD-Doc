Spinning/sv

Spinning is the ability of FreeCAD to display a dynamic rotation of the modell without user interaction.


Bruk

Frirotation är som standard aktiverat. För att aktivera eller stäng av frirotation, gå till Redigera→Alternativ menyn, gå till Display/3D Vy inställningarna och klicka på Aktivera animering.

För att få ett 3D-objekt att frirotera, rotera den med musen och släpp musknapparna innan du slutar musrörelsen. Objektet fortsätter att rotera.

View Turntable

Another way to put in rotation an object is use Tools -> View turntable... command
