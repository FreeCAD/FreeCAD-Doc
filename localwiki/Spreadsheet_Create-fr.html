Description

Cet outil ajoute un nouvel tableau objet pour le document actif. Le tableau peut alors être édité avec les méthodes d'édition FreeCAD standards (clic droit dans l'arborescence ou en utilisant la commande Std Edit)

Utilisation

  1. Appuyez sur le bouton Spreadsheet Create
  2. Appuyez sur la touche Std Edit pour modifier le contenu du tableur
  3. Modifier le contenu des cellules
  4. Fermer l'onglet de l'éditeur de tableur lorsque vous avez terminé, les valeurs sont enregistrées automatiquement lorsque vous les tapez

Options

  • Le contenu des cellules peut être modifié en double-cliquant dessus, ou en utilisant la ligne d'édition en haut de l'éditeur
  • Les valeurs sont enregistrées à la volée et stockées dans le document FreeCAD * Vous pouvez entrer du texte, des chiffres , ou des formules, en démarrant votre entrée avec le caractère =. Seulement les signes arithmétiques simples (+, -, * et /) et les parenthèses sont pris en charge pour le moment. Exemple: = B4 * 2 pour afficher le contenu de la cellule B4 multiplié par 2.
  • Vous pouvez insérer des valeurs automatiquement récupérées depuis le modèle en utilisant un objet contrôleur de Cellule
  • Les objets tableurs peuvent être importés et exportés vers le format csv

Script

Les Tableurs peuvent être créés à partir de scripts Python et de macros en utilisant la fonction makeSpreadsheet (): 

 import Spreadsheet
 mySpreadsheet = Spreadsheet.makeSpreadsheet()

Le contenu du tableur peut alors être manipulé comme ceci: 

 mySpreadsheet.Proxy.A1 = "my text"
 print mySpreadsheet.Proxy.A1

Les objets tableurs peuvent également être créés uniquement dans l'espace de python, sans existence dans le document FreeCAD. Ils ne seront pas enregistrés lorsque vous fermerez le fichier FreeCAD, ceci est destiné uniquement à des fins temporaires: 

 import Spreadsheet
 myPySpreadsheet = Spreadsheet.Spreadsheet()
 myPySpreadsheet.A1 = "my other text"
 print myPySpreadsheet.A1

Limitations

  • Cet outil n'est pas disponible en dessous de la version FreeCAD 0,14
  • La taille maximale d'une feuille de calcul est actuellement limité à 26x256 cellules (colonnes entre A et Z et rangées entre 1 et 256)
  • Colonnes entre A et ZZ et lignes entre 1 et 16384 Disponible dans la version 0.17
