Description

Cet outil ajoute un nouvel objet contrôleur de propriété à un objet spreadsheet sélectionné. Le contrôleur peut alors gérer une certaine propriété d'un certain objet, et mettre à jour avec la valeur d'une cellule du tableur, chaque fois que des cellules changent.

Utilisation

  1. Sélectionnez un objet Spreadsheet existant
  2. Appuyez sur le bouton Property Controller
  3. ajuster les propriétés nécessaires

Propriétés

  • DONNÉESType de cible: Le type de l'élément à contrôler: une propriété de l'objet cible, ou une contrainte sur la base donnée, si l'objet cible est un croquis
  • DONNÉESTarget Object: l'objet à contrôler
  • DONNÉESTarget Property: le nom de la propriété à contrôler, ou le numéro de la contrainte liée à une référence
  • DONNÉESCell: la cellule du tableur hôte pour prendre la valeur de

Script

Les tableurs peuvent être créés à partir de scripts Python et de macros en utilisant la fonction makeSpreadsheetPropertyController (): 

 import Spreadsheet
 mySpreadsheet = Spreadsheet.makeSpreadsheet()
 controller = Spreadsheet.makeSpreadsheetPropertyController(mySpreadsheet)

Limitations

  • Cet outil n'est pas disponible pour les versions antérieures à FreeCAD 0,14
  • Actuellement, il n'est pas possible d'annuler les modifications que le contrôleur apporte à son objet contrôlé.
