Spreadsheet legacy/fr

L'atelier feuille de calcul vous permet de créer d'éditer , d'effectuer des calculs, de récupérer des données à partir d'un modèle, et d'exporter son contenu vers d'autres applications de tableur comme LibreOffice ou Microsoft Excel. L'atelier feuille de calcul dispose d'un éditeur interne, qui vous permet de modifier le contenu de votre feuille.


Tools

  • Spreadsheet: Crée une nouvelle feuille dans votre document actif.
  • Cell Controller: Ajoute une cellule contrôleur dans la feuille de calcul active.
  • Property Controller: Ajoute une propriété contrôleur dans la feuille de calcul sélectionnée.

Format de fichier

  • Comma-separated values (.CSV): Importe et exporte les données du tableur en format de fichier .csv
