Spreadsheet legacy/it

L'ambiente Spreadsheet consente di creare e modificare un foglio di calcolo, inoltre permette di eseguire le operazioni, di recuperare i dati da un modello e di esportare il suo contenuto in altre applicazioni analoghe, come LibreOffice o Microsoft Excel. L'ambiente Spreadsheet è dotato di un proprio editor interno che consente di modificare il contenuto del foglio.


Strumenti

  • Crea foglio: Crea un nuovo oggetto spreadsheet nel documento attivo.
  • Controllo Cella: Aggiunge un controllo sulla cella di un foglio di calcolo selezionato.
  • Property Controller: Aggiunge un controllo delle proprietà di un foglio di calcolo selezionato.

Formati dei file

  • Comma-separated values (.CSV): Importa ed esporta i dati dei fogli di calcolo da, oppure in, un file .csv .
