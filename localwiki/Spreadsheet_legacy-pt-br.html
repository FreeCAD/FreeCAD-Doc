Spreadsheet legacy/pt-br

The spreadsheet workbench allows you to create and edit spreadsheets, perform calculations, and retrieve data from a model, and export its contents to other spreadsheet applications such as LibreOffice or Microsoft Excel. The spreadsheet workbench features an internal editor, that allows you to edit the spreadsheet contents.


Tools

  • Spreadsheet: Creates a new spreadsheet object in the active document.
  • Cell Controller: Adds a cell controller to a selected spreadsheet.
  • Property Controller: Adds a property controller to a selected spreadsheet.

File formats

  • Comma-separated values (.CSV): Imports and exports spreadsheet data from/to a .csv file
