Panoramica

Il menu principale standard di FreeCAD consente di accedere alle funzioni che vengono utilizzate in tutti gli ambienti di lavoro. Pu√≤ avere un aspetto leggermente diverso, secondo l'ambiente desktop (KDE, Gnome, Windows, OS X, ecc).

File

File fornisce le funzioni relative a input e output, ad esempio Apri, Stampa, Esportazione, ecc.

Modifica

Modifica contiene le funzioni relative alla modifica del modello, ad esempio Copia, Incolla, etc.

Visualizza

Visualizza contiene le funzioni relative alla visualizzazione del modello, come ad esempio quali pannelli mostrare, zoom, ecc.

Strumenti

Strumenti fornisce le funzioni per il debug dei modelli e per personalizzare il comportamento di FreeCAD.

Macro

Macro fornisce le funzioni relative alla registrazione e alla esecuzione di gruppi di comandi FreeCAD.

Ambiente

Ambiente contiene le opzioni per le funzioni del workbench in uso. Contiene le funzioni specifiche dell'ambiente che si sta utilizzando.

Misure

Misure contiene le opzioni per le funzioni di misura delle distanze Delta o 3D. Questo menu appare solo quando si utilizza l'ambiente Part.

Finestre

Finestre permette di gestire le finestre dei documenti FreeCAD.

Aiuto

Aiuto consente di accedere al sistema di aiuto di FreeCAD e fornisce informazioni sulla propria installazione FreeCAD.
