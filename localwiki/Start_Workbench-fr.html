Depuis Premier Pas:


Lorsque vous démarrez FreeCAD pour la première fois, il vous est présenter avec le centre de démarrage



Le Start Center (centre de départ) permet de passer rapidement à l'un des Ateliers les plus usuels, ouvrir un des fichiers récents, ou voir les dernières nouvelles du monde de FreeCAD. Vous pouvez modifier l'atelier par défaut dans l'éditeur de préférences.
