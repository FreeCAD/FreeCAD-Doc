Description

La commande ferme le document actif et vous permet de continuer à travailler sur d'autres documents. Si le document n'a pas été enregistré, ou s'il a été modifié, un message apparaît et vous demande si vous souhaitez l'enregistrer:

Dans l'affirmative,

  • Si le document a déjà été enregistré, utilisez le nom et l'ensemble de l'adresse,
  • Si le document n'a jamais été enregistré, ouvrez la fenêtre pour définir le chemin et le nom du document:


La commande Fermer tout ferme tous les documents. Avec cette commande, vous quittez FreeCAD.
