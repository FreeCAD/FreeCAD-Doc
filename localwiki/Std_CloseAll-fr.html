Description

Cette commande ferme tous les documents du projet, même ceux qui n'étaient pas actifs. Avec cette commande, vous quittez FreeCAD. S' il y a des documents non enregistrés ou modifiés, et elle affiche un message vous demandant si vous souhaitez enregistrer:

Dans l'affirmative,

  • Si le document a déjà été enregistré, utilisez le nom et l'ensemble de l'adresse,
  • Si le document n'a jamais été enregistré, ouvrez la fenêtre pour définir le chemin et le nom du document:


La commande Fermer ferme le document actif et vous permet de continuer à travailler sur d'autres documents de projet.
