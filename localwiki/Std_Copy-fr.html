Description

Fonction de copie d'un ou plusieurs objet(s) de la même manière que sur un traitement de texte courant. Les objets peuvent être copiés entre deux documents.

  1. Sélectionnez un ou plusieurs objet(s) à dupliquer.
  2. Pressez sur la touche , ou sur les touches ctrl + C ou sur le menu Edition → Copier.

Limitations

  • En général, il n'est pas possible de copier des objets FreeCAD dans d'autres applications, ou copier des objets provenant d'autres applications dans un document FreeCAD.
