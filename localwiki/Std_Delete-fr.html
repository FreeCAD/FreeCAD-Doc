Description

Cette commande éfface un ou les objets sélectionné(s). S'il n'y a pas d'objet sélectionné, la commande est nulle.

Sélectionnez l'objet à éffacer, puis cliquez sur le bouton Éffacer ou sur la touche du clavier Supp ou allez dans le menu Édition → Effacer.

L'opération peut être annulée par les fonctions annuler, et, rétablir.
