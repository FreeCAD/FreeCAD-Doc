Description

Cette fonction exécute une macro dans l'éditeur actif. la commande est accessible par Macro → Lancer la macro, en cliquant sur le bouton

Utilisation

Une fois entré dans l' éditeur de macro, le bouton Exécuter la macro se colorie en vert . Cliquer sur ce bouton pour exécuter la macro.
