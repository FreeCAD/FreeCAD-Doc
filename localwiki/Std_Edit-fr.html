Description

Cette commande permet d'entrer ou de quitter le mode édition d'un objet sélectionné. Vous devez soit sélectionner un objet (puis passer en mode édition) ou en vous êtes en mode édition (puis quitter le mode édition) pour que cette commande fonctionne.

Le comportement produit lors de l'entrée en mode édition, dépend du type d'objet. Certains types d'objets ne définissent pas de comportement en mode édition, dans ce cas, c'est le comportement standard de l'application (un widget de transformation) qui est utilisé.

Vous pouvez également effectuer la même action, en double-cliquant sur un objet dans l'arborescence.

Script

Cette commande, passe en mode édition d'un objet déterminé : 

FreeCADGui.ActiveDocument.setEdit("myObjectName",0)

Cette commande ferme le mode édition : 

FreeCADGui.ActiveDocument.resetEdit()
