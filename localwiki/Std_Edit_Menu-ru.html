Std Edit Menu/ru

Правка


Отменить откатывает последнюю команду.

Вернуть повторяет последнюю команду, реверсированную командой Отменить.

Вырезать помещает выбранные объекты в буфер обмена и удаляет их.

Копировать помещает выбранные объекты в буфер обмена.

Вставить помещает содержимое буфера обмена в текущий документ.

Дубликат копирует объект и вставляет копию на то же место, что и оригинал.

Обновить перерисовывает дисплей.

Выделить область позволяет выделить объекты в прямоугольной области.

Выбрать всё выделяет все объекты в текущем документе.

Удалить удаляет всё выбранное.

Размещение... открывает диалог размещения для выбранного объекта.

Выравнивание... открывает диалог выравнивания для выбранных объектов.

Редактировать/закончить редактирование переключает режим редактирования для выбранных объектов.

Параметры открывает диалог пользовательских предпочтений.
