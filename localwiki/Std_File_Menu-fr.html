Std File Menu/fr


File

New creates a new FreeCAD document.

Open opens an existing FreeCAD document file.

Fermer ferme le document actif.

Fermer tout ferme tous les documents ouverts.

Enregistrer enregistre le document actif sur disque.

Enregistrer sous... enregistre le document actif sous un autre nom.

Save a Copy v 0.16

Revert v 0.16

Importer... importe un fichier d'une autre application dans le document actif de FreeCAD.

Exporter... exporte le document actif dans un format reconnu par une autre application.

Fusionner un projet... ajoute et fusionne un projet FreeCAD dans le document actif.

Informations sur le projet... permet de personnaliser le projet en ajoutant le nom de l'auteur, révisions, données, etc.

Imprimer... imprime le contenu de la fenêtre du document actif sur papier.

Aperçu avant impression... affiche l'image de votre projet tel qu'il sera sur votre feuille.

Exporter au format PDF... exporte le document actif dans un fichier au format PDF visualisable avec Adobe reader.

Fichiers récent... ce menu ouvre un boite de dialogue et liste les derniers documents ouverts dans FreeCAD quelque soit l'origine ou l'extension.

Quitter quitte FreeCAD.
