Std File Menu/it


File


Nuovo crea un nuovo documento di FreeCAD.

Apri apre un documento esistente di FreeCAD.

Chiudi chiude il documento attivo.

Chiudi tutto chiude tutti i documenti aperti.

Salva salva il documento attivo su disco.

Salva con nome... salva con un nuovo nome il documento attivo su disco.

Salva una Copia v 0.16

Ripristina v 0.16

Importa Apre e carica file di progetti aggiungendoli al documento attivo nell'ambiente di lavoro di FreeCAD.

Esporta Permette di esportare in diversi formati un oggetto selezionato in un documento attivo di FreeCAD.

Unisci progetto... Questa funzione inserisce nel documento attivo di un progetto in corso un progetto archiviato in precedenza.

Informazioni sul progetto Fornisce informazioni sul progetto: nome, autore, ecc. e permette di inserire un commento al progetto.

Stampa apre il dialogo per selezionare la stampante, impostare le preferenze di stampa disponibili e avviare il processo di stampa.

Anteprima di stampa permette di definire l'aspetto della pagina da stampare, dimensioni dell'immagine, bordi, ecc...

Esporta PDF esporta la vista corrente in un file pdf.

File recenti espande una lista che elenca gli ultimi documenti utilizzati.

Esci esce dall'applicazione.
