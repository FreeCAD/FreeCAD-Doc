Std File Menu/ru


Файл

Создать создаёт новый документ FreeCAD.

Открыть открывает файл существующего документа FreeCAD.

Закрыть закрывает текущий документ.

Закрыть всё закрывает все открытые документы.

Сохранить записывает текущий документ на диск.

Сохранить как записывает текущий документ на диск под новым именем.

Сохранить копию v 0.16

Revert v 0.16

Импортировать: Читает файл, созданный другим приложением, в текущий документ.

Экспортировать: Пишет текущий документ в файл для использования в другом приложении.

Объединить проект добавляет объекты из файла FreeCAD к текущему документу.

Информация о проекте: добавляет/обновляет метаданные проекта - автор, версия, дата и т.д.

Печать печатает содержимое текущего окна документа.

Предварительный просмотр показывает на экране изображение для печати окна текущего документа.

Экспортировать PDF записывает окно текущего документа в файл PDF.

Последние файлы даёт быстрый доступ к последним открытым файлам документов.

Выход выключает FreeCAD.
