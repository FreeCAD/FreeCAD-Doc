Registrare una macro

Descrizione

Questa funzione registra una macro. La funzione è accessibile tramite Macro → Registra macro ..., oppure tramite il pulsante della barra degli strumenti delle macro .

Utilizzo

Avviare la funzione con uno dei modi indicati prima. Appare la seguente finestra:

  • Nome della macro : Inserire il nome della nuova macro.
  • Percorso della macro : Posizione della libreria delle macro. Cliccare su ... , per modificare il percorso (path).
  • Registra : Registrare la macro. Avvia la registrazione di tutto quello che viene fatto in seguito. Il pulsante della barra degli strumenti delle macro assume il colore verde .
  • Ferma : Arrestare la registrazione della macro.
  • Annulla : Annulla la funzione e chiude la finestra.

A lavoro ultimato cliccare su per interrompere la registrazione della macro.

La macro può essere recuperata tramite Macro → Macro ..., oppure tramite il pulsante della barra degli strumenti delle macro per essere eseguita o modificata a piacere.

Terminare la registrazione della macro

Descrizione

Questa funzione arresta la registrazione di una macro. La funzione è accessibile da Macro → Interrompi la registrazione, oppure facendo clic sul pulsante della barra degli strumenti delle macro .

Utilizzo

Quando si avvia la registrazione di una macro il pulsante della barra degli strumenti delle macro assume il colore verde .

Cliccare sul quadratino verde, per terminare la registrazione. Il file della macro viene chiuso e salvato automaticamente sul disco nella posizione prevista.

La macro può essere recuperata tramite Macro → Macro ..., oppure tramite il pulsante della barra degli strumenti delle macro per essere eseguita o modificata a piacere.

Modificare, eseguire o creare una Macro

Descrizione

Questa funzione permette di creare, eseguire o modificare la macro selezionata.

Utlizzo

È accessibile tramite Macro → Macro ..., oppure tramite il pulsante della barra degli strumenti delle macro .

  • Nome della macro : la finestra elenca tutte le macro registrate che si trovano nel Percorso macro.
  • Esegui : Avvia la macro selezionata.
  • Chiudi : Chiude la finestra e esce dalla funzione
  • Crea  : Apre l'editor Python di FreeCAD, e apre una finestra di dialogo per assegnare un nome alla macro, e apre un nuovo documento nell'area di lavoro. Il file è un file Python, e verrà salvato con l'estensione .FCmacro. Il codice del file è identico a quello di un file Python.py, l'estensione è stata scelta per associare il file a FreeCAD.
  • Elimina : Elimina la macro selezionata.
  • Modifica : Modifica la macro selezionata nell'editor Python di FreeCAD.
  • Percorso macro : Posizione della libreria delle macro. Cliccare su ... , per modificare il percorso (path).

Eseguire la macro

Descrizione

Questa funzione esegue la macro selezionata o attiva nell'editor. È accessibili tramite Macro → Esegui la macro, oppure tramite il pulsante della barra degli strumenti delle macro: .

Utilizzo

Quando si entra in Editor della macro, il pulsante assume il colore verde . Fare clic su questo pulsante per eseguire la macro.
