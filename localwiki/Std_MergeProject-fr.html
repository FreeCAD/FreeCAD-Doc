Description

Cette fonction insère dans le document actif d'un projet en cours un projet précédemment enregistré.

La fonction ouvre une fenêtre de sélection de projet à joindre à celle en cours de traitement.

Utilisation

  1. Choisissez Fichier → Fusionner un projet... à partir du menu principal.
  2. Une boîte de dialogue s'ouvrira vous demandant le nom et l'emplacement du fichier projet FreeCAD pour le fusionner dans le projet actuel.

Limites

Comme les noms des Entités/Objets du projet fusionné seront changés pour être adaptés
au schéma du projet en cours, les liens vers la géométrie externe risquent d'être perdus.
