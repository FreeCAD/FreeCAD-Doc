Descrizione

Permette di visualizzare o nascondere alcune schede o finestre della struttura grafica dell'interfaccia di FreeCAD.


Utilizzo

  • Avviare la funzione dal menu principale Visualizza → Pannelli
  • Appare la finestra per attivare o disattivare la visualizzazione di alcune parti della GUI:


Report

  • Il campo di testo Report fornisce un rapporto sull'esito del caricamento di FreeCad. Sono ad esempio elencate le operazioni non riuscite.
  • Cliccando con il destro nel rapporto si apre il menu delle sottofunzioni:


Console Python

  • La Console Python elenca tutti i comandi impartiti e da essa si possono impartire nuove istruzioni.


  • Per altri riferimenti sulla Console Python consultare anche Introduzione a Python e Script in FreeCAd.

Struttura

  • Visualizza una finestra con la struttura dei documenti attivi.


Proprietà

  • In questa finestra sono mostrate le proprietà Vista e le proprietà Dati di un oggetto selezionato nella struttura o nella vista 3D.

Da queste schede è possibile definire le proprietà degli elementi del documento.

Per una descrizione delle schede Vista e Dati consultare ad esempio la pagina Prisma, oppure gli altri comandi negli strumenti specifici degli ambienti di lavoro.


Selezione

  • Questa finestra elenca tutti gli elementi attualmente selezionati.


Vista combinata

  • Riunisce in una unica finestra le tre schede più utilizzate: la struttura del documento, le proprietà, e le azioni possibili quando si avviano funzioni specifiche, ad esempio Schiera lineare.
