Cette fonction ouvre la boîte de dialogue suivante pour exporter la vue en cours dans un fichier pdf.


Contrairement à d'autres types d'exportation qui vous permettent d'exporter des objets individuels, lorsque vous exportez dans ce format ce ne sont pas les objets qui sont exportés, mais la vue actuelle.

Utilisation

  • Réglez la vue que vous souhaitez voir apparaître sur la page pdf.
  • Lancer le menu Fichier → Exporter au Format PDF ....
  • Dans la fenêtre qui s' ouvre définissez l'emplacement et le nom, puis cliquez sur Enregistrer.
