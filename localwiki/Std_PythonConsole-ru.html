Определение

Создаёт дополнительное окно, показывающее и принимающее команды python.

Использование

Выберите в меню Вид → Панели → Консоль Python.
Когда Вы используете команду графического интерфейса FreeCAD, которые запрограммированы с помощью python
команды python показываются в этом окне.
Вы можете так же писать команды python, которые будут выполнены FreeCAD.

Ограничения


Примечания

FreeCAD содержит встроенный интерпретатор python.
Чтобы узнать его номер версии, выберите Справка → О FreeCAD → Скопировать в буфер обмена.
Вставив данные в текстовый редактор, можно увидеть версию интерпретатора python.
