Std Tools Menu/ru


Панели инструментов


Редактор параметров позволяет менять многие установки, которые управляют FreeCAD. Он не устанавливает параметры модели.

Сохранить изображение делает картинку с текущего окна трёхмерного вида.

Инспектор сцены показывает детали представления трёхмерного окна в Open Inventor.

Граф зависимостей показывает как зависят между собой различные объекты внутри модели.

Утилита проекта помогает в исправлении испорченных файлов проекта.

Поворотный просмотр показывает вращающийся вид модели в трёхмерном окне.

Калькулятор единиц обеспечивает быстрое преобразование между системами изменений, например, метрической и дюймовой.

Настройка позволяет менять меню, быстрые клавиши и панели инструментов.
