Description

Ouvre la boîte de dialogue Convertisseur d'unités, qui permet de convertir les quantités entrées en différentes unités.

Synopsis

Le premier champ défini une valeur et une unité correspondante (vous pouvez voir dans la description des grandeurs quels types d'unités sont reconnues). Dans le second champ, vous spécifiez l'unité résultante. Le troisième champ représente le résultat. En appuyant sur la touche Retour le résultat de la conversion sera ajoutée dans le journal pour un usage ultérieur.

Si l'arrière plan d'un des champs devient rouge, cela signifie une erreur d'entrée. Soit l'analyseur obtient une erreur d'expression dans l'unité ou l'unité source et cible ne correspondent pas. En plaçant le curseur sur le champ de saisie rouge, la chaîne d'erronée sera renseignée.

Le bouton Copy copie le résultat dans le presse-papier, vous pouvez donc utiliser l'information dans une autre fenêtre de dialogue de FreeCAD.

La boîte de dialogue est une boîte de dialogue non modale et peut rester ouverte aussi longtemps que vous le souhaitez. Elle n’influe pas sur le fonctionnement de FreeCAD.

Dialogue

La boîte de dialogue avec les trois champs et la zone de journal.
