Description

Cette fonctionnalité vous permet de créer de nouvelles fenêtres de vues 3D dans l'espace de travail. Utile pour vérifier le travail simultanément à partir de différents points de vue.

Utilisation

  • Lancer le menu Affichage→ Créer une nouvelle vue.

Dans la fenêtre ci-dessous, la zone de travail apparaît un nouvel onglet:

  • Dans la fenêtre sélectionnez la vue que vous souhaitez afficher

Autres Utilités

  • Dans le menu fenêtre vous pouvez définir la disposition des fenêtres, de carrelage ou en cascade et choisir de les activer.
  • Le menu Affichage→ Fenêtre Document vous permet de libérer la fenêtre active de la structure principale et de la positionnée librement sur l'écran. Cette fonction peut également être accessible par un clic droit dans un espace de travail vide. La fenêtre non ancrée est un élément indépendant de l'interface graphique de FreeCAD et n'est plus répertoriée.
  • Effectuer Enregistrer l'image pour enregistrer un écran.
