Description

Vous permet d'adapter la vue pour afficher les éléments sélectionnés dans le document actif. Les éléments qui pourraient être sélectionnés dans les documents inactifs n'affectent pas la vue.

Utilisation

Sélectionnez les objets à afficher. Ouvrir le menu Affichage → vues standards cliquez sur → Affiche la Sélection
