Beschreibung

Das Werkzeug »Längenbemaßung« fügt ein lineares Maß in einer Ansicht hinzu. Das Längenmaß kann der Abstand zwischen zwei Ecken, der Länge einer Kante oder der Abstand zwischen zwei Kanten sein. Der Abstand ist zuerst der projizierte Abstand (wie in der Zeichnung dargestellt), aber dieser kann mit der Verwendung des Link Dimension Werkzeug zum eigentlichen 3D-Abstand geändert werden.

Anwendung

  1. Punkte oder Kanten auswählen, die für die Abmessung benötigt werden.
  2. Auf die Schaltfläche Bemaßung in die Zeichnung einfügen klicken.
  3. Die Bemaßung wird in die Ansicht eingefügt. Sie kann anschließend an die gewünschte Position gezogen werden.

Optionen

Keine.

Eigenschaften

  • Daten-EigenschaftX: Horizontale Position des Bemaßungstexts relativ zur Ansicht.
  • Daten-EigenschaftY: Vertikale Position des Bemaßungstexts relativ zur Ansicht.
  • Daten-EigenschaftFormatSpec: Zusätzlicher Text zum Bemaßungstext hinzufügen (%.2f »zusätzlicher Text«).
  • Daten-EigenschaftType: Länge, Radius, Durchmesser usw.. Wird normalerweise vom Benutzer nicht geändert.
  • Daten-EigenschaftMeasureType: »True« - bezogen auf die 3D-Geometrie, oder »Projected« - bezogen auf die Zeichnung. Wird normalerweise vom Benutzer nicht geändert.
  • Daten-EigenschaftFont: Name der Schriftart für den Bemaßungstext.
  • Daten-EigenschaftFontsize: Schriftgröße des Bemaßungstexts in mm.
  • Daten-EigenschaftLineWidth: Stärke der Bemaßungslinie.
  • Daten-EigenschaftColor: Text- und Linienfarbe.

Programmierung

Längenbemaßungen können mit Python zu den Seiten hinzugefügt werden. 

dim1 = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewDimension','Dimension')
dim1.Type = "Distance"
dim1.References2D=[(view1, 'Edge1')]
rc = page.addView(dim1)

Hinweise

  • Alle Bemaßungen sind extrem anfällig für das »berüchtigte Problem der topologischen Benennung«. Zum jetzigen Zeitpunkt ist es nicht möglich, Referenzen auf geometrische Objekte in 2D (projiziert) oder 3D (tatsächliche) synchron mit den Änderungen im Modell zu halten. Es wird daher empfohlen, Bemaßungen erst am Ende der Zeichnungserstellung hinzuzufügen.
