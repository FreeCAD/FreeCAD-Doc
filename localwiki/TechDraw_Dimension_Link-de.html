Beschreibung

Mit dem Werkzeug »Bemaßung verlinken« wird eine Verlinkung zwischen einer 3D-Geometrie und einer oder mehreren Abmessungen auf der Seite erstellt. Dieser Link ermöglicht die Abmessung vom tatsächlichen 3D-Wert anstelle des projezierten 2D-Wert zu verwenden.

Anwendung

  1. Erstelle wie gewohnt die Abmessungen auf der Zeichnungsseite.
  2. Wähle eine Geometrie (z. B. eine Kante) in der 3D-Modellansicht, die mit einer der Bemaßungen übereinstimmt.
  3. Auf die Schaltfläche Eine Abmessung mit einer 3D Geometrie verlinken klicken.
  4. Es öffnet sich ein Dialogfenster. Wähle eine oder mehrere Abmessungen, die mit der ausgewählten 3D-Geometrie verlinkt werden sollen.
  5. Mit Klick auf »OK« bestätigen

Optionen

Keine.

Eigenschaften

  1. Die MeasureType Eigenschaft einer verknüpften Abmessung wird von »Projected« auf »True« gesetzt.

Programierung

Mit Python können die Abmessungen zu der 3D-Geometrie verlinkt werden.

Beschränkungen

  1. Die Verlinkungen der Maße sind sehr empfindlich für das Problem »Topological Naming«. Es wird empfohlen, das die Verknüpfung der Maße einer der letzten Schritte in der Zeichnungserstellung ist.
  2. Das Werkzeug »Bemaßungen verlinken« verhindert nicht eine unlogische Verlinkung.
  3. Es gibt momentan keine Möglichkeit, eine Verlinkung wieder zu trennen. Die MeasureType Eigenschaft kann auf »Projected« geändert werden, und die Abmessungen verwenden dann den »Projected« Wert anstelle des »True« Werts.
