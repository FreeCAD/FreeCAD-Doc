Descrizione

Lo strumento Tratteggio geometrico riempie una faccia con un modello basato su uno specifico tratteggio AutoDesk PAT (tratteggi di autocad .pat). Una piccola serie di modelli di esempio sono disponibili in ".../Mod/TechDraw/PAT/FCPAT.pat".

Uso

  1. Selezionare una regione chiusa in una vista. La regione diventa verde.
  2. Premere il pulsante Tratteggio geometrico
  3. Si apre una finestra in cui è possibile selezionare il modello, una scala per il modello e uno spessore di linea.
  4. Potrebbe essere necessario premere Ricalcola e/o "scuotere" la vista genitore per aggiornare il modello.

Opzioni

Nessuna.

Proprietà

  • DATASource: La vista e la faccia che devono ricevere il modello di tratteggio.
  • DATAFile Pattern: La posizione del file PAT da utilizzare.
  • DATAName Pattern: Il nome dello specifico PAT all'interno di File Pattern.
  • DATAScale Pattern: La scala da applicare al modello (deve essere > 0.0).
  • VIEWWeight Pattern: Lo spessore delle linee del modello.
  • VIEWColor Pattern: Il colore delle linee del modello.

Scrip

Il tratteggio può essere aggiunto alle aree nelle Viste utilizzando Python. 

hatch = FreeCAD.ActiveDocument.addObject('TechDraw::DrawGeomHatch','GeomHatch')
hatch.Source = (view1,["Face0"])
hatch.FilePattern = "path/to/myPATfile.pat"
hatch.NamePattern = "Diamond"
rc = page.addView(hatch)

Note

  • Notare che i modelli di tratteggio geometrico sono inclusi quando una pagina di disegno viene salvata come file in formato SVG.
