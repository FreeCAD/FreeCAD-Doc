Description

L'outil Image insère une image bitmap (png, jpeg, jpg, bmp, etc.) à partir d'un fichier dans la page en tant que vue

Comment faire

  1. Appuyez sur le bouton Image
  2. Une boîte de dialogue de fichier s'ouvrira. Sélectionnez un emplacement et un nom de fichier.

Options

Aucune.

Propriétés

  • n/a

Script

Les fichiers Image Bitmap peuvent être insérés dans une page de dessin en utilisant le code Python. 

dvi = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewImage','TestImage')
rc = page.addView(dvi)
dvi.ImageFile = "pathToMy/imageFile.png"
dvi.Height = 200
dvi.Width  = 200


Notes

  • Utilisez la propriété Echelle pour ajuster la taille de l'image.
  • Utilisez les propriétés Largeur et Hauteur pour découper l'image.
