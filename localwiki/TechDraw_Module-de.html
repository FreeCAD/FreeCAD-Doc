TechDraw Module/de

FreeCad ist hauptsächlich eine 3D-CAD-Software, mit der sich Gegenstände als räumliche Modelle virtuell abbilden lassen. Mit Hilfe der im FreeCad-Arbeitsbereich "Technisches Zeichnen" zusammen gefassten Werkzeuge lassen sich aus den 3D-Modellen die früher ausschließlich benutzten Technischen Zeichnungen (2D-Ansichten und -Schnitte der Gegenstände u.a.) ableiten. Mit zusätzliche Werkzeugen können Abmessungen eingetragen, die Zeichnungen beschriftet (innerhalb und außerhalb eines Schriftfelds) und mit einem Zeichnungsrahmen versehen werden.

Zur Verbesserung der Vorstellung kann eine im FreeCAD-System ohnehin vorhandene räumliche Ansicht beigefügt werden, die früher wegen des Aufwandes bei "händischer" Arbeit meistens fehlte.

Wer Technische Zeichnungen nur von einfach geformten und folglich leicht vorstellbaren Gegenständen anfertigen möchte, sollte ein 2D-CAD-Programm benutzen und nicht den relativ aufwändigen (längere Einarbeitungszeit u.a.) Umweg über eine vorerst räumliche Darstellung mit Hilfe eines 3D-CAD-Programms gehen.

verfügbar in Version 0.17

Seiten

Diese Werkzeuge sind zum Erstellen von Seiten.

  • Neue Standardvorlage: Fügt eine neue Seite mit der Standard - Vorlage hinzu.
  • Neue Vorlagenauswahl: Fügt eine neue Seite mit einer auswählbaren Vorlage hinzu.

Ansichten

Dies sind Werkzeuge zum Erstellen von Ansichten.

  • Neue Ansicht eines Teils in aktive Zeichnung einfügen: Fügt eine 2D-Ansicht von einem oder mehreren abbildbaren Objekten hinzu.
  • Fügt die Ansicht von mehreren Seiten eines Teils in die aktive Zeichnung ein: Ruft einen Dialog zum Erstellen von Ansichten von einem oder mehreren Abbildbaren Objekten aus mehreren Richtungen auf.
  • Eine Schnitt-Ansicht eines Teils in die aktive Zeichnung einfügen: Fügt eine Schnittansicht von einem oder mehreren abbildbaren Objekten hinzu.
  • Eine Detailansicht eines Teils der aktiven Zeichnung einfügen: Fügt eine Detailansicht eines Teils einer vorhandenen Ansicht hinzu.
  • Fügt eine Notiz in die aktive Zeichnung ein: Fügt einer Seite einen Textblock mit einer Anmerkung hinzu.
  • Fügt ein Objekt der Draft WB in die aktive Zeichnung ein: Fügt eine Ansicht aus dem Draft-Arbeitsbereich hinzu.
  • Fügt eine Ansicht eines Arch Zeichnungsausschnitts in die aktive Zeichnung ein: Fügt die Ansicht einer Schnittebene des Arbeitsbereichs Architektur (Arch) in die aktive Zeichnung ein.
  • Ansicht einer Tabelle in die Zeichnung einfügen: Fügt die Ansicht einer ausgewählten Tabellenkalkulation in die Zeichnung ein.

Clips

These are tools to create and manage clipped views.

  • Clip: Inserts a clip group into a Page.
  • ClipPlus: Adds a View to a clip group.
  • ClipMinus: Removes a View from a clip group.

Abmessungen

Mit diese Werkzeugen werden Abmessungen hinzugefügt. Lineare Abmessungen können sich auf zwei Punkte, eine Linie oder zwei Linien beziehen.

  • Längenbemaßung einfügen: Fügt eine Längenbemaßung hinzu.
  • Eine horizontale Abstands-Bemaßung einfügen: Fügt eine horizontale Längenbemaßung hinzu.
  • Eine vertikale Abstands-Bemaßung einfügen: Fügt eine horizontale Längenbemaßung hinzu.
  • Einen Radius der gewählten Ansicht bemaßen: Fügt eine Radiusbemaßung zu einem Kreis oder Kreisbogen hinzu.
  • Ein neues Durchmessermaß in die gewählte Ansicht einfügen: Fügt eine Durchmesserbemaßung zu einem Kreis oder Kreisbogen hinzu.
  • Winkel bemaßen: Fügt eine Winkelbemaßung zwischen zwei geraden Linienkanten hinzu.
  • Eine Abmessung mit einer 3D-Geometrie verlinken: Eine oder mehrere Bemaßungen mit einer 3D-Geometrie verlinken.

Import/Export

Mit diesem Werkzeug werden Seiten für weitere Anwendungsprogramme exportiert.

  • Save SVG: Saves a Page as an SVG file.
  • Save DXF: Saves a Page as a DXF file.

Darstellungen

Mit diesen Werkzeugen werden das Aussehen der Seiten und Ansichten verändert.

  • Hatch Area: Applies a hatch pattern from a file to a Face.
  • Geometrische Schraffur: Hatches a Face using an Autodesk PAT specification.
  • New Symbol: Inserts an SVG symbol into a Page.
  • New Image: Inserts a png/jpg image into a Page.
  • Toggle Frames: Turns View frames and labels on or off in a Page.
  • Redraw Page: Redraws a Page without recalculating.

Zusatztfuktionen

  • Schraffur: TechDraw supports a number of hatching techniques.
  • Liniengruppe: Das Aussehen verschiedener Linientypen kann verändert werden.

Einstellungen

Vom Benutzer gewünschte Einstellungen (Liniendicke, Linienart u.v.m.) lassen sich in zwei verschiedenen Eingabefenster vornehmen. Aufruf: »Einstellungen« unter dem Hauptmenü »Bearbeiten«.

Programmierung

Das TechDraw-Modul verfügt über eine Python-API, mit der die Funktionen in Skripten und Makros verwenden können.

Beschränkungen

  • TechDraw-Zeichnungen und Python API sind nicht mit dem Drawing-Modul austauschbar. Es ist möglich, Drawing-Seiten mit Python (moveViews.py) in TechDraw-Seiten zu konvertieren. Es ist auch möglich, sowohl TechDraw-Seiten als auch Drawing-Seiten im selben FreeCAD-Dokument zu verwenden.
  • Im Vergleich zum Drawing Modul gibt es geringfügige Unterschiede bei den speziell bearbeitbaren Texten in den TechDraw-Vorlagen. Mehr dazu in den Forum Diskussionen here.
  • TechDraw ist ab FreeCAD 0.17 verfügbar.
  • TechDraw-Objekte nicht ausschneiden, kopieren oder einfügen. Das funktioniert selten gut.

Problemumgehungen

  • Im Moment nicht erforderlich.
