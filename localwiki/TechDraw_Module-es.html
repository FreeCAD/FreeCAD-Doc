TechDraw Module/es

FreeCAD es principalmente una herramienta de modelado 3D. Sin embargo, hay momentos en los que debemos producir dibujos 2D para documentación, contratos, permisos, etc. El Entorno de trabajo de Dibujo Técnico (TechDraw) se utiliza para producir dibujos técnicos básicos basados en el modelo 3D. Cada dibujo es una Página, y una Página contiene muchas vistas de objetos dibujables. La lista de objetos dibujables incluye Part :: Features, PartDesign :: Bodies, App :: Part groups y Document Object groups.

Dimensiones y áreas sombreadas se pueden agregar a las vistas.

Tenga en cuenta que si su objetivo principal es la producción de planos 2D complejos, es posible que desee considerar una herramienta dedicada a eso como LibreCad, QCad, TurboCad, etc.

available in version 0.17

Páginas

Estas son las herramientas para crear páginas.

  • New Default: Agrega una nueva página usando el predeterminado Template.
  • New Pick: Agrega una nueva página usando un seleccionado Template.

Vistas

Estas son las herramientas para crear vistas.

  • New View: Agrega una vista 2D de uno o más objetos dibujables.
  • New Projection Group: Invoca un diálogo para crear vistas de uno o más objetos dibujables desde múltiples direcciones.
  • New Section: Agrega una vista de sección transversal de uno o más objetos dibujables.
  • New Detail: Agrega una vista detallada de una parte de una vista existente.
  • New Annotation: Agrega un bloque de texto a una página.
  • New Draft: Agrega una vista de un objeto de módulo Draft.
  • New Arch: Agrega una vista de un módulo de Arch Section Plane Object.
  • Spreadsheet: Inserta una vista de una spreadsheet seleccionada en un dibujo.

Clips (Recortes)

Estas son herramientas para crear y administrar vistas recortadas.

  • Clip: Inserta un grupo de clip en una página.
  • ClipPlus: Agrega una vista a un grupo de clip.
  • ClipMinus: Elimina una vista de un grupo de clip.

Dimensiones

Estas son herramientas para trabajar con dimensiones. Las dimensiones lineales pueden basarse en dos puntos, una línea o dos líneas.

  • New Length: Agrega una dimensión de longitud.
  • New Horizontal: Agrega una dimensión de distancia horizontal.
  • New Vertical: Agrega una dimensión de distancia vertical.
  • New Radius: Agrega una dimensión de radio a un círculo o arco circular.
  • New Diameter: Agrega una dimensión de Diámetro a un círculo o un arco circular.
  • New Angle: Agrega una dimensión de Ángulo entre dos bordes rectos.
  • New Links: Vincula 1 o más dimensiones a la geometría 3D.

Importar/Exportar

Estas son herramientas para exportar páginas a otras aplicaciones.

  • Save SVG: Guarda una página como un archivo SVG.

Decoración

Estas son herramientas para cambiar el aspecto de las páginas y las vistas.

  • Hatch Area: Aplica un patrón de sombreado desde un archivo a una cara.
  • Geometric Hatch: Sombrea una cara usando una especificación PAT de Autodesk.
  • New Symbol: Inserta un símbolo SVG en una página.
  • New Image: Inserta una imagen png / jpg en una página.
  • Toggle Frames: Activa y desactiva ver marcos y etiquetas en una página.
  • Redraw Page: Redibuja una página sin volver a calcular.

Características adicionales

  • Hatching: TechDraw admite varias técnicas de sombreado.
  • Line Groups: los grosores predeterminados se pueden asignar a varios tipos de línea.

Configuraciones de preferencia

El módulo TechDraw tiene una pantalla preferences para ajustar los valores predeterminados.

Scripting

El módulo TechDraw presenta una Python API para que pueda usar sus funciones en scripts y macros.

Limitaciones

  • Los dibujos de TechDraw y la API de Python no son intercambiables con el viejo módulo de Dibujo. Es posible convertir páginas de dibujo en páginas de TechDraw con Python (moveViews.py). Es posible tener tanto TechDraw como Drawing Pages en el mismo documento de FreeCAD.
  • Existen pequeñas diferencias en la especificación de textos editables en plantillas en comparación con el módulo de dibujo. Vea la discusión del foro aquí.
  • TechDraw está disponible comenzando con FreeCAD 0.17.
  • No corte / copie / pegue objetos TechDraw. Esto generalmente no funciona bien.

Soluciones provisionales

  • No se requiere por el momento.
