TechDraw Module/fr

FreeCAD est principalement un outil de modélisation 3D. Cependant, il y a des moments où nous devons produire des dessins en 2D pour la documentation, les contrats, les permis, etc. L'atelier Dessin technique (TechDraw) est utilisé pour produire des dessins techniques de base qui découlent du modèle 3D. Chaque dessin est une page, et une page contient de nombreuses vues d'objets projetables. La liste des objets projetables comprend Part::Features, PartDesign::Bodies, groupes App::Part et groupes Document Object.

Les dimensions et les zones hachurées peuvent être ajoutées aux vues.

Notez que si votre objectif principal est la production de dessins 2D complexes, vous pouvez envisager un outil dédié tel que LibreCad, QCad, TurboCad, etc.

Disponible dans la version 0.17

Pages

Ce sont les outils pour créer des pages.

  • Nouvelle Page par défaut: Ajoute une nouvelle page en utilisant le modèle par défaut.
  • Nouvelle Page selon un modèle: Ajoute une nouvelle page en utilisant un modèle sélectionné.

Vues

Ce sont les outils pour créer des vues.

  • Nouvelle Vue unique: Ajoute une vue 2D d'une seule pièce.
  • Nouveau Groupe de projection: Invoque une boîte de dialogue pour créer des vues d'une seule pièce à partir de plusieurs directions.
  • Nouvelle Vue en coupe: Ajoute une vue en coupe d'une pièce.
  • Nouvelle Vue de détail: Ajoute une vue de détail d'une partie d'une pièce.
  • Nouvelle Annotation: Ajoute un bloc de texte à une page.
  • Nouvelle Vue d'un objet Draft: Ajoute une vue d'un objet du module Draft.
  • Nouvelle Vue d'une Section Arch: Ajoute une vue d'un objet Section du module Arch.
  • Spreadsheet: Insère une vue d'une feuille de calcul sélectionnée dans le dessin actif.

Masques

Ce sont des outils pour créer et gérer des plans de coupe.

  • Clip: Ajoute un groupe de masquage sur la feuille active.
  • ClipPlus: Ajoute une vue à un groupe dans le dessin actif.
  • ClipMinus: Supprime une vue d'un groupe dans le dessin actif.

Cotations

Ce sont les outils pour ajouter des cotations.

  • Nouvelle longueur: Ajouter une cotation de longueur.
  • Nouvelle distance horizontale: Ajouter une cotation de longueur horizontale.
  • Nouvelle distance verticale: Ajouter une cotation de longueur verticale.
  • Nouveau rayon: Ajouter une cotation radiale.
  • Nouveau diamètre: Ajouter une cotation de diamètre.
  • Nouvel angle: Ajouter une cotation d'un angle.
  • Nouveau lien: Lier une cotation ou plus avec une géométrie 3D.

Import/Export

Ce sont les outils pour exporter les pages vers d'autre applications.

  • Sauvegarder au format SVG: Sauvegarder une page dans un fichier SVG.
  • Sauvegarder au format DXF: Sauvegarder une page dans un fichier DXF.

Décoration

Ce sont les outils pour changer l'apparence des pages et des vues.

  • Hachures par Motif: Hachurer une face en utilisant un fichier image.
  • Hachures Géométriques: Hachurer une face en utilisant une spécification Autodesk PAT.
  • Nouveau symbole: Insérer un symbole SVG dans une page.
  • Nouvelle image: Insérer une image png/jpg dans une Page.
  • Basculer l'affichage des cadres: Basculer l'affichage des cadres des Vues et des étiquettes dans une Page.
  • Redessiner la Page: Redessine la page sans recalculer son contenu.

Fonctions additionnelles

  • Hachures: TechDraw prend en charge un certain nombre de techniques de hachures.
  • Groupes de lignes: les poids par défaut peuvent être affectés à différents types de lignes.

Paramètres des préférences

Le module TechDraw possède un écran de préférences pour ajuster les valeurs par défaut.

Scripting

Le module TechDraw comporte une API Python afin que vous puissiez utiliser ses fonctions dans les scripts et les macros.

Limitations

  • Les dessins TechDraw et l'API Python ne sont pas interchangeables avec le module Dessin. Il est possible de convertir des pages de dessin en pages TechDraw en utilisant Python (moveViews.py). Il est possible d'avoir à la fois des Pages TechDraw et Drawing dans le même document FreeCAD.
  • Il existe des différences mineures dans la spécification des textes modifiables dans les modèles par rapport au module Dessin. Voir la discussion du forum ici.
  • TechDraw est disponible à partir de FreeCAD 0.17.
  • Ne pas couper / copier / coller des objets TechDraw. Cela ne fonctionne généralement pas bien.

Solutions de contournement

  • Aucun requis pour le moment.
