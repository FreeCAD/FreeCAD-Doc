TechDraw Module/it

FreeCAD è principalmente uno strumento di modellazione 3D. In certi casi, però, si devono produrre dei disegni 2D per la documentazione, i contratti, le autorizzazioni, ecc. L'ambiente di Disegno tecnico (TechDraw) serve proprio per produrre questi disegni tecnici basati su un modello 3D. Ogni disegno è denominato Page (pagina), e ogni Page può contenere molte Viste di Oggetti disegnabili. La lista di oggetti disegnabili include Part::Features, PartDesign::Bodies, App::Part groups e Gruppi di oggetti del documento.

Alle viste possono essere aggiunte le dimensioni e le aree tratteggiate.

Nota. Se il vostro obiettivo primario è la produzione di disegni 2D complessi, si può prendere in considerazione uno strumento dedicato, come LibreCAD, QCad, TurboCad, etc.

disponibile dalla versione 0.17

Pagine

Questi sono gli strumenti per crere le Pagine.

  • Nuovo disegno standard: Aggiunge una nuova pagina utilizzando il Modello predefinito.
  • Nuovo disegno da modello: Aggiunge una nuova pagina utilizzando un Modello selezionato.

Viste

Questi sono gli strumenti per crere le Viste.

  • Vista: Aggiunge una vista 2D di uno o più oggetti 3D.
  • Gruppo di proiezione: Richiama una finestra di dialogo per la creazione di Viste da più direzioni di uno o più oggetti.
  • Sezione: Aggiunge una vista in sezione di ununo o più oggetti.
  • Dettaglio: Aggiunge la vista di un dettaglio di una porzione di una Vista esistente.
  • Annotazione: Aggiunge un blocco di testo a una pagina.
  • Vista di Draft: Aggiunge una vista di un oggetto di Draft.
  • Vista di Arch: Aggiunge una vista di un oggetto Piano di sezione di Arch.
  • Foglio di calcolo: Inserisce in un disegno una vista di un foglio di calcolo selezionato.

Clip

Questi sono strumenti per creare e gestire ritagli di viste.

  • Clip: Inserisce un gruppo di clip in una pagina.
  • ClipPlus: Aggiunge una vista a un gruppo di clip.
  • ClipMinus: Rimuove una vista da un gruppo di clip.

Dimensioni

Questi sono gli strumenti per lavorare con le Dimensioni.

  • Lunghezza: Aggiunge una dimensione Lunghezza.
  • Orizzontale: Aggiunge una dimensione di distanza orizzontale.
  • Verticale: Aggiunge una dimensione di distanza verticale.
  • Raggio: Aggiunge una dimensione Raggio.
  • Diametro: Aggiunge una dimensione Diametro.
  • Angolo: Aggiunge una dimensione Angolo.
  • Link: Collega 1 o più dimensioni alla geometria 3D.

Importazione e Esportazione

Si tratta di strumenti per esportare le pagine in altre applicazioni.

  • Salva SVG: Salva una pagina come file SVG
  • Salva DXF: Salva una pagina come file DXF.

Aspetto

Si tratta di strumenti per modificare l'aspetto delle Pagine e della Vista

  • Tratteggio area: Applica a una faccia un modello di tratteggio preso da un file.
  • Tratteggio geometrico: Tratteggia una faccia usando uno specifico Autodesk PAT.
  • Simbolo: Inserisce un simbolo SVG in una pagina.
  • Immagine: Inserisce una immagine png/jpg nella Pagina.
  • Impaginazione: Mostra o nasconde le strutture delle Viste e le etichette nelle Pagine.
  • Ridisegna Pagina: Ridisegna una pagina senza ricalcolarla.

Ulteriori funzioni

  • Tipi di tratteggio: TechDraw supporta diversi tipi di tratteggio tecnico.
  • Gruppi di linee: si possono assegnare valori di default a vari tipi di linee.

Impostare le preferenze

Il modulo TechDraw module ha una scheda per impostare le preferenze di default.

Script

Il modulo TechDraw dispone di una API Python quindi si possono utilizzare le sue funzioni negli script e nelle macro.

Limitazioni

  • I disegni di TechDraw e le sue API Python non sono intercambiabili con il modulo Drawing. È possibile convertire le pagine di Drawing in pagine di TechDraw utilizzando Python (moveViews.py). È possibile avere sia pagine di TechDraw che pagine di Drawing nello stesso documento FreeCAD.
  • Ci sono delle piccole differenze nelle specifiche dei testi modificabili nei modelli di cartiglio rispetto al modulo Drawing. Vedere nel forum questa discussione.
  • TechDraw è disponibile a partire da FreeCAD 0.17.
  • Non tagliare / copiare / incollare oggetti TechDraw. Queste operazioni generalmente non funzionano bene.
