Description

L'outil Annotation ajoute un bloc de texte à une page de dessin.

Comment faire

  1. Si vous avez plusieurs pages de dessin dans votre document, vous devez sélectionner la page souhaitée dans l'arborescence.
  2. Appuyez sur le bouton Annotation
  3. Un bloc de texte contenant "Texte par défaut" apparaîtra sur la page. Utilisez l'éditeur de propriétés pour modifier le texte. Faites glisser l'annotation à la position requise.
  4. Vous devrez peut-être appuyer sur recalculer pour que votre texte change.

Options

Aucune.

Propriétés

ADD_PROPERTY_TYPE(Text ,("Default Text"),vgroup,App::Prop_None,"Le texte qui doit être affiché");
ADD_PROPERTY_TYPE(Font ,(fontName.c_str()),vgroup,App::Prop_None, "Le nom de la police à utiliser");
ADD_PROPERTY_TYPE(TextColor,(0.0f,0.0f,0.0f),vgroup,App::Prop_None,"La couleur du texte"); 
ADD_PROPERTY_TYPE(TextSize,(8),vgroup,App::Prop_None,"La taille du texte en mm");
ADD_PROPERTY_TYPE(MaxWidth,(-1.0),vgroup,App::Prop_None,"La largeur maximale du bloc d'annotation");
ADD_PROPERTY_TYPE(LineSpace,(80),vgroup,App::Prop_None,"Ajustement d'interligne");
TextStyle.setEnums(TextStyleEnums);
ADD_PROPERTY(TextStyle, ((long)0));
  • DONNÉESText: Le texte à afficher.
  • DONNÉESFont: Le nom de la police à utiliser. L'annotation utilisera la meilleure correspondance des polices installées.
  • DONNÉESTextColor: La couleur du texte
  • DONNÉESTextSize: La taille du texte en mm.
  • DONNÉESMaxWidth: La largeur maximale du bloc d'annotation. -1 indique pas de largeur maximale.
  • DONNÉESLineSpace: Ajustement d'interligne (%).
  • DONNÉESTextStylee: "Normal", "Gras", "Italique", "Gras-Italique"

Remarque: L'annotation hérite de toutes les propriétés de vue de base applicables, à l'exception de l'échelle. Utilisez la propriété TextSize à la place.

Script

Les annotations peuvent être ajoutées à Pages en utilisant le code Python. 

anno = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewAnnotation','TestAnno')
anno.Text = ['Different Text']
anno.TextStyle = 'Bold'
rc = page.addView(anno)
