Beschreibung

Das Werkzeug »Neue Detailansicht« erstellt als neue Ansicht einen kleinen Bereich aus einer vorhandenen Ansicht.

Anwendung

  1. Markiere eine Ansicht im 3D-Fenster oder in der Baumansicht.
  2. Wenn im Dokument mehrere Seiten vorhanden sind, muss die gewünschte Seite in der Baumansicht markiert werden.
  3. Auf die Schaltfläche Detailansicht in die Zeichnung einfügen klicken

Optionen

Keine.

Eigenschaften

  • Daten-EigenschaftBaseView: Die Referenzansicht für die Detailansicht.
  • Daten-EigenschaftAnchor: Das Zentrum der Detailansicht innerhalb der Referenzansicht.
  • Daten-EigenschaftRadius: Radiuswert für den angezeigten Detailbereich.
  • Daten-EigenschaftScale: Skalierungsstufe.
  • Daten-EigenschaftReference: Identifikationsnummer zur Kennung der dargestellten Referenzansicht.

Hinweis: Die Detailansicht erbt alle anwendbaren Ansichtseigenschaften.

Programmierung

Mit Python kann die Detailansicht auf den Seiten hinzugefügt werden. 

Detail = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewDetail','Detail')
...TBA

Hinweis

  • Die Detailansicht kann in einer runden oder quadratischen Ansicht dargestellt werden. Dieses wird durch die Voreinstellung »Mattierungsstil« in den TechDraw Einstellungen gesteuert.
