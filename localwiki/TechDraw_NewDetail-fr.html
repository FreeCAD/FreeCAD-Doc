Description

L'outil Détail crée une vue d'une petite zone d'une vue existante.

Comment faire

  1. Sélectionnez une vue d'une pièce dans la fenêtre ou l'arborescence 3D.
  2. Si vous avez plusieurs pages de dessin dans votre document, vous devrez également sélectionner la page désirée dans l'arborescence.
  3. Appuiyez sur le bouton Détail

Options

Aucune.

Propriétés

  • DONNÉESBaseView: La vue sur laquelle se base ce Détail.
  • DONNÉESAnchor: Le centre de la vue de détail dans la vue de référence.
  • DONNÉESRadius: Contrôle la taille de la zone affichée.
  • DONNÉESScale: Niveau de grossissement.
  • DONNÉESReference: Un identifiant pour indiquer la zone de la vue de référence affichée.

Note: le Détail hérite de toutes les propriétés de vue applicables.

Script

Les détails peuvent être ajoutés aux Pages en utilisant le code Python. 

Detail = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewDetail','Detail')
...TBA

Notes

  • La vue de détail peut être affichée dans une "zone" ronde ou carrée. Ceci est contrôlé par le réglage de préférence "Matting".
