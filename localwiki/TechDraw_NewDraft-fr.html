Description

L'outil Vue Draft insère une vue d'un objet Part ou Groupe sélectionné dans une page de dessin. Contrairement à l'outil standard Vue, les vues créées avec cet outil sont gérées par Atelier Draft et spécialement conçues pour afficher des objets 2D. Voir les notes.

Comment faire

  1. Sélectionnez un objet dessin dans la vue 3D ou dans l'arborescence
  2. Si vous avez plusieurs pages de dessin dans votre document, vous devez sélectionner la page souhaitée dans l'arborescence.
  3. Appuyez sur le bouton Vue Draft
  4. Une vue de l'objet dessin apparaîtra sur la page.

Options

  • La création d'une vue Draft d'un groupe traitera récursivement tous les objets trouvés dans ce groupe et dans ses sous-groupes. La vue est mise à jour automatiquement lorsque le contenu du groupe change
  • Il n'y a pas de suppression de ligne cachée. Chaque face trouvée dans l'objet manipulé sera simplement projetée le long du vecteur Direction, aucune action spécifique n'est prise lorsque les faces se chevauchent
  • La vue Draft prend également en charge tous les objets Draft qui ne sont pas basés sur des pièces, tels que les dimensions et les textes
  • La couleur, la largeur de ligne et le motif de ligne peuvent être spécifiés dans les propriétés. Les motifs de ligne peuvent être affinés en donnant directement une valeur stroke-dasharray, telle que 3,5
  • Les faces projetées sont remplies avec la couleur de la face

Propriétés

  • DONNÉESSource: L'objet Draft à afficher
  • DONNÉESLineWidth: La largeur des lignes, indépendamment de l'échelle
  • DONNÉESFontSize: La taille de tous les textes apparaissant dans cette vue (textes et dimensions)
  • DONNÉESDirection: La direction de projection à utiliser
  • DONNÉESColor: La couleur des lignes
  • DONNÉESLineStyle: Un style de ligne à utiliser pour cette vue. Peut être continu, pointillé, tireté, point ou un motif de ligne SVG comme 0.20,0.20
  • DONNÉESLineSpacing: L'espacement à utiliser entre les lignes de textes pour les textes multilignes

Remarque: Vue Draft hérite de toutes les propriétés Vues de base applicables.

Script

Vue Draft peut être ajouté à Pages en utilisant le code Python. 

dv = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewDraft','TestDraft')
dv.Source = myDraftbject
rc = page.addView(dv)

Notes

Depuis que la vue Draft est réalisée dans l'atelier Draft, TechDraw a un contrôle limité sur son apparence. Vous devrez peut-être effectuer des modifications dans Draft pour obtenir la représentation souhaitée.
