Description

Remarque: cet outil est obsolète et sera supprimé. NewView prend désormais en charge plusieurs objets source. L'outil Multi crée une vue de plusieurs parties 3D. Les pièces sont positionnées par rapport à leurs positions 3D. Des dimensions peuvent être ajoutées entre les parties, le cas échéant.

Comment faire

  1. Sélectionnez plusieurs parties dans la fenêtre ou l'arborescence 3D.
  2. Si vous avez plusieurs pages de dessin dans votre document, vous devrez également sélectionner la page désirée dans l'arborescence.
  3. Appuiyez sur le bouton Multi

Options

Aucune.

Propriétés

  • DONNÉESBaseView: La vue partielle sur laquelle est basé ViewMulti. (Notez qu'une seule des parties est listée ici).

Remarque: Multi hérite de toutes les propriétés View.

Script

Multis peut être ajouté aux Pages en utilisant Python. 

Multi = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewMulti','Multi')
