Descrizione

Note: questo strumento è deprecato e verrà rimosso. Ora Nuova Vista supporta più oggetti sorgente. Lo strumento Vista multipla crea una vista di più parti 3D. Le parti sono posizionate rispettando le loro posizioni 3D. Si possono aggiungere le distanze tra le parti, se è necessario.

Uso

  1. Selezionare più parti nella finestra 3D o nella struttura ad albero.
  2. Se nel documento ci sono più pagine di disegno, è anche necessario selezionare la pagina desiderata nella struttura.
  3. Premere il pulsante Vista multipla

Opzioni

Nessuna.

Proprietà

  • DATABaseView: La vista della Parte su cui si basa questa Vista multipla. (Nota Viene elencata solo una delle Parti)

Nota: Vista multipla eredita tutte le proprietà Vista applicabili.

Script

Le Viste multiple possono essere aggiunte alle Pagine utilizzando Python. 

Multi = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewMulti','Multi')
