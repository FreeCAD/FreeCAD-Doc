Beschreibung

Das Werkzeug »Neue Standardvorlage« erstellt eine neue Seite unter Verwendung der in TechDraw Einstellungen angegebenen Standardvorlage. Wenn diese Vorlage nicht angegeben wurde (user.cfg), wird »Neue Standardvorlage» in folgendem Pfad suchen: "mein-install-verzeichnis/Mod/TechDraw/Templates/A4_LandscapeTD.svg".

Beschreibung

  1. Auf die Schaltfläche Neues Zeichenblatt einfügen klicken. Hinweis: Ein aktives Dokument muss vorhanden sein.

Optionen

Keine.

Eigenschaften

  • Daten-EigenschaftProjection Type: Projektionswinkel als erste (europäische Darstellung) oder die dritte (amerikanische Darstellung) Projektionsmethode für diese Seite.
  • Daten-EigenschaftKeepUpdated: »False«: Seite wird nicht aktualisiert, wenn das 3D-Modell geändert wird. Bei komplexen 3D-Modellen kann das hilfreich sein. Siehe auch Hinwiese.
  • Daten-EigenschaftTemplate: Link zum Vorlagen-Pfad.
  • Daten-EigenschaftViews: Im Dialogfenster eine Liste von Ansichten, die auf der Seite vorhanden sind.
  • Daten-EigenschaftScale: Standard-Skalierungsfaktor für die Ansichten auf der Seite.

Programmierung

Das Werkzeug »Neue Standardvorlage« kann mit Makros und von der Python-Konsole aus mithilfe der folgenden Funktionen verwendet werden: 

page = FreeCAD.ActiveDocument.addObject('TechDraw::DrawPage','Page')
template = FreeCAD.ActiveDocument.addObject('TechDraw::DrawSVGTemplate','Template')
template.Template = templateFileSpec
page.Template = FreeCAD.ActiveDocument.Template
  • Erstellt eine neue Seite im aktuellen Dokument

Hinweise

  • Wenn auf der Seite die Eigenschaft »KeepUpdated« auf »False« gesetzt, oder die TechDraw Einstellung »Seiten aktuell zu halten" deaktiviert ist, wird die Seite bei einer 3D-Modelländerung nicht aktualisiert. Wenn Geometrien oder Bemaßungen fehlen, könnte das die Ursache sein. Sobald die Seite aktualisiert wird, sollten die Fehler behoben sein.
