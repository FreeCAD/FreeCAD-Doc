Beschreibung

Das Werkzeug »Neue Vorlagenauswahl« erstellt eine neue Seite mit der in einem Dialogfenster auswählbaren Vorlagen. Das Startverzeichnis für das Dialogfenster kann in den TechDraw Einstellungen festgelegt werden.

Beschreibung

  1. Auf die Schaltfläche Ein neues Zeichenblatt aus einer Vorlage einfügen klicken.

Optionen

Keine.

Eigenschaften

  • Daten-EigenschaftProjection Type: Projektionswinkel als erste (europäische Darstellung) oder die dritte (amerikanische Darstellung) Projektionsmethode für diese Seite.
  • Daten-EigenschaftScale: Standard-Skalierungsfaktor für die Ansichten auf der Seite.

Programmierung

Das Werkzeug »Neue Vorlagenauswahl« kann mit Makros und von der Python-Konsole aus mithilfe der folgenden Funktionen verwendet werden: 

templateFileSpec = QtGui.QFileDialog.getOpenFileName(self.baseWidget,
                                                     dialogCaption, 
                                                     dialogDir,
                                                     dialogFilter)
page = FreeCAD.ActiveDocument.addObject('TechDraw::DrawPage','Page')
template = FreeCAD.ActiveDocument.addObject('TechDraw::DrawSVGTemplate','Template')
template.Template = templateFileSpec
page.Template = FreeCAD.ActiveDocument.Template
