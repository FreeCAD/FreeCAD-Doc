Descrizione

Lo strumento Nuovo disegno da modello crea una nuova pagina utilizzando il file di un modello selezionato in una finestra di dialogo. La directory di partenza per il dialogo può essere specificata nelle Preferenze

Uso

  1. Premere il pulsante Nuovo disegno da modello

Opzioni

Nessuna.

Proprietà

  • DATAProjection Type: Tipo predefinito di proiezione (primo o terzo angolo) per questa pagina.
  • DATAScale: Scala predefinita per le Viste in questa pagina.

Script

Lo strumento Nuovo disegno da modello può essere utilizzato nelle macro e dalla console Python utilizzando le seguenti funzioni: 

templateFileSpec = QtGui.QFileDialog.getOpenFileName(self.baseWidget,
                                                     dialogCaption, 
                                                     dialogDir,
                                                     dialogFilter)
page = FreeCAD.ActiveDocument.addObject('TechDraw::DrawPage','Page')
template = FreeCAD.ActiveDocument.addObject('TechDraw::DrawSVGTemplate','Template')
template.Template = templateFileSpec
page.Template = FreeCAD.ActiveDocument.Template
  • Crea una nuova pagina nel documento corrente
