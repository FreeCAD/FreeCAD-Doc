TechDraw Preferences/fr

Les écrans de préférences du module TechDraw se trouvent dans la fenêtre Préférences (Menu Édition → Préférences).

Vos écrans de préférences peuvent sembler légèrement différents selon la plate-forme ou le schéma de formatage.

TechDraw 1


Général

  • Angle de Projection - ProjectionGroups utilisera la projection du premier ou du troisième angle. (Voir projection multivues)
  • Lignes cachées - le style (solide, pointillé, etc) à utiliser pour les lignes cachées.
  • Détecter les faces- s'il est coché, TechDraw tentera de construire des faces en utilisant les segments de ligne renvoyés par l'algorithme de suppression de ligne cachée. Les faces doivent être détectées pour utiliser hachurage, mais il peut y avoir une pénalité de performance dans les modèles complexes.
  • ShowSectionEdges (montrer les limites de la section) - met en surbrillance la bordure de la section coupée dans section vues. Habituellement seulement pour le débogage.
  • KeepPagesUpToDate - garder les pages de dessin en synchronisation avec le modèle 3D en temps réel : peut ralentir les temps de réponse. Dans les modèles complexes, décocher cette case permettra des réponses plus rapides. Utilisez la propriété "KeepUpdated" de Page pour mettre à jour une seule page de manière sélective.
  • Template Dot Size - Contrôle la taille du petit point vert utilisé pour sélectionner le texte éditable dans un Modèle.

Couleurs

vous pouvez définir vos couleurs préférées ici.

Labels

  • Label Font - le nom de la police par défaut pour les étiquettes.
  • Label Size - taille par défaut (mm) pour le texte de l'étiquette.
  • Template dot size - points dans l'unité désirée

Fichiers

  • Default Template - utiliser ce sélecteur de fichier pour sélectionner un fichier modèle par défaut pour vos dessins.
  • Template Directory - le répertoire de départ pour Nouvelle page à partir du modèle.
  • Hatch Image - fichier Svg ou Bitmap par défaut pour hachures.
  • PAT File - fichier de définition de modèle PAT par défaut pour hachures.
  • Pattern Name - le nom de votre motif PAT préféré.

TechDraw 2

Dimensions

  • Show Units - ajouter le spécificateur d'unité (mm, in, etc) aux valeurs.
  • Use Global Decimals - utiliser le nombre de décimales de Preferences>General>Units.
  • Alternate Decimals - Si Utiliser les décimales globales est décoché, utiliser ce nombre de décimales.
  • Font Size - taille pour le texte de cote.
  • Color - default color for Dimension text and lines.
  • Diameter Symbol - Les dimensions du diamètre seront préfixées par ce texte.
  • Arrow Style - choisissez la marque de fin de ligne de cote que vous préférez.
  • Arrow Size - taille de la marque de fin en mm.

Apparences

  • Line Group - le nom par défaut de la spécification du groupe de lignes.
  • Matting Style - afficher les vues détaillées dans des cercles ou des rectangles.
  • Center Line Style (Style des lignes d'axes) - couleur par défaut pour les lignes d'axes (horizontales et verticales).
  • Center Line Color (Couleur des lignes d'axes) - couleur par défaut pour les lignes d'axes.
  • Section Line Style - style par défaut pour la ligne de coupe dans la vue de référence de la vue en coupe.
  • Section Line Color - couleur par défaut pour les lignes de section.
  • Hatch Weight - Épaisseur de ligne par défaut pour les hachures géométriques. Voir hachurage.
