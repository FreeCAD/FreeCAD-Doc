Descrizione

Lo strumento Esporta pagina salva la corrente pagina di disegno come file DXF

Utilizzo

  1. Premere il pulsante Save page as Dxf
  2. Si apre una finestra di dialogo Salva file. Selezionare un percorso e il nome file.

Opzioni

Nessuna.

Proprietà

  • n/a

Script

Questa funzione è disponibile tramite Python usando 

TechDraw.writeDXFPage(page,filename)

Note

  • Attualmente vengono esportati solo i bordi della vista. Vertex, annotazioni e dimensioni sono pianificate per l'implementazione in un secondo tempo.
  • Questa funzione è disponibile in v0.18 e successive.
