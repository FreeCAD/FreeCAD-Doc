Description

L'outil Exporter page enregistre la page de dessin en cours en tant que fichier SVG

Comment faire

  1. Appuyez sur le bouton Exporter page
  2. Une boîte de dialogue Sauvegarder Fichier s'ouvre. Sélectionnez un emplacement et un nom de fichier.

Options

Aucune.

Propriétés

  • n/a

Script

Cette fonction est disponible via un code Python en utilisant: 

TechDrawGui.exportPageAsSvg(DrawPageObject,FilePath)

Notez que le module FreeCADGui doit être actif pour utiliser cette fonction.

Notes

  • Les motifs de hachures ne sont pas exportés vers SVG en raison d'une limitation dans le support SVG de Qt4.
  • Les positions et tailles de texte ne sont pas correctes dans le fichier exporté. L'utilisation de la police "système" par défaut dans le dessin améliore considérablement le problème de taille.
  • La création de fichier DXF à partir du dessin Svg enregistré dans Inkscape: Pour exporter vers DXF avec l'échelle correcte dans Inkscape v0.48, choisissez "Desktop Cutting Plotter (AutoCAD DXF R14) (* .dxf)" comme format d'exportation et choisissez "Unité de base": "px" dans la boîte de dialogue suivante. Le dessin n'est pas encore à l'échelle mais a une échelle de 10: 1. Pour obtenir l'échelle correcte sur LibreCAD, choisissez "Edition → Préférences de dessin en cours → Facteur général" de 0.1. Vérifiez avec une distance connue, de toute façon!
