Descrizione

Lo strumento Esporta pagina salva la corrente pagina di disegno come file SVG

Uso

  1. Premere il pulsante Esporta pagina
  2. Si apre una finestra di dialogo Salva file. Selezionare un percorso e il nome file.

Opzioni

Nessuna.

Proprietà

  • n/a

Script

Questa funzione è disponibile tramite Python usando: 

TechDrawGui.exportPageAsSvg(DrawPageObject,FilePath)

Si noti che il modulo FreeCADGui deve essere attivo per utilizzare questa funzione.

Note

  • Le campiture di tratteggio non vengono esportate nel file SVG causa di una limitazione nel supporto SVG di Qt4.
  • Nel file esportato le posizioni e le dimensioni dei testi non sono corrette. L'uso del carattere predefinito "sistema" nel disegno migliora considerevolmente il problema delle dimensioni.
  • Creare un DXF dal disegno Svg salvato in Inkscape: Per esportare in DXF con la scala corretta in Inkscape v0.48, selezionare "Desktop Cutting Plotter (AutoCAD DXF R14) (* .dxf)" come formato di esportazione e scegliere "Unità di base": "px" nella finestra di dialogo successiva. Il disegno non è ancora in scala ma ha una scala di 10: 1. Per ottenere la scala corretta su LibreCAD, selezionare "Modifica → Preferenze disegno corrente → Fattore generale" uguale a 0,1. Verificare comunque su una distanza nota!
