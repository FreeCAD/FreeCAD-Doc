Beschreibung

Das Symbolwerkzeug fügt eine SVG-Datei als Ansicht in die Zeichnungsseite ein.

Anwendung

  1. Auf die Schaltfläche Symbol klicken.
  2. Der Dateiauswahldialog öffnet sich. Den Pfad und den Namen der entsprechenden SVG-Datei auswählen.

Optionen

Keine.

Eigenschaften

  • nicht verfügbar

Scripting

SVG-Dateien können mittels Python in eine Zeichnungsseite eingefügt werden. 

sym = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewSymbol','TestSymbol')
rc = page.addView(anno)
f = open(unicode(symbolFileSpec,'utf-8'),'r')
svg = f.read()
f.close()
sym.Symbol = svg
rc = page.addView(sym)


Anmerkungen

  • Das Symbol kann größer als erwartet dargestellt werden. In diesen Fall kann die Skalierungseigenschaft genutzt werden um die Größe anzupassen.
