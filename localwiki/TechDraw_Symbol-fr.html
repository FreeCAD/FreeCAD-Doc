Description

L'outil Symbole insère un fichier SVG dans la page en tant que vue

Comment faire

  1. Appuyez sur le bouton Symbole
  2. Une boîte de dialogue s'ouvre. Sélectionnez un emplacement et un nom de fichier.

Options

Aucune.

Propriétés

  • n/a

Script

Les fichiers SVG peuvent être insérés dans une page de dessin en utilisant le code Python. 

sym = FreeCAD.ActiveDocument.addObject('TechDraw::DrawViewSymbol','TestSymbol')
rc = page.addView(anno)
f = open(unicode(symbolFileSpec,'utf-8'),'r')
svg = f.read()
f.close()
sym.Symbol = svg
rc = page.addView(sym)


Notes

  • Le symbole apparaît plus grand que prévu. Utilisez la propriété Échelle pour ajuster la taille.
