Descrizione

Lo strumento Impaginazione mostra o nasconde le strutture delle Viste, le etichette ed i vertici.



Uso

  1. Se nel documento ci sono più pagine di disegno, è necessario selezionare la pagina desiderata nella struttura.
  2. Premere il pulsante Impaginazione
  3. Se le strutture delle Viste sono attualmente visibili, scompaiono. Se le strutture delle Viste non sono attualmente visibili, appaiono.
  4. È possibile avere diversi stati di visualizzazione della struttura per le diverse Viste. Se ciò accade, premere una o due volte il pulsante Impaginazione per risincronizzare le Viste.
