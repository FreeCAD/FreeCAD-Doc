WebTools Workbench/it

Questo è un ambiente di lavoro esterno contenente una serie di strumenti per comunicare con i servizi web dall'interno FreeCAD. È ospitato in https://github.com/yorikvanhavre/WebTools

  • BIM server: Apre una finestra BimServer
  • Git: Gestione dei file con GIT
  • Sketchfab exporter: Esporta e carica oggetti nel proprio account SketchFab available in version 0.17
Online version: "http://www.freecadweb.org/wiki/index.php?title=WebTools_Workbench/it&oldid=230130"
