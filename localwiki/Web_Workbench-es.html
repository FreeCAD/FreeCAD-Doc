Web Workbench/es

El Ambiente de trabajo Web proporciona una ventana del navegador web dentro de FreeCAD, basada en WebKit. Desde este navegador, puede navegar por la web e iniciar sesión en sitios web (se admiten cookies). El ambiente de trabajo web también es una base para desarrollar herramientas relacionadas con la web en FreeCAD. El navegador web proporciona un par de botones de navegación estándar, como detener, recargar, página anterior y siguiente.
