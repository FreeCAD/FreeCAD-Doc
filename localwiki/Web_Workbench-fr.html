Web Workbench/fr

Web Workbench fournit une fenêtre de navigateur Web dans FreeCAD, basée sur WebKit. Depuis ce navigateur, vous pouvez naviguer sur le Web, vous connecter à des sites Web (les cookies sont pris en charge). Le workbench Web est également une base pour le développement d'outils Web dans FreeCAD. Le navigateur Web fournit quelques boutons de navigation standard tels que l'arrêt, le rechargement, la page précédente et la page suivante.
