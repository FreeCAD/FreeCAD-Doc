Types of template elements

Types of template elements inserted into the texts to istinguish and standardize.
My opinion is that in the texts is good to use black and white.
The color should be used only for special things.
The color according to the importance.
Besides these elements must have always the same formatting:

  1. Command path (Part -> Primitives -> ...)
  2. User actions (Press the Draft Circle button)(Press ESC or the Cancel button)
  3. Shortcut commands (press C then I keys)
  4. Default values (Value FALSE, or TRUE (Default, FALSE). Default, 25, 25, 25)
  5. Links
  6. Property
    • View (VIEWBounding Box)
    • Data (DATALength)
    • Task (TASKS Length)

  7. Summary boxes, macro
  8. Alerts
  9. Very important message
